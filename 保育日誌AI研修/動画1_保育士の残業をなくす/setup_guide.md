# 動画1：保育士の残業をなくす - API設定ガイド

## Google AI Studio APIキー取得ガイド

### なぜAPIキーが必要？

動画2で作成するDifyアプリを継続的に使用するためには、APIキーの設定が必須です。
- Difyには無料クレジットがありますが、使い切ると継続利用できません
- APIキーを設定することで、無料枠を超えても使い続けられます
- Google AI Studioなら完全無料でAPIキーを取得できます

### Google AI Studio APIキー取得手順

#### 1. Google AI Studioにアクセス
1. [Google AI Studio](https://aistudio.google.com) にアクセス
2. Googleアカウントでログイン
3. 利用規約に同意

#### 2. APIキーの作成
1. 左メニューの「Get API key」をクリック
2. 「Create API key in new project」をクリック
3. プロジェクト名を入力
   - 例：「保育日誌AI」
   - 例：「○○保育園アプリ」
4. 「Create」をクリック

#### 3. APIキーの保存
⚠️ **重要：APIキーは一度しか表示されません**

1. 表示されたAPIキーをコピー
2. 安全な場所に保存
   - メモ帳
   - パスワード管理ツール
   - セキュアなドキュメント

### Google AI StudioのAPIキー利用条件

#### 無料利用枠
- **料金**: 完全無料
- **利用制限**: 毎分15回まで（RPM: Requests Per Minute）
- **1日の制限**: 1,500回まで
- **クレジットカード**: 不要
- **用途**: 日誌作成には十分な量

#### 利用可能なモデル
- **Gemini 1.5 Flash**: 高速処理（推奨）
- **Gemini 1.5 Pro**: 高精度処理
- **Gemini 1.0 Pro**: 基本モデル

### OpenAI API（補足情報）

#### OpenAI APIの特徴
- **初期設定**: クレジットカード登録が必要
- **料金体系**: 従量課金制（使った分だけ支払い）
- **無料クレジット**: 初回登録時のみ$5程度
- **モデル**: GPT-3.5, GPT-4など

#### OpenAI APIキー取得手順
1. [platform.openai.com](https://platform.openai.com) にアクセス
2. アカウント作成
3. クレジットカード情報を登録
4. API Keys → Create new secret key
5. キーをコピーして保存

### APIキーの管理方法

#### セキュリティ対策
- ❌ **してはいけないこと**
  - APIキーを他人に教える
  - 公開リポジトリにアップロード
  - メールで送信
  - SNSに投稿

- ✅ **すべきこと**
  - パスワード同様に管理
  - 定期的に再生成
  - 使用状況を監視
  - 不要になったら削除

#### APIキーが漏洩した場合
1. 即座にAPIキーを無効化
2. 新しいAPIキーを生成
3. アプリケーションの設定を更新
4. 使用履歴を確認

### トラブルシューティング

#### Q: APIキーが機能しない
**A:** 以下を確認してください
- コピー時に余分なスペースが入っていないか
- APIキーが有効化されているか
- プロジェクトが正しく選択されているか

#### Q: 利用制限に達した
**A:** Google AI Studioの場合
- 1分間待つ（毎分15回制限）
- 複数のAPIキーを使い分ける
- 処理を分散させる

#### Q: どちらのAPIを選ぶべき？
**A:** 用途による選択
- **Google AI Studio**: 無料で始めたい、クレジットカード登録したくない
- **OpenAI**: GPT-4を使いたい、より高度な機能が必要
