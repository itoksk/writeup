### 【3:00-5:00】デモ（見せ場）

**画面：Google AI Studioの画面を表示**

**講師セリフ：**
「では、実際にやってみましょう。
まず無料で使えるGoogle AI Studioを開きます。

Googleアカウントでログインするだけで、
すぐに使い始められます。

今日の保育の様子を簡単にメモ書きで入力します。」

**画面操作：入力欄にカーソル**

**入力内容（タイピング表示）：**
```
今日は園庭で泥んこ遊び。
Aちゃんは最初躊躇していたが、友達に誘われて参加。
笑顔で泥団子作り。
```

**講師セリフ：**
「このように、箇条書きでメモ程度に入力して、実行ボタンを押すと...」

**実行ボタン押下 → 1秒待機**

**AI出力（徐々に表示）：**
```
本日は園庭で泥んこ遊びを行いました。

Aちゃんは活動開始時には躊躇する様子が見られましたが、
友達からの誘いかけを受けて活動に参加しました。

その後は笑顔で泥団子作りを楽しみ、
友達との関わりの中で主体的に遊ぶ姿が見られました。

泥の感触を楽しみながら、創造的な遊びを展開し、
社会性の発達も確認できた有意義な活動となりました。
```

**講師セリフ：**
「ほら！もう日誌の下書きが完成です。
メモ書きが、きちんとした文章になりました。

しかも、子どもの成長を捉えた教育的な視点も入っています。
あとは、これをコピーして日誌に貼り付けるだけ。」

**画面：コピーボタンをクリック → 「コピーしました」表示**

### 【5:00-6:30】API キー取得方法（重要）

**画面：Google AI Studioの設定画面**

**講師セリフ：**
「さて、次回のDifyで園専用アプリを作るために、
APIキーを取得しておきましょう。

ここが重要なポイントです。
Difyは無料クレジットがありますが、
APIキーを設定しないと継続的に使えません。」

**操作実演：**
1. 左メニュー「Get API key」をクリック
2. 「Create API key in new project」をクリック
3. プロジェクト名を入力（例：「保育日誌アプリ」）
4. APIキーが表示されたらコピー

**講師セリフ：**
「このAPIキーは一度しか表示されません。
必ずコピーして、安全な場所に保存してください。

このGoogle AI StudioのAPIキーのメリットは：
- 完全無料で取得できる
- 毎分最大15回まで無料で使える
- クレジットカード不要

日誌作成には十分な量です。」

## 撮影・編集メモ

### 必要素材
- Google AI Studioの実際の画面録画
- Difyの実際の画面録画
- APIキー取得手順の画面録画
- Before/Afterのイラスト素材
- フローチャート図
- 効果音（クリック音、完了音）

### 編集ポイント
- デモ部分は画面を大きく表示
- APIキー取得部分は特に丁寧に
- 入力時はタイピング音を追加
- AI出力時は「生成中...」のアニメーション
- 重要ポイントは赤枠で強調

### BGM
- 明るく前向きな音楽
- デモ時は音量を下げる

### テロップ
- 重要な数字は大きく表示
- APIキー関連は赤字で強調
- 専門用語には説明を追加
