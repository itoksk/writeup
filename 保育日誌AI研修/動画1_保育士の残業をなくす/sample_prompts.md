# 動画1：保育士の残業をなくす - サンプルプロンプト集

## デモ用入力例

### 基本的な活動記録

#### 例1：園庭遊び
```
今日は園庭で泥んこ遊び。Aちゃんは最初躊躇していたが、友達に誘われて参加。笑顔で泥団子作り。
```

#### 例2：読み聞かせ
```
絵本の読み聞かせ。みんな集中して聞いていた。Bくんが「もう一回！」とリクエスト。
```

#### 例3：給食
```
給食の時間。苦手な野菜も頑張って食べた。Cちゃんは完食して嬉しそう。
```


## 簡易入力パターン

### パターンA：活動のみ
```
[活動名]。[簡単な様子]。
```
**例：** `散歩。花を見つけた。`

### パターンB：個別観察含む
```
[活動名]。[子ども]が[行動]。[感情/反応]。
```
**例：** `プール遊び。Dくんが潜れるようになった。とても嬉しそう。`

### パターンC：複数の出来事
```
[時間帯]は[活動1]。[様子1]。
その後[活動2]。[様子2]。
```
**例：** 
```
午前は園庭遊び。みんな元気いっぱい。
午後は製作活動。集中して取り組んだ。
```

## AIへの基本指示プロンプト

### 初期版（シンプル）
```
保育日誌を作成してください。
今日の活動内容と子どもの様子を記録してください。
```

### 期待される出力形式
```
【今日の活動】
活動内容を具体的に記載

【子どもの様子】
個別の観察記録

【明日への連携】
継続観察ポイント
```

## 個人情報保護のガイドライン

保育日誌作成時は以下のルールを厳守してください：

- 👤 子どもの実名は使用しない（イニシャルでOK）
- 🏠 住所、電話番号などの個人情報は記載しない
- 👨‍👩‍👧 保護者の個人情報も同様に扱う
- 📸 写真を使用する場合は顔が特定できないよう配慮

## 実践のポイント

1. **簡潔な入力**：長文より箇条書きが効果的
2. **具体的な観察**：「楽しそう」より「笑顔で○○した」
3. **成長の記録**：「できるようになった」ことを意識
4. **継続性**：前日からの変化を意識して記録
