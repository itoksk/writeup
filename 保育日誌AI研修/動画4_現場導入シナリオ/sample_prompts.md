## 園内研修用サンプル

### 練習問題1
**入力：**
```
ブロック遊び。協力して家を作った。
```
**課題：** あなたならどんな日誌にしますか？

**模範解答例：**
> 今日のブロック遊びでは、子どもたちが自然に役割分担をしながら、大きな家を作り上げました。「ここは玄関！」「2階も作ろう！」と意見を出し合い、協力する姿に成長を感じました。

### 練習問題2
**入力：**
```
給食。野菜嫌いのGくんが完食。
```
**課題：** 成長を感じる文章にしてください。

**模範解答例：**
> 給食の時間、いつも野菜を残していたGくんが、今日は全部食べることができました！「ピーマンも食べられたよ！」と誇らしげな表情。小さな一歩ですが、大きな成長です。

### 練習問題3
**入力：**
```
雨。室内遊び。退屈そう。
```
**課題：** ポジティブな日誌に変換してください。

**模範解答例：**
> 雨の日の室内遊びは、普段とは違う特別な時間に。新聞紙遊びや室内サーキットで体を動かし、雨音を聞きながらの絵本タイムも楽しみました。雨の日ならではの発見がありました。
