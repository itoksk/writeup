## 運用ルール

### 基本ルール
1. **個人情報保護**
   - 実名は使用しない（イニシャルOK）
   - 住所・電話番号は入力禁止
   - 保護者情報も同様に扱う

2. **利用タイミング**
   - 推奨：午睡中（13:30-14:30）
   - 次善：降園後（17:00-17:30）
   - 随時：活動直後のメモ入力

3. **入力ルール**
   - 1日最低1回は入力
   - 箇条書きでOK
   - 5分以内で入力完了を目標

4. **共有ルール**
   - 良い文章は共有フォルダへ
   - 週1回のベストプラクティス共有
   - 困った時は主任に相談

## スマートフォン設定手順

### iOS（iPhone/iPad）
1. Safariで Dify にアクセス
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 名前を「日誌AI」に変更
5. 「追加」をタップ
6. 設定 → キーボード → 音声入力をON

### Android
1. Chromeで Dify にアクセス
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. 名前を「日誌AI」に変更
5. 「追加」をタップ
6. 設定 → 言語と入力 → 音声入力をON

## トラブルシューティング

### よくある問題と対処法

| 問題 | 原因 | 対処法 |
|------|------|--------|
| ログインできない | パスワード忘れ | パスワードリセット |
| 文章が生成されない | 入力が短すぎる | もう少し詳しく入力 |
| 文章が固い | プロンプト未調整 | 管理者に連絡 |
| 保存できない | ネットワークエラー | Wi-Fi確認 |
| 音声入力できない | 設定OFF | 設定から音声入力ON |
