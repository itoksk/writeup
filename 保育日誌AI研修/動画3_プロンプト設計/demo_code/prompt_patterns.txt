# プロンプトパターン集（動画3デモ用）

## 基本パターンの進化

### パターン1：最小限
```
保育日誌を作成してください。
```

### パターン2：役割追加
```
あなたは保育士です。
保育日誌を作成してください。
```

### パターン3：経験値追加
```
あなたは15年目のベテラン保育士です。
保育日誌を作成してください。
```

### パターン4：対象明確化
```
あなたは15年目のベテラン保育士です。
保護者向けの保育日誌を作成してください。
```

### パターン5：文体指定
```
あなたは15年目のベテラン保育士です。
保護者が読んで安心する温かい文体で、
保育日誌を作成してください。
```

### パターン6：完全版
```
あなたは15年目のベテラン保育士です。
保護者が読んで安心する温かい文体で、
子どもの成長を丁寧に観察した保育日誌を作成してください。

以下の要素を含めてください：
- 子どもの表情や言葉を具体的に
- 活動の教育的意味を簡潔に
- 明日につながる観察ポイント

文末は「〜でした」「〜ます」で統一し、
1段落は3文以内にしてください。
```

## 場面別カスタマイズパターン

### 朝の活動用
```
あなたは明るく元気な保育士です。
朝の活動について、子どもたちの
いきいきとした様子が伝わる日誌を作成してください。
特に、登園時の表情の変化や、
友達との再会の喜びを含めてください。
```

### 給食時間用
```
あなたは食育を大切にする保育士です。
給食時間の様子を、食への興味関心と
成長が感じられる日誌にしてください。
苦手克服のエピソードがあれば
ポジティブに記載してください。
```

### 午睡時用
```
あなたは子どもの生活リズムを大切にする保育士です。
午睡の様子と、その前後の子どもたちの
様子の変化を含めた日誌を作成してください。
休息の大切さが伝わるように。
```

### 自由遊び用
```
あなたは子どもの主体性を尊重する保育士です。
自由遊びでの子どもたちの創造性と
社会性の育ちが見える日誌を作成してください。
遊びの中での学びを見つけてください。
```

## 文体バリエーション

### 親しみやすい文体
```
子どもたちの素敵な一日を
お母さん、お父さんに伝えるつもりで、
親しみやすく温かい日誌を書いてください。
「〜でしたよ」「〜だったんです」など
話しかけるような文体で。
```

### 専門的な文体
```
保育の専門家として、
発達心理学的な視点を含めた
客観的かつ専門的な観察記録を作成してください。
発達段階に応じた評価を含めて。
```

### 簡潔な文体
```
要点を押さえた簡潔な日誌を作成してください。
箇条書きを活用し、
一文は20字以内を目安に。
全体で150字以内。
```

## 含める要素の指定パターン

### 5W1H重視
```
以下の要素を必ず含めてください：
- いつ（時間帯）
- どこで（場所）
- 誰が（参加者）
- 何を（活動内容）
- なぜ（目的・ねらい）
- どのように（方法・様子）
```

### 成長記録重視
```
以下の成長の観点を含めてください：
- できるようになったこと
- 挑戦したこと
- 友達との関わり
- 言葉の発達
- 情緒の安定
```

### エピソード重視
```
以下を含めてください：
- 印象的なエピソード（1つ以上）
- 子どもの具体的な発言
- 表情の変化
- 保育士の気づき
```

## 制約条件のパターン

### 文字数制限
```
必ず200字以内でまとめてください。
改行は2回まで。
```

### 段落構成指定
```
3段落構成で作成してください：
1段落目：活動内容（2文）
2段落目：子どもの様子（3文）
3段落目：まとめと展望（1文）
```

### 使用語彙制限
```
保護者が読みやすいよう、
専門用語は使わず、
日常的な言葉で表現してください。
```

## 園の特色反映パターン

### モンテッソーリ園
```
モンテッソーリ教育の理念に基づき、
子どもの「敏感期」と「集中現象」に
注目した観察記録を含めてください。
```

### 自然保育園
```
自然との関わりを重視し、
季節の変化や自然物への興味、
五感を使った体験を中心に記載してください。
```

### 英語教育園
```
英語活動での様子を含め、
言語習得の過程や、
異文化への興味を記載してください。
```

## テスト用比較セット

### 入力（全パターン共通）
```
プール遊び。水が怖いCちゃんも参加。
```

### 基本パターンでの出力例
```
プール遊びを行いました。
Cちゃんも参加しました。
```

### 完全パターンでの出力例
```
今日は待ちに待ったプール遊びの日！
水が苦手なCちゃんも、友達の楽しそうな声に誘われて、
そーっと足から水に入ってみました。

「つめたいけど、きもちいい！」という言葉が聞けて、
水に慣れる大切な一歩となりました。

明日も少しずつ水遊びの楽しさを
感じられるよう支援していきたいと思います。
```

## 改善のヒント

### より具体的に
Before: 「楽しく活動しました」
After: 「『もっとやりたい！』という声が上がるほど夢中でした」

### より温かく
Before: 「参加した」
After: 「勇気を出して挑戦しました」

### より教育的に
Before: 「遊んだ」
After: 「遊びを通して協調性と創造性が育まれました」
