# 動画3：日誌文章を自然に仕上げるプロンプト設計と改善方法

## 講師用台本（全10分）

---

### 【0:00-0:30】導入フレーズ

**画面：Before/Afterの文章比較**

**講師セリフ：**
「AIの文章が『なんか機械的...』と感じたことありませんか？

『活動を実施しました』みたいな固い文章...

実はプロンプトの書き方を少し変えるだけで、
ベテラン保育士が書いたような温かい文章に変わるんです。」

---

### 【0:30-2:00】Before/After

**画面：同じ入力で異なる出力の比較**

**入力表示：**
```
散歩に行った。花を見つけた。
```

**Before（機械的な文章）：**
```
本日は散歩を実施しました。
児童は花を発見しました。
天候は晴天でした。
活動は終了しました。
```

**講師セリフ：**
「これが今の状態だとしたら...」

**After（自然な文章）：**
```
今日はぽかぽか陽気の中、みんなで近くの公園までお散歩に出かけました。

道中では「あ！たんぽぽ見つけた！」「こっちにもお花があるよ！」と、
春の訪れを感じる発見がたくさん。

子どもたちの好奇心あふれる姿が印象的な一日でした。
```

**講師セリフ：**
「こんなに変わります！
この違い、プロンプトの工夫だけで実現できるんです。」

---

### 【2:00-5:00】デモ（見せ場）

**画面：Difyの編集画面**

**講師セリフ：**
「では、魔法の言葉を追加していきましょう。
ポイントは3つです。」

**画面：改善前のプロンプト表示**

```
保育日誌を作成してください。
```

**講師セリフ：**
「これだけでは、AIは何をすればいいか分かりません。
では、改善していきます。」

**画面：改善ポイント1を追加（赤字で強調）**

```
あなたは15年目のベテラン保育士です。
保育日誌を作成してください。
```

**講師セリフ：**
「まず、AIに『役割』を与えます。
15年目のベテラン保育士という設定にしました。」

**画面：改善ポイント2を追加**

```
あなたは15年目のベテラン保育士です。
保護者が読んで安心する温かい文体で、
保育日誌を作成してください。
```

**講師セリフ：**
「次に、『文体』を指定します。
保護者が安心する温かい文体、と具体的に。」

**画面：改善ポイント3を追加**

```
あなたは15年目のベテラン保育士です。
保護者が読んで安心する温かい文体で、
子どもの成長を丁寧に観察した保育日誌を作成してください。

以下の要素を含めてください：
- 子どもの表情や言葉を具体的に
- 活動の教育的意味を簡潔に
- 明日につながる観察ポイント

文末は「〜でした」「〜ます」で統一し、
1段落は3文以内にしてください。
```

**講師セリフ：**
「最後に、『構造』を明確にします。
含める要素と、文章のルールを指定しました。」

---

### 【5:00-6:30】構築手順

**画面：改善の3ステップ**

**講師セリフ：**
「プロンプト改善は、この3ステップで行います。」

**チェックリスト表示（順番にチェック）：**
```
□ 役割設定（誰として書くか）
  → ベテラン保育士として

□ 文体指定（どんな雰囲気で）
  → 温かく、具体的に

□ 構造化（何を含めるか）
  → 観察ポイントを明確に
```

**講師セリフ：**
「この3つを意識すれば、必ず良い文章になります。」

---

### 【6:30-9:00】設定実演

**画面：実際の比較テスト**

**講師セリフ：**
「同じ入力で、改善前後を比較してみましょう。」

**テスト入力：**
```
プール遊び。水が怖いCちゃんも参加。
```

**改善前の出力：**
```
プール遊びを行いました。
Cちゃんも参加しました。
終了しました。
```

**講師セリフ：**
「味気ないですね...では、改善後は？」

**改善後の出力：**
```
今日は待ちに待ったプール遊びの日！

水が苦手なCちゃんも、友達の楽しそうな声に誘われて、
そーっと足から水に入ってみました。

「つめたいけど、きもちいい！」という言葉が聞けて、
水に慣れる大切な一歩となりました。

明日も少しずつ水遊びの楽しさを
感じられるよう支援していきたいと思います。
```

**講師セリフ：**
「すごい違いですよね！
Cちゃんの成長が、生き生きと伝わってきます。」

**画面：さらなる工夫の例**

**講師セリフ：**
「用途によって、プロンプトを使い分けることもできます。」

**保護者向けプロンプト：**
```
保護者が読んで安心し、
子どもの成長を感じられる文章で
```

**記録用プロンプト：**
```
客観的な事実を中心に、
簡潔で要点をまとめた文章で
```

**講師セリフ：**
「このように、目的に応じて調整できます。」

---

### 【9:00-9:30】注意点・活用ポイント

**画面：注意事項スライド**

**講師セリフ：**
「プロンプト設計の注意点です。」

**箇条書き表示：**
- 園の方針に合わせて調整
- 定期的に見直しと改善
- 職員みんなで良い例を共有
- やりすぎ注意（自然さを大切に）

**講師セリフ：**
「大切なのは、園らしさを残すこと。
全員が同じ文章にならないよう、最後は人の手で調整しましょう。」

---

### 【9:30-10:00】まとめ・次回予告

**画面：改善のポイント総まとめ**

**ポイント表示：**
```
プロンプト改善の魔法
1. 役割を与える（ベテラン保育士）
2. 文体を指定する（温かく具体的に）
3. 構造を明確にする（含める要素）
```

**講師セリフ：**
「プロンプトの工夫で、AIの文章が見違えるほど自然になりました。

でも、『作ったはいいけど、現場でどう使うの？』
という疑問があるかもしれません。

次回は、このAIを日常業務にどう組み込むか、
現場での活用シナリオを詳しく紹介します。

朝の忙しい時間でも、午睡中でも、
スマホからサッと使える方法もお見せしますよ。

それでは、次回もお楽しみに！」

## 撮影・編集メモ

### 必要素材
- プロンプト編集画面の録画
- Before/After比較のアニメーション
- 文章生成の様子（リアルタイム）

### 編集ポイント
- プロンプト追加部分は赤字で強調
- 比較は左右分割画面
- 重要な単語はテロップで強調

### BGM
- 思考を促す落ち着いた音楽
- 成功時は明るい効果音

### 補足資料
- プロンプトテンプレート集を別途用意
- 園ごとのカスタマイズ例を収録
