# 動画3：プロンプト設計 - サンプルプロンプト集

## プロンプト改善の5段階

### レベル1：最小限のプロンプト
```
保育日誌を作成してください。
```
**問題点：** 指示が曖昧で、出力品質が安定しない

### レベル2：役割を追加
```
あなたは保育士です。
保育日誌を作成してください。
```
**改善点：** AIの立場が明確になった

### レベル3：経験値を追加
```
あなたは15年目のベテラン保育士です。
保育日誌を作成してください。
```
**改善点：** より専門的な視点が期待できる

### レベル4：文体を指定
```
あなたは15年目のベテラン保育士です。
保護者が読んで安心する温かい文体で、
保育日誌を作成してください。
```
**改善点：** 読み手と文体が明確になった

### レベル5：完成形
```
あなたは15年目のベテラン保育士です。
保護者が読んで安心する温かい文体で、
子どもの成長を丁寧に観察した保育日誌を作成してください。

以下の要素を含めてください：
- 子どもの表情や言葉を具体的に
- 活動の教育的意味を簡潔に
- 明日につながる観察ポイント

文末は「〜でした」「〜ます」で統一し、
1段落は3文以内にしてください。
```
**完成：** 具体的で実用的なプロンプト

## 用途別プロンプトテンプレート

### 📝 保護者向け日誌
```
あなたは子どもの成長を温かく見守る保育士です。
保護者が読んで我が子の園での様子が目に浮かび、
安心できる日誌を作成してください。

特に以下を意識してください：
- 子どもの笑顔や楽しそうな様子
- 友達とのポジティブな関わり
- 小さな成長や発見
- 「〜できました」という達成感
```

### 📊 記録用日誌（簡潔版）
```
保育記録として簡潔に本日の活動をまとめてください。
客観的事実を中心に、150字以内で記載。
時系列で活動内容と特記事項のみ。
```

### 📅 週報用まとめ
```
今週の保育活動を振り返り、以下の形式でまとめてください：
1. 今週のテーマと主な活動
2. 子どもたちの成長ポイント（3点）
3. 来週への展開と準備事項

全体で300字程度、ポジティブな総括を。
```

### 👶 個別観察記録用
```
特定の子どもの観察記録を作成してください。
以下の観点から記述：
- 活動への参加度
- 友達との関わり
- 言語表現の発達
- 情緒の安定
- 特筆すべき行動や発言

発達心理学的な視点を含め、200字程度で。
```

## 効果的なキーワード集

### 🌟 雰囲気を決めるキーワード
| キーワード | 効果 |
|------------|------|
| 温かい雰囲気で | 親しみやすい文章になる |
| 子ども目線で | 子どもの気持ちが伝わる |
| 共感的に | 保護者の共感を得やすい |
| 優しい眼差しで | 愛情深い記述になる |

### 🔍 具体性を高めるキーワード
| キーワード | 効果 |
|------------|------|
| 具体的なエピソードを含めて | 場面が想像しやすくなる |
| 子どもの言葉を引用して | リアリティが増す |
| 表情の変化を描写して | 感情が伝わる |
| 五感を使った表現で | 臨場感が出る |

### 🎓 教育的視点のキーワード
| キーワード | 効果 |
|------------|------|
| 発達段階を考慮して | 専門的な視点が加わる |
| 学びのポイントを含めて | 教育的価値が明確になる |
| 成長の過程として | 継続的な視点を持てる |
| 主体性を大切に | 子ども中心の記述になる |

## 比較実験：同じ入力で異なる出力

### 入力例
```
プール遊び。水が怖いCちゃんも参加。
```

### 基本プロンプトの出力
> プール遊びを行いました。Cちゃんも参加しました。

### 改善プロンプトの出力
> 今日は待ちに待ったプール遊びの日！水が苦手なCちゃんも、友達の楽しそうな声に誘われて、そーっと足から水に入ってみました。「つめたいけど、きもちいい！」という言葉が聞けて、水に慣れる大切な一歩となりました。

## 園の特色別カスタマイズ

```
あなたはモンテッソーリ教育を実践する保育士です。
子どもの自主性と集中力を大切にし、
「敏感期」を意識した観察記録を含めて

```

### 🌲 自然保育重視の園
```
あなたは自然体験を大切にする森のようちえんの保育士です。
季節の変化、自然との触れ合い、
五感を使った体験を中心に
日誌を作成してください。
```

### 🤝 インクルーシブ保育の園
```
あなたは多様性を大切にするインクルーシブ保育の実践者です。
一人ひとりの個性を尊重し、
みんなが共に育ち合う様子を
温かく記録してください。
```

## プロンプト改善チェックリスト

プロンプトを作成したら、以下の項目をチェック：

- [ ] **役割**（誰として）が明確か
- [ ] **対象**（誰に向けて）が明確か
- [ ] **文体**（どんな雰囲気で）が指定されているか
- [ ] **構成**（何を含めるか）が示されているか
- [ ] **制約**（文字数など）が設定されているか
- [ ] **園の特色**が反映されているか
- [ ] **テスト**して比較したか

## 実践演習

### 演習1：基本から改善へ
1. まず最小限のプロンプトで生成
2. 段階的に要素を追加
3. 出力の違いを比較

### 演習2：園の特色を反映
1. 自園の教育方針を整理
2. プロンプトに組み込み
3. 実際の日誌と比較

### 演習3：用途別使い分け
1. 同じ活動を異なる用途で記述
2. 保護者向け、記録用、週報用を作成
3. それぞれの特徴を確認
