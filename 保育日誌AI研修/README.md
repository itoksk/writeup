# 保育日誌作成支援AI 活用研修カリキュラム

## 研修概要
保育士の日誌作成業務を効率化し、子どもと向き合う時間を増やすためのAI活用研修です。

## 研修目標
- 日誌作成時間を30分から5分に短縮（85%削減）
- 月間残業時間を15時間から3時間に削減（80%削減）
- 文章の質を均一化し、保護者への伝達力を向上

## カリキュラム構成
全4回（各10分）の動画研修

### 動画1：保育士の残業をなくす！1日の振り返りをAIが手助け！
AIによる日誌作成支援の全体像と効果を理解

### 動画2：Difyで作る！保育日誌チャットボットの基本設定
実際にDifyでチャットボットを構築

### 動画3：日誌文章を自然に仕上げるプロンプト設計と改善方法
園らしい温かい文章を生成するプロンプトの工夫

### 動画4：現場導入シナリオと活用のコツ：日々の保育がもっと楽になる
日常業務への組み込みと園内展開の実践方法

## 必要環境
- インターネット接続環境
- PC またはタブレット
- Difyアカウント（無料版で開始可能）

## 対象者
- 保育士・幼稚園教諭
- 園長・主任保育士
- ICT担当職員

## 研修時間
- 動画視聴：40分（10分×4本）
- 実習時間：約1時間
- 合計：約2時間

## サポート
- 月1回のフォローアップセッション
- プロンプト改善ワークショップ
- 成功事例共有会
