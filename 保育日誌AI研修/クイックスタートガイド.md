## このガイドは誰のため？
- **今すぐ**AIを使いたい方

---

## 🏃 たった5ステップで完成！

### STEP 1️⃣：Difyにアクセス（30秒）

```
🔗 https://cloud.dify.ai
```
↑このリンクをクリック

Googleアカウントでログイン
（Gmailを持っていればOK）

---

### STEP 2️⃣：アプリ作成（1分）

1. 上部の **「スタジオ」** をクリック
2. **「アプリを作成」** → **「最初から作成」**
3. **「チャットボット」** を選択
4. **「基本」** を選択
5. 名前に **「保育日誌AI」** と入力
6. **「作成」** をクリック

---

### STEP 3️⃣：魔法のプロンプト（2分）

左側の **「指示」** をクリックして、以下をコピペ：

```
あなたは15年のベテラン保育士です。
簡単なメモから温かい保育日誌を作成してください。

【今日の活動】活動内容と子どもの様子（2-3文）
【子どもの様子】個別観察とエピソード（2-3文）  
【明日への連携】次への展開（1文）

200字程度、温かい文体で。個人名はイニシャルで。
```

**「保存」** をクリック

---

### STEP 4️⃣：テスト（30秒）

右側の入力欄に以下を入力してEnter：

```
プール遊び。みんな楽しそう。Aちゃんも水慣れた。
```

AIが日誌を作成してくれます！

---

### STEP 5️⃣：公開（1分）

1. 右上の **「公開」** をクリック
2. **「Webアプリとして実行」** を選択
3. **「アプリを実行」** をクリック

完成！URLをブックマーク！

## 📱 スマホで使う方法（追加1分）

### iPhone
1. SafariでURLを開く
2. 共有ボタン → ホーム画面に追加

### Android  
1. ChromeでURLを開く
2. メニュー → ホーム画面に追加

## 💡 使い方のコツ

### 良い入力例
```
✅ 「外遊び。砂場で協力して山作り。」
✅ 「給食完食。苦手な野菜も頑張った。」
✅ 「絵本。みんな集中。『もう一回』の声。」
```

### 悪い入力例
```
❌ 「遊んだ」（情報不足）
❌ 「田中太郎くんが...」（実名NG）
```

---

## 🆘 困ったときは

| 問題 | 解決方法 |
|------|----------|
| ログインできない | Cookieを有効にする |
| 日本語にならない | Settings → Language → 日本語 |
| 文章が生成されない | プロンプトを再度コピペ |
| エラーが出る | ページを更新（F5キー） |
