# 🚀 Dify完全マスターガイド - 保育日誌AI作成の全手順

## 📌 はじめに - Difyとは？

**Dify（ディファイ）**は、プログラミング知識ゼロでもAIアプリケーションを作成できる画期的なプラットフォームです。

### なぜDifyが最適なのか？
- ✅ **完全無料**で始められる（200メッセージまで）
- ✅ **10分**でチャットボット完成
- ✅ **日本語完全対応**
- ✅ **ドラッグ&ドロップ**の簡単操作
- ✅ **OpenAI/Claude/Gemini**など複数AI対応

## 📱 STEP 1: Difyアカウント作成（3分）

### 1-1. Dify公式サイトにアクセス

🔗 **URL**: https://cloud.dify.ai

> 💡 **重要**: 必ず `cloud.dify.ai` にアクセスしてください。これがクラウド版です。

### 1-2. アカウント作成手順

1. **右上の「始める」ボタンをクリック**
   ```
   画面位置：右上の青いボタン
   表示テキスト：「始める」または「Get Started」
   ```

2. **ログイン方法を選択**
   - 🔵 **Googleアカウントで続行**（推奨）
   - ⚫ **GitHubアカウントで続行**
   - 📧 **メールアドレスで登録**

3. **Googleアカウントでの登録手順**（最も簡単）
   ```
   1. 「Continue with Google」をクリック
   2. Googleアカウントを選択
   3. 「許可」をクリック
   4. 自動的にDifyに戻る
   ```

### 1-3. 初期設定

1. **言語を日本語に変更**
   ```
   場所：右上のプロフィールアイコン → Settings
   Language: Japanese（日本語）を選択
   ```

2. **ワークスペース名を設定**
   ```
   例：「○○保育園AI」
   ```

## ⚙️ STEP 3: AIモデル設定（3分）

### 3-1. モデルプロバイダー設定

1. **設定画面を開く**
   ```
   場所：右上のプロフィール → 設定 → モデルプロバイダー
   ```

2. **無料で使えるモデル（初心者向け）**
   ```
   モデル：GPT-3.5-turbo
   特徴：無料で200メッセージまで利用可能
   設定：特に不要（デフォルトで利用可能）
   ```

### 3-2. OpenAI APIキーの設定（オプション）

> 💡 **注意**: 最初は無料枠で十分です。必要になってから設定しましょう。

1. **OpenAI APIキーの取得**
   ```
   1. https://platform.openai.com にアクセス
   2. サインアップ/ログイン
   3. 左メニュー「API keys」をクリック
   4. 「Create new secret key」をクリック
   5. キーをコピー（一度しか表示されません！）
   ```

2. **DifyにAPIキーを設定**
   ```
   1. Difyの設定 → モデルプロバイダー
   2. OpenAIの「セットアップ」をクリック
   3. APIキーを貼り付け
   4. 「保存」をクリック
   ```

## 📝 STEP 4: プロンプト設定（超重要！）

### 4-1. システムプロンプトの設定場所

1. **編集画面の「指示」セクションを探す**
   ```
   場所：アプリ編集画面の左側
   アイコン：💭 のようなマーク
   ラベル：「指示」または「Instructions」
   ```

2. **プロンプト入力エリアをクリック**

### 4-2. 最強のプロンプトをコピー＆ペースト

```markdown
あなたは15年以上の経験を持つベテラン保育士です。
保護者が読んで安心し、子どもの成長を感じられる温かい保育日誌を作成してください。

## 入力形式
簡単なメモや箇条書きから、完成した日誌を作成します。

## 出力形式
以下の構成で日誌を作成してください：

### 【今日の活動】
- 活動内容を2〜3文で具体的に記載
- 子どもたちの反応や楽しんでいる様子を含める
- 「〜でした」「〜ました」の丁寧語で統一

### 【子どもの様子】
- 個別の観察記録を2〜3点含める
- 成長や変化、頑張りを具体的に記載
- 子どもの言葉は「」で引用
- ポジティブで温かい表現を使用

### 【明日への連携】
- 継続観察のポイントを1文で
- 次の活動への期待感を含める

## 重要なルール
1. 文字数：全体で200〜300字程度
2. 文体：保護者が安心する温かい表現
3. 視点：子どもの成長と喜びに焦点
4. 個人名：イニシャルで表記（例：Aちゃん、Bくん）
5. 専門用語は使わず、分かりやすい言葉で

## 入力例
「プール遊び。水が怖いCちゃんも参加。」

## 出力例
【今日の活動】
今日は待ちに待ったプール遊びを楽しみました。水しぶきを上げながら、みんな笑顔いっぱいで遊んでいました。

【子どもの様子】
水が苦手だったCちゃんも、お友達に励まされて勇気を出して参加しました。「冷たいけど気持ちいい！」と初めてプールの楽しさを感じることができました。他の子どもたちも「Cちゃん、すごいね！」と温かく見守る姿が印象的でした。

【明日への連携】
明日も水遊びを予定しているので、今日の経験を活かしてさらに楽しめるよう支援していきます。
```

### 4-3. プロンプトの保存

1. **「保存」ボタンをクリック**

## 🧪 STEP 5: テストと調整（2分）

### 5-1. テスト実行

1. **右側のプレビュー画面を使用**
   ```
   場所：画面右側の「プレビュー」エリア
   ```

2. **テスト入力例**
   ```
   入力1：「外遊び。砂場で山作り。」
   入力2：「給食完食。野菜も食べた。」
   入力3：「絵本読み聞かせ。集中して聞いていた。」
   ```

3. **出力を確認**
   - ✅ 温かい文章になっているか
   - ✅ 具体的な描写があるか
   - ✅ 保護者が安心する内容か

### 5-2. 微調整方法

1. **文章が固い場合**
   ```
   プロンプトに追加：
   「親しみやすく、まるで保育士が直接話しかけているような文体で」
   ```

2. **文章が長すぎる場合**
   ```
   プロンプトに追加：
   「全体で150〜200字程度にまとめてください」
   ```

3. **具体性が足りない場合**
   ```
   プロンプトに追加：
   「子どもの表情、動作、発言を必ず含めてください」
   ```

## 🆘 トラブルシューティング

### よくある問題と解決法

| 問題 | 原因 | 解決方法 |
|------|------|----------|
| ログインできない | Cookie無効 | ブラウザのCookieを有効化 |
| 日本語が文字化け | 文字コード | UTF-8に設定 |
| 生成が遅い | 無料プラン制限 | 有料プランを検討 |
| 文章が不自然 | プロンプト不足 | プロンプトを詳細化 |
| エラー429 | レート制限 | 1分待ってから再試行 |
