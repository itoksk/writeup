## 📄 ファイル説明

**用途**: メインデモ用の高品質注文書
**品質スコア予想**: 85-95点（緑色表示）

**特徴**:
- 完全な企業情報（会社名、担当者、連絡先）
- 詳細な商品明細（品番、商品名、数量、単価）
- 正確な金額計算（小計、消費税、合計）
- 明確な納期と配送情報
- 特記事項と承認印

**AIが抽出する情報**:

- 主要商品: ビジネスノートPC
- 合計数量: 125個（25台×5種類）
- 合計金額: 4,309,250円
- 納期: 2024年07月25日

### 2. 品質スコア検証用_異なる注文書.txt
**用途**: 品質スコア別色分けのデモ用
**品質スコア予想**: 45-55点（赤色表示）

**特徴**:
- 簡素な手書きメモ風
- 情報不足（住所、詳細連絡先なし）
- 商品詳細不明確
- 金額の詳細計算なし

## 🎬 デモ実演手順

### ステップ1: 高品質注文書のデモ
1. `デモ用サンプル注文書.txt` をPDF化
2. Google Driveの「AI受注処理」フォルダに投入
3. `demo6_CompleteAutomation()` を実行
4. スプレッドシートに緑色（高品質）で記録されることを確認

### ステップ2: 品質スコア比較デモ
1. `品質スコア検証用_異なる注文書.txt` をPDF化
2. 同様に処理実行
3. スプレッドシートに赤色（低品質）で記録されることを確認
4. 品質スコアの違いによる色分けの効果を実演

### ステップ3: Before/After効果の実証
1. 手作業での転記時間を測定（想定15分）
2. 自動化システムでの処理時間を測定（実際1分未満）
3. 処理精度の比較（手作業ミス vs 自動化完璧）
4. データ活用可能性の説明

## 📊 期待される結果

```

色分け: 緑色（確認済み）
```
