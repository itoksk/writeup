# 動画1: なぜあなたの時間は"PDFの転記作業"に消えるのか？

## 📝 セリフと流れ（10分構成）

### 🎬 オープニング（0:00-1:30）
**セリフ**:
「田中さん（仮名）は、毎朝9時に出社すると、まず受信トレイを開きます。昨夜から今朝にかけて届いた12件の注文書PDF。1つ目を開いて、会社名『株式会社○○商事』、担当者『山田太郎様』、商品名『A型パーツ 50個』…1文字ずつ、慎重にExcelに入力していきます。

1件目完了まで17分。『あと11件か…』とため息をつく田中さん。気がつけば昼休み。午後にはさらに8件追加。結局、1日の大半がこの転記作業で終わってしまいます。

あなたも、田中さんと同じ状況ではありませんか？

このコースは、そんなあなたの人生を変えます。15分の作業を1分に短縮し、あなたの時間を『本当に価値のある仕事』に取り戻すための、実践的なガイドです。

約束します。8本の動画を完走したあなたは、もう二度と単純な転記作業に時間を奪われることはありません。」

**画面表示**: タイトル + Before/After の数値比較グラフ

### 🔍 現状の課題提示（1:30-3:30）
**セリフ**:
「先ほどの田中さんの1日を、数字で見てみましょう。

朝の12件：12件 × 15分 = 3時間
午後の8件：8件 × 15分 = 2時間
合計：1日5時間が純粋な転記作業

しかし、これだけではありません。
・転記ミスの確認：30分
・データの整合性チェック：20分  
・上司への進捗報告：10分
・システムへの再入力（重複作業）：30分

実際には1日6時間30分。8時間勤務のうち、なんと81%が機械的な作業です。

田中さんは入社時、『お客様との関係を深め、売上向上に貢献したい』と志していました。しかし現実は…朝から晩まで、ただ文字を写すだけの日々。『これが自分のやりたかった仕事だろうか？』そんな疑問を抱えながら、今日も転記作業を続けています。

田中さんの時給を2,000円とすると、1日13,000円、月26万円分の時間が、単純作業で消えているのです。」

### ✨ 理想の状態（After）（3:30-5:30）
**セリフ**:
「では、システム導入後の田中さんの1日を見てみましょう。

朝9時、同じように20件のPDFが届きます。田中さんがやることは、それらを『AI受注処理』フォルダにドラッグ&ドロップするだけ。20件でわずか2分の作業です。

その後、AIが動き出します：
・Vision APIが全PDFの文字を瞬時に認識
・ChatGPTが必要な情報だけを整理して抽出
・スプレッドシートに構造化されたデータが自動記録
・Slackで『新規受注20件処理完了』の通知
・各お客様に感謝メールが自動送信

全処理が完了するまで、わずか18分。田中さんはコーヒーを飲んでいる間に、1日分の転記作業が終わっています。

創出された6時間で、田中さんは何をしているでしょうか？

・午前：新規顧客への提案資料作成
・午後：既存顧客との関係深化のための訪問
・夕方：来月の販売戦略会議への準備

そして18時、定時で帰宅。『今日も充実した1日だった』と満足げに家路につく田中さん。これが、私たちが目指す未来です。」

- 完成システムのデモ（高速再生）

### 🛠️ システム全体像（5:30-7:30）
**セリフ**:
「このシステムの美しさは、そのシンプルさにあります。

5つのステップ：
1. **PDF投入**：ドラッグ&ドロップで完了
2. **OCR処理**：Google Vision APIが瞬時に文字認識
3. **AI抽出**：ChatGPTが人間レベルの理解で情報整理
4. **データベース保存**：スプレッドシートに構造化して記録
5. **自動通知**：Slack + Gmail で完璧な連携

使用技術の90%は無料のGoogleサービス。Google Apps Script、Drive、Gmail、スプレッドシート。有料なのはVision APIとChatGPT APIのみで、月額約1,500円。

コスト効果を計算してみましょう：
・田中さんの時給：2,000円
・短縮時間：1日6時間 = 月120時間
・月間効果：120時間 × 2,000円 = 24万円
・システム費用：月1,500円
・純利益：238,500円/月
・投資回収率：約15,900%

1年間で考えると、286万円の価値創出です。これは田中さん個人の話。チーム全体に導入すれば、効果は何倍にもなります。」
