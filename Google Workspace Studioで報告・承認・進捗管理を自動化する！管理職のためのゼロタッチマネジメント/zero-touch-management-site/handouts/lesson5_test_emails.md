# Lesson 5 テスト用サンプルメール

承認依頼エージェントをテストするためのサンプルメールです。以下の文面をメールとして自分宛に送信し、エージェントの動作をテストしてください。

---

## サンプル1: 経費申請（5万円未満）→ 直属上長承認

**件名:** 【経費申請】交通費精算のお願い

```
経費申請書

申請者: 山田太郎
部署: 営業部
申請日: 2024年1月15日

■ 申請内容
出張交通費の精算をお願いいたします。

■ 明細
・1/10 東京→大阪（新幹線）: 13,870円
・1/10 大阪市内タクシー: 2,340円
・1/11 大阪→東京（新幹線）: 13,870円

■ 合計金額
30,080円

■ 出張目的
A社との商談（新規案件提案）

■ 備考
領収書は別途提出済みです。

よろしくお願いいたします。
```

**期待される判定:**
- 申請種類: 経費申請
- 金額: 30,080円（5万円未満）
- 承認者: 直属上長
- リマインド: 48時間後
- エスカレーション: 72時間後

## テスト手順

1. 上記のサンプルメールを自分のGmailアドレス宛に送信
2. 件名に「申請」「稟議」「経費」「休暇」のいずれかが含まれていることを確認
3. エージェントが起動し、Chat DMで通知が届くことを確認
4. Geminiの判定結果（申請種類、金額、承認者など）が正しいか確認
5. スプレッドシートに記録されていることを確認
