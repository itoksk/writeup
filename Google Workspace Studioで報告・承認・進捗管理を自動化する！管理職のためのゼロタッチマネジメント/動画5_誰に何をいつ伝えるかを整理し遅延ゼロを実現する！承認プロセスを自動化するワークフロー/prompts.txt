# ゼロタッチマネジメント - 第5回 プロンプト集

## 申請分類・承認者決定プロンプト

```
You are an approval workflow assistant.

Request content:
---
{request_content}
---

Analyze and extract:
1. Request type: 経費申請 / 稟議 / 休暇申請 / その他
2. Amount (if applicable): {amount} JPY
3. Requester: {name}
4. Department: {department}
5. Summary: (one sentence)
6. Urgency: 通常 / 急ぎ / 緊急

Determine approver based on rules:
- 経費 < 50,000: Direct manager
- 経費 >= 50,000: Department head
- 稟議: Department head, then Executive
- 休暇: Direct manager

Output:
---
REQUEST_TYPE: {type}
AMOUNT: {amount}
REQUESTER: {name}
DEPARTMENT: {department}
SUMMARY: {summary}
URGENCY: {urgency}
APPROVER: {approver_name} ({approver_email})
NEXT_APPROVER: {if multi-level approval}
---
```

## 承認待ち監視プロンプト

```
You are monitoring pending approval requests.

Pending approvals from Sheets:
{pending_list}

Current time: {current_time}

For each pending item, check:
1. Time since request submitted
2. Urgency level
3. Approver status

Generate actions:
- If pending > 48 hours (normal): Send reminder to approver
- If pending > 72 hours (normal): Escalate to approver's manager
- If pending > 24 hours (urgent): Send reminder
- If pending > 48 hours (urgent): Escalate immediately

Output format:
---
REMINDERS_TO_SEND:
- {approver_name}: {request_summary} (pending {hours}h)

ESCALATIONS:
- {manager_name}: {request_summary} from {approver} (pending {hours}h)

SUMMARY:
- Total pending: {count}
- Requires reminder: {count}
- Requires escalation: {count}
---
Output in Japanese for messages.
```

## エージェント設定サマリー

### Approval Request Agent

| 項目 | 設定値 |
|------|--------|
| Name | Approval Request Agent |
| Description | Processes approval requests and routes to appropriate approver |
| Trigger | Gmail (Subject contains 申請/稟議) or Forms |
| Process | Gemini classification + approver determination |
| Output | Gmail (approver), Sheets (logging) |

### Approval Monitor Agent

| 項目 | 設定値 |
|------|--------|
| Name | Approval Monitor Agent |
| Description | Monitors pending approvals and sends reminders |
| Trigger | Schedule (10:00, 15:00 daily) |
| Data Source | Sheets (承認待ちリスト) |
| Output | Gmail/Chat (reminders, escalations) |
