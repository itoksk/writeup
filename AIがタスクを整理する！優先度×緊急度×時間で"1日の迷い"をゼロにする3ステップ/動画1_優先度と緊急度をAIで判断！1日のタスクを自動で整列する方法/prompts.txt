# デモ用プロンプト集 - 動画1
# 優先度と緊急度をAIで判断！1日のタスクを自動で整列する方法

---

## 🔰 初心者向け：まずはこれだけ！（最小限プロンプト）

**使い方（3ステップ）:**
1. 下のプロンプトをすべてコピー
2. ChatGPTに貼り付け
3. 【ここを編集】の部分だけ、あなたのタスクに書き換えて送信

```
今日やるべきタスクを、優先順位が高い順に並べてください。
理由も簡単に教えてください。

【ここを編集：あなたのタスクを書く】
- 見積書の送付
- 週報の作成
- 新規顧客への初回メール
- チームミーティングの準備
- 上司への報告資料修正
```

**💡 ポイント:**
- 「【ここを編集】」の下の5つのタスクを、あなたの今日のタスクに書き換えるだけでOK
- タスクは3個でも10個でもOK
- 「-」（ハイフン）を忘れずに

## プロンプト1: タスクの優先度・緊急度判定（基本版）

**使い方:**
1. 下のプロンプトをすべてコピー
2. ChatGPTに貼り付け
3. 「タスク一覧：」の下を、あなたのタスクに書き換え
4. 送信ボタンを押す

```
あなたは私の業務アシスタントです。
以下のタスクを「重要度」と「緊急度」で4象限マトリクスに分類してください。
それぞれの理由も簡潔に書き添えてください。
出力形式は表形式でお願いします。

タスク一覧：
【ここを編集：あなたのタスクを書く】
- 見積書の送付
- 週報の作成
- 新規顧客への初回メール
- チームミーティングの準備
- 上司への報告資料修正
```

### 想定出力例

ChatGPTはこんな風に答えてくれます：

| 緊急度 | 重要度 | タスク | 理由 |
|-------|-------|--------|------|
| 高 | 高 | 上司への報告資料修正 | 本日締切・業務評価に直結 |
| 高 | 低 | 見積書の送付 | 顧客対応期限が今日中 |
| 低 | 高 | 新規顧客への初回メール | 今後の関係性構築に重要 |
| 低 | 低 | チームミーティング準備 | 優先度低・期日も先 |

## プロンプト2: 優先順位の自動整列

**使い方:**
1. プロンプト1を実行した後、続けて下のプロンプトをコピペ
2. 送信ボタンを押す

```
上記の分類結果を基に、今日やるべきタスクを優先順に並び替えて、
実行順序を番号付きリストで提案してください。
各タスクに推奨時間帯（午前/午後）も添えてください。
```

### 想定出力例

ChatGPTはこんな風に答えてくれます：

**今日やるべきタスク（優先順）**

1. 上司への報告資料修正（午前：10:00-11:00）
2. 見積書の送付（午前：11:00-11:30）
3. 新規顧客への初回メール（午後：14:00-14:30）
4. 週報の作成（午後：16:00-16:30）

**明日以降に回すタスク**
- チームミーティング準備（期日に余裕あり）

## 📝 ステップバイステップガイド（初めての方へ）

### ステップ1：ChatGPTを開く

1. ブラウザ（Chrome、Safariなど）を開く
2. アドレスバーに「chatgpt.com」と入力してEnterを押す
3. ログインしていない場合は、ログインする
   - Googleアカウントでログインできます
   - 無料版で大丈夫です

### ステップ2：プロンプトをコピーする

1. この資料の「🔰 初心者向け」セクションのプロンプトをコピー
   - マウスでドラッグして全選択
   - Ctrl+C（Windows）または Command+C（Mac）でコピー

### ステップ3：ChatGPTに貼り付ける

1. ChatGPTの画面下部にある入力欄をクリック
2. Ctrl+V（Windows）または Command+V（Mac）で貼り付け

### ステップ4：タスクを編集する

1. 「【ここを編集】」の下のタスクを、あなたの今日のタスクに書き換える
   - 例：「見積書の送付」→「月次報告書の作成」
2. タスクは3個でも10個でもOK
3. 必ず「-」（ハイフン）で始めてください

### ステップ5：送信する

1. Enterキーを押す（またはボタンをクリック）
2. 10秒ほど待つとChatGPTが答えてくれます

## ❓ よくある質問（FAQ）

### Q1: プロンプトって何ですか？
A: ChatGPTに送る「指示文」のことです。この資料に載っているプロンプトをコピペして使えば、誰でも同じ結果が得られます。

### Q2: タスクの書き方に決まりはありますか？
A: 必ず「-」（ハイフン）で始めてください。タスクは箇条書きで、1行に1つずつ書きます。

良い例：
```
- 見積書の送付
- 週報の作成
```

悪い例：
```
見積書の送付、週報の作成
```

### Q3: ChatGPTの答えが変な時はどうすればいいですか？
A: もう一度質問してみてください。「もう一度お願いします」と送信するだけでOKです。ChatGPTは毎回少し違う答えを返します。

### Q4: 無料版と有料版、どっちを使えばいいですか？
A: まずは無料版で大丈夫です。有料版（ChatGPT Plus、月額20ドル）の方が応答が速いですが、結果は同じです。

### Q5: タスクが20個以上あるときは？
A: 一度に全部入れても大丈夫ですが、最初は5〜10個程度から試すことをおすすめします。

## 💡 使いこなしのコツ

### コツ1：毎朝のルーティンにする
朝9時に出社したら、まずChatGPTにタスクを整理してもらう。これを習慣にすると、朝の迷いがゼロになります。

### コツ2：チームで共有する
朝会で「今日のタスクの優先順位はこうです」とAIの結果を見せれば、説明が楽になります。

### コツ3：AIの提案を100%信じなくてOK
AIは便利ですが、完璧ではありません。「これは違うな」と思ったら、自分で調整してください。

## 🚨 注意点

### 注意点1：AIの判断を鵜呑みにしない
AIはあなたの会社の事情や、特定の人間関係を知りません。AIの提案を見て、「これは違うな」と思ったら、自分で調整してください。

具体例：
- 上司が午後不在なら、報告は午前に前倒し
- クライアントの締切が実は明日なら、優先度を上げる

### 注意点2：タスクの大きさを揃える
タスクの大きさ（所要時間）がバラバラだと、AIも困ります。できるだけ、同じくらいの大きさのタスクで揃えましょう。

良い例：
- 見積書作成（30分）
- 週報作成（30分）
- 新規顧客メール（15分）

悪い例：
- プロジェクト完遂（数週間）
- メール1通（5分）
