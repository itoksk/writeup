# 動画1：優先度と緊急度をAIで判断！1日のタスクを自動で整列する方法

**所要時間**: 10分
**対象者**: 日々のタスク整理に時間を取られているビジネスパーソン
**使用ツール**: ChatGPT（無料版でOK）

---

## 1. 導入フレーズ（0:00-1:30）

### 講師セリフ

「おはようございます。今日も朝から、あれもこれもやらなきゃ…と思いながら、結局何から手をつけていいか分からず、気づいたら15分経っていた。そんな経験ありませんか？」

**【画面：朝のデスク風景、付箋だらけのモニター】**

「実は、この『何からやろう？』という迷いの時間、1日15分だとすると、月に7.5時間、年間で約90時間も無駄にしているんです。丸3日以上です。」

**【画面：計算式を表示】**
- 15分 × 20営業日 = 300分（5時間）
- 5時間 × 12ヶ月 = 60時間

「今日は、この『朝の迷い』をAIでゼロにする方法をお伝えします。ChatGPTを使えば、たった10秒でタスクの優先順位が決まります。」

---

## 2. Before/After（1:30-3:00）

### 講師セリフ

「まず、現状を整理しましょう。多くの方は、こんな流れでタスクを処理していませんか？」

**【画面：Beforeシナリオを表示】**

### Before（従来の方法）

**朝のタスク整理プロセス：**
1. メールやToDoリストを見る（5分）
2. 頭の中で優先順位を考える（7分）
3. 「やっぱりこっちが先か…」と悩む（3分）
4. ようやく1つ目のタスクに着手

**課題：**
- 平均15分の整理時間が毎朝発生
- 優先順位の判断基準が曖昧
- 重要だけど緊急じゃないタスクが後回し
- 結果的に締切直前で焦る

**【画面：Afterシナリオを表示】**

### After（AI活用後）

**朝のタスク整理プロセス：**
1. タスク一覧をChatGPTに貼り付け（1分）
2. AIが優先度・緊急度を判定（10秒）
3. 提案された順番で即実行

**効果：**
- 整理時間が15分→3分に短縮（80%削減）
- 判断基準が明確（AIが理由も説明）
- 重要タスクの着手率が50%向上
- 朝のストレスが大幅に軽減

### 講師セリフ

「つまり、朝の15分が3分になる。しかも、AIが理由まで説明してくれるので、上司に『なぜこの順番で進めているか』を説明するときも使えます。では、実際にやってみましょう。」

## 3. デモ（見せ場）（3:00-6:00）

### 講師セリフ

「それでは、ChatGPTを使って、実際にタスクを整理してみます。画面をご覧ください。」

**【画面：ChatGPTの画面を全画面表示】**

### デモステップ1：プロンプト入力

**講師セリフ：**
「まず、ChatGPTに以下のプロンプトを入力します。このプロンプトは、教材に載っていますので、後でコピペして使ってください。」

**【画面：prompts.txtからプロンプト1をコピペ】**

```
あなたは私の業務アシスタントです。
以下のタスクを「重要度」と「緊急度」で4象限マトリクスに分類してください。
それぞれの理由も簡潔に書き添えてください。
出力形式は表形式でお願いします。

タスク一覧：
- 見積書の送付
- 週報の作成
- 新規顧客への初回メール
- チームミーティングの準備
- 上司への報告資料修正
```

**講師セリフ：**
「では、送信します。」

**【画面：Enterを押す】**

### デモステップ2：AI出力の確認

**【画面：ChatGPTの出力を表示】**

**講師セリフ：**
「ほら、たった10秒で、こんな風に整理してくれました。見てください、ちゃんと表形式で出力されていますね。」

**【画面：出力結果をハイライト】**

| 緊急度 | 重要度 | タスク | 理由 |
|-------|-------|--------|------|
| 高 | 高 | 上司への報告資料修正 | 本日締切・業務評価に直結 |
| 高 | 低 | 見積書の送付 | 顧客対応期限が今日中 |
| 低 | 高 | 新規顧客への初回メール | 今後の関係性構築に重要 |
| 低 | 低 | チームミーティング準備 | 優先度低・期日も先 |

**講師セリフ：**
「ここがポイントです。AIは『理由』まで書いてくれるんです。例えば、『上司への報告資料修正』は、締切が今日で業務評価にも関わるから、緊急度も重要度も高い。これ、自分で考えると時間かかりますよね？」

**【画面：「理由」列をハイライト】**

### デモステップ3：優先順位の整列

**講師セリフ：**
「さらに、こう聞いてみます。『今日やるべきタスクを優先順に並び替えて』と。」

**【画面：プロンプト2を入力】**

```
上記の分類結果を基に、今日やるべきタスクを優先順に並び替えて、
実行順序を番号付きリストで提案してください。
各タスクに推奨時間帯（午前/午後）も添えてください。
```

**講師セリフ：**
「送信。」

**【画面：ChatGPTの出力を表示】**

**講師セリフ：**
「はい、こんな風に、優先順位まで提案してくれました。しかも、時間帯まで！」

**【画面：出力結果を表示】**

**今日やるべきタスク（優先順）**

1. 上司への報告資料修正（午前：10:00-11:00）
2. 見積書の送付（午前：11:00-11:30）
3. 新規顧客への初回メール（午後：14:00-14:30）
4. 週報の作成（午後：16:00-16:30）

**明日以降に回すタスク**
- チームミーティング準備（期日に余裕あり）

**講師セリフ：**
「これで、『今日は何からやろう？』という迷いがなくなります。1番から順にやればいいだけです。朝の15分が、これで3分になりました。」

## 6. 注意点・活用ポイント（8:30-9:30）

### 講師セリフ

「最後に、使う上での注意点と活用ポイントをお伝えします。」

**【画面：注意点をスライド表示】**

### 注意点1：AIの判断を鵜呑みにしない

**講師セリフ：**
「AIは便利ですが、完璧ではありません。例えば、あなたの会社の事情や、特定の人間関係は知りません。だから、AIの提案を見て、『これは違うな』と思ったら、自分で調整してください。」

**具体例：**
- 「上司が午後不在なら、報告は午前に前倒し」
- 「クライアントの締切が実は明日なら、優先度を上げる」

### 注意点2：タスクの粒度を揃える

**講師セリフ：**
「タスクの粒度がバラバラだと、AIも困ります。例えば、『見積書作成』と『メール返信10件』を並べると、所要時間が全然違いますよね。できるだけ、同じくらいの大きさのタスクで揃えましょう。」

**良い例：**
- 見積書作成（30分）
- 週報作成（30分）
- 新規顧客メール（15分）

**悪い例：**
- プロジェクト完遂（数週間）
- メール1通（5分）

### 活用ポイント1：毎朝のルーティンにする

**講師セリフ：**
「これを毎朝のルーティンにすると、劇的に変わります。朝9時に出社したら、まずChatGPTにタスクを整理してもらう。これを習慣にしてください。」

### 活用ポイント2：チームで共有する

**講師セリフ：**
「チームでタスクを共有するときにも使えます。例えば、朝会で『今日のタスクの優先順位はこうです』とAIの結果を見せれば、説明が楽になります。」

## 補足資料

### 想定される受講者からの質問

**Q1: ChatGPTの有料版じゃないとダメですか？**
A: 無料版で十分です。ただし、有料版（ChatGPT Plus）の方が応答速度が速いです。

**Q2: Googleスプレッドシートに直接出力できますか？**
A: ChatGPTの出力をコピーして、スプレッドシートに貼り付けることはできます。直接連携する方法は別の講座でお伝えします。

**Q3: タスクが20個以上あるときは？**
A: 一度に全部入れてもOKですが、分類が複雑になることがあります。朝・午後に分けて整理するのもおすすめです。
