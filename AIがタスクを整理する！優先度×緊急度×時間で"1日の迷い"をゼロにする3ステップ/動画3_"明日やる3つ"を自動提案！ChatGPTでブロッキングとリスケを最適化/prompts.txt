# デモ用プロンプト集 - 動画3
# "明日やる3つ"を自動提案！ChatGPTでブロッキングとリスケを最適化

---

## 🔰 初心者向け：まずはこれだけ！（最小限プロンプト）

**使い方（3ステップ）:**
1. 下のプロンプトをすべてコピー
2. ChatGPTに貼り付け
3. 【ここを編集】の部分を書き換えて送信

```
明日やるべきタスクを、優先順位が高い順に3つ提案してください。
それぞれのタスクに「推奨時間帯（午前/午後）」も教えてください。

【明日のタスク一覧】
【ここを編集：あなたの明日のタスクを書く】
- 上司への報告資料修正（重要で緊急、25分）
- 新規顧客への初回メール（重要だが急がない、15分）
- データ整理（急がない、20分）
- 会議準備（急ぎ、40分）
- 営業資料作成（やや急ぎ、45分）
```

**💡 ポイント:**
- タスク名の後に「（重要度、緊急度、所要時間）」を書くと、AIが正確に判断してくれます
- タスクは3個でも10個でもOK
- 「-」（ハイフン）を忘れずに

## プロンプト1：明日やるべき上位3件の提案（基本版）

**使い方:**
1. 下のプロンプトをすべてコピー
2. ChatGPTに貼り付け
3. 【タスクデータ】の部分を、あなたのタスクに書き換え
4. 送信ボタンを押す

```
以下のタスクデータを基に、
優先度・緊急度・所要時間の3要素から"明日やるべき上位3件"を提案してください。
各タスクに推奨時間帯（午前/午後）を添えてください。
最後に、Googleカレンダーに貼り付けられる形式でも出力してください。

【タスクデータ】

【ここを編集：あなたのタスクを書く】

■ 上司への報告資料修正
- 優先度：高
- 緊急度：高
- 所要時間：25分

■ 新規顧客への初回メール
- 優先度：高
- 緊急度：低
- 所要時間：15分

■ 顧客データ整理
- 優先度：低
- 緊急度：中
- 所要時間：20分

■ チームミーティング準備
- 優先度：低
- 緊急度：高
- 所要時間：40分

■ 来週の営業資料作成
- 優先度：中
- 緊急度：高
- 所要時間：45分
```

### 想定出力例

#### 明日やるべき上位3件

| タスク | 推奨時間帯 | 優先理由 |
|-------|-----------|---------|
| 上司への報告資料修正 | 午前 | 締切が迫っており影響度が高い |
| 来週の営業資料作成 | 午後 | 集中時間に適している |
| 新規顧客への初回メール | 午前 | 反応を当日中に得られる |

#### Googleカレンダー入力形式

```
10:00-10:30 上司への報告資料修正
11:00-11:15 新規顧客への初回メール
13:00-13:45 来週の営業資料作成
```

**💡 使い方:**
この形式をコピーして、Googleカレンダーに貼り付けるだけで、スケジュールが完成します。

## プロンプト2：タイムブロッキング形式での1日スケジュール提案（中級者向け）

**使い方:**
1. 下のプロンプトをコピー
2. 【タスク一覧】を書き換え
3. 送信

```
以下のタスクを、タイムブロッキング形式で1日のスケジュールに落とし込んでください。
勤務時間は9:00-18:00（休憩12:00-13:00）です。
優先度・緊急度・所要時間を考慮して、最適なスケジュールを提案してください。
また、タスク間に5分のバッファを入れてください。

【タスク一覧】

【ここを編集：あなたのタスクを書く】

1. 上司への報告資料修正（優先：高、緊急：高、25分）
2. 新規顧客への初回メール（優先：高、緊急：低、15分）
3. 顧客データ整理（優先：低、緊急：中、20分）
4. チームミーティング準備（優先：低、緊急：高、40分）
5. 来週の営業資料作成（優先：中、緊急：高、45分）
6. 週報作成（優先：低、緊急：低、15分）
7. メール返信10件（優先：中、緊急：中、30分）

【制約条件】
- 午前中（9:00-12:00）は集中作業に適している
- 午後（13:00-15:00）は会議や資料作成に適している
- 午後（15:00-18:00）は定型作業や軽作業に適している
```

### 想定出力例

#### タイムブロッキング形式スケジュール

| 時間 | タスク | 優先度 | 理由 |
|------|--------|--------|------|
| 09:00-09:30 | 上司への報告資料修正 | 高/高 | 締切が迫っており、午前の集中力を活用 |
| 09:35-10:20 | 来週の営業資料作成 | 中/高 | 集中力が必要な作業を午前に配置 |
| 10:25-10:45 | チームミーティング準備 | 低/高 | 緊急度が高いため午前中に完了 |
| 10:50-11:05 | 新規顧客への初回メール | 高/低 | 重要だが短時間で完了可能 |
| 11:10-11:40 | メール返信10件 | 中/中 | ランチ前の軽作業 |
| 12:00-13:00 | 🍴 ランチ休憩 | - | - |
| 13:00-13:20 | 顧客データ整理 | 低/中 | 午後のウォームアップ |
| 13:25-13:40 | 週報作成 | 低/低 | 定型作業を午後に配置 |

**💡 タイムブロッキングのメリット:**
タスク間に5分のバッファを入れることで、トイレ休憩や水分補給の時間を確保できます。

## 📝 ステップバイステップガイド（初めての方へ）

### ステップ1：明日のタスクを書き出す

**夜寝る前に、明日のタスクを書き出します**

1. メモ帳（またはスマホのメモ）を開く
2. 明日やるべきタスクを5〜10個書き出す
3. それぞれの「重要度」「緊急度」「所要時間」を書く

**記録例：**
```
【明日のタスク】
- 月次報告書の作成（重要、急ぎ、60分）
- クライアントメール返信（重要、急ぎ、15分）
- 会議資料準備（普通、急ぎ、30分）
- データ入力（重要じゃない、急がない、20分）
```

**💡 ポイント:**
- 重要度：自分や会社にとって重要か
- 緊急度：締切が近いか、今日中にやる必要があるか
- 所要時間：大体の時間でOK

### ステップ2：プロンプトをコピーして編集する

1. この資料の「🔰 初心者向け」プロンプトをコピー
2. ChatGPTに貼り付け
3. 【明日のタスク一覧】の部分を、ステップ1で書いた内容に書き換え

### ステップ3：送信して結果を確認

1. Enterキーを押して送信
2. 10秒ほど待つとChatGPTが答えてくれます
3. 「明日やる3つ」をメモして、寝る

### ステップ4：翌朝、カレンダーを見て即スタート

1. 翌朝、前日にメモした「明日やる3つ」を見る
2. 1番から順に実行する
3. 朝の迷いがゼロ！

## ❓ よくある質問（FAQ）

### Q1: Googleカレンダーへの自動連携はできますか?
A: 今回のデモでは手動コピペですが、Google Apps Script（GAS）を使えば自動連携も可能です。まずは手動コピペから始めてみてください。

### Q2: 緊急タスクが頻繁に発生する場合は?
A: AIに「リスケジュール提案」を依頼してください。優先度の低いタスクを明日以降に回す提案をしてくれます（プロンプト3を参照）。

### Q3: 週間スケジュールも作れますか?
A: はい、プロンプト4を使えば、AIが1週間のスケジュールを提案してくれます。

### Q4: 「3つ」にこだわらないといけませんか?
A: 「3つ」はあくまで目安です。タスクが多い日は5つでも、少ない日は2つでもOKです。重要なのは、「明日の優先順位が明確になる」ことです。

### Q5: タスクの重要度や緊急度が分からない時は？
A: 最初は「重要」「普通」「重要じゃない」「急ぎ」「急がない」のような簡単な言葉でOKです。動画1で学んだ4象限マトリクスを思い出してください。

## 🚨 注意点

### 注意点1：AIの提案を100%信じなくてOK
AIは便利ですが、完璧ではありません。「これは違うな」と思ったら、自分で調整してください。

例：
- 上司が午後不在なら、報告は午前に前倒し
- クライアントの締切が実は今日なら、優先度を上げる

### 注意点2：緊急タスクが入った時は柔軟に対応
スケジュールを組んでも、緊急タスクが入ることはあります。そのときは、AIに「リスケジュール提案」を依頼してください。

## プロンプト3：リスケジュール提案（緊急タスクの割り込み対応）

**こんな時に使います:**
- 急に緊急タスクが発生した
- 予定していたタスクが終わらなかった
- スケジュールを組み直したい

```
以下のスケジュールで本日の予定を組んでいましたが、
急遽「緊急案件：【ここを編集：緊急タスクの内容】（60分）」が発生しました。
このタスクを優先的に組み込み、他のタスクを調整したスケジュールを提案してください。
調整の際は、優先度の低いタスクを明日以降に回すことも検討してください。

【現在のスケジュール】
【ここを編集：今日のスケジュールを書く】
09:00-09:30 上司への報告資料修正
09:35-10:20 来週の営業資料作成
10:25-10:45 チームミーティング準備
10:50-11:05 新規顧客への初回メール

【緊急タスク】
- 【ここを編集：緊急タスクの内容】（優先：最高、緊急：最高、60分）

【制約】
- 勤務時間：9:00-18:00（休憩12:00-13:00）
- 【ここを編集：今日中に必須のタスク】は本日中に必須
- その他は明日以降に回してもOK
```

## プロンプト4：週間スケジュール提案（1週間分のタスク配分）

**こんな時に使います:**
- 月曜の朝に、1週間のスケジュールを立てたい
- タスクが多すぎて、どう配分すればいいか分からない

```
以下のタスクを、今週（月曜～金曜）のスケジュールに配分してください。
各日の勤務時間は9:00-18:00（休憩12:00-13:00）です。
優先度・緊急度・所要時間を考慮して、無理のないスケジュールを提案してください。

【今週のタスク一覧】

【ここを編集：今週のタスクを書く】

■ 月曜
- 週報作成（15分）
- クライアントA訪問準備（30分）

■ 火曜
- クライアントA訪問（3時間、14:00-17:00固定）
- 議事録作成（20分）

■ 水曜～金曜
- 新規提案書作成（120分）
- 見積書作成×5件（各25分、計125分）
- メール返信（毎日30分）
- チームミーティング（毎週水曜15:00-16:00固定）
- 月次報告書作成（90分、金曜中に完了必須）

【制約】
- クライアントA訪問（火曜14:00-17:00）は固定
- チームミーティング（水曜15:00-16:00）は固定
- 月次報告書は金曜中に完了必須
```
