# Google Workspace Studio カスタマーサクセス完全自動化講座

## コース概要
Google Workspace Studioを活用して、顧客からの問い合わせ対応、フォローアップ、満足度調査までカスタマーサクセス業務を完全自動化する実践講座です。5つのレッスンで、24時間365日の自動対応と、顧客との継続的なコミュニケーションを実現します。

## 対象者
- 日々の問い合わせ対応に追われているカスタマーサポート担当者
- 顧客フォローアップを効率化したい営業マネージャー
- NPS・満足度調査を定期的に実施したいCXチーム
- 解約防止・アップセル提案を自動化したいカスタマーサクセス担当者

## 前提知識
- Google Workspaceの基本操作（Gmail、スプレッドシート）
- プログラミング知識は不要（完全ノーコード）

## 達成目標
- 問い合わせメールの自動分類と一次回答の自動生成
- 契約更新前のフォローアップ自動化
- 定期的な満足度調査の自動実施と分析
- 解約リスク顧客の早期検知とアラート


## カリキュラム（全5回）

### 第1回：一次対応を90%削減！AIが即座に回答ドラフトを生成する問い合わせ自動処理フロー
**メール受信→内容分析→自動回答ドラフト作成**

- **学習目標**: 顧客からの問い合わせメールを自動分類し、Geminiで回答ドラフトを自動生成する
- **内容**:
  - Gmailスターターでの問い合わせ受信設定
  - Geminiによる問い合わせ内容の自動分類（質問/クレーム/要望/その他）
  - 過去FAQからの類似回答検索
  - 回答ドラフトの自動生成
  - 緊急度判定と優先対応フラグ
  - 担当者への通知と回答レビュー依頼
- **作成するエージェント**: 問い合わせ自動分類エージェント

### 第2回：更新漏れゼロを実現！30日前から始める段階的リマインドシステム
**更新日監視→リマインド送信→担当者アサイン**

- **学習目標**: 契約更新日の30日前から自動フォローアップを開始し、更新率を向上させる
- **内容**:
  - 顧客マスタからの更新日自動監視
  - 更新30日前・14日前・7日前の段階的リマインド
  - 顧客セグメント別のメールテンプレート
  - 未応答顧客の担当者エスカレーション
  - 更新完了の自動トラッキング
- **作成するエージェント**: 契約更新フォローアップエージェント

### 第3回：低評価は即フォロー！取引完了時に自動送信するNPS調査システム
**対象抽出→調査送信→回答集計→アラート発報**

- **学習目標**: 取引完了後に満足度調査を自動送信し、低評価には即座にフォローアップを行う
- **内容**:
  - 取引完了トリガーでの調査自動送信
  - NPS（推奨度）と満足度の自動収集
  - 低評価（NPS 6以下）への即時アラート
  - 回答データのスプレッドシート自動記録
  - 月次NPSトレンドレポートの自動生成
- **作成するエージェント**: NPS調査自動化エージェント

### 第4回：チャーンを30日前に予測！AIが見つけ出す解約リスク早期検知システム
**行動分析→リスクスコア計算→アラート→対策提案**

- **学習目標**: 顧客の行動データからチャーンリスクを予測し、早期にアラートを発報する
- **内容**:
  - ログイン頻度・利用量の自動モニタリング
  - 問い合わせ履歴からの不満シグナル検知
  - チャーンリスクスコアの自動計算
  - リスク顧客への優先フォロー依頼
  - Geminiによる対策提案の自動生成
- **作成するエージェント**: チャーンリスク検知エージェント

### 第5回：経営判断を加速する！全KPIをリアルタイム可視化するCSダッシュボード
**データ集約→KPI計算→ダッシュボード更新→週次レポート配信**

- **学習目標**: 顧客対応状況を可視化するダッシュボードを自動更新し、週次レポートを自動配信する
- **内容**:
  - 問い合わせ対応時間・完了率の自動集計
  - NPS推移と顧客セグメント別分析
  - 解約率・更新率のリアルタイム更新
  - 週次CSレポートの自動生成と配信
  - 重要顧客のヘルススコア一覧
- **作成するエージェント**: CSダッシュボード更新エージェント

## 技術スタック
- Google Workspace Studio
- Gemini（メール分析・回答生成・リスク分析）

- Googleスプレッドシート
- Googleフォーム（満足度調査）
- Google Chat / Slack連携

## 主要なステップ・機能

### スターター
| スターター | 用途 |
|-----------|------|
| Gmailスターター | 問い合わせメール受信トリガー |
| スケジュールスターター | 定期的な監視・レポート生成 |
| Formsスターター | 満足度調査回答受信 |

### ステップ
| ステップ | 用途 |
|---------|------|
| Geminiステップ | メール分類、回答生成、リスク分析 |
| スプレッドシート連携 | 顧客データ読み書き、集計 |
| 条件分岐 | 緊急度判定、リスクレベル分岐 |
| Gmail送信ステップ | 自動返信、フォローアップ送信 |
| Chat DMステップ | 担当者への緊急通知 |

## 自動化される業務フロー

```
問い合わせ対応フロー：
メール受信 → 内容分類 → 緊急度判定 → FAQ検索 →
回答ドラフト生成 → 担当者レビュー → 返信送信 → 完了記録

契約更新フロー：
更新日監視 → リマインド送信 → 応答確認 →
未応答エスカレーション → 更新完了 → 記録更新

チャーン防止フロー：
行動データ収集 → リスクスコア計算 → 高リスク検知 →
担当者アラート → 対策提案 → フォロー実行 → 結果記録
```

## 達成できること
- 問い合わせ一次対応時間：24時間→1時間以内
- 契約更新フォロー漏れ：0件
- NPS調査回収率：30%→60%
- チャーンリスク早期検知：解約30日前に90%検知

## 注意事項
- 外部APIキーは不要（Google Workspace内で完結）
- 対応エディション: Business Standard以上推奨
- 自動返信は下書き作成→担当者承認フローを推奨

## 必要条件
- Google Workspace アカウント（Workspace Studio へのアクセス権）
- 顧客マスタスプレッドシート
- 問い合わせ管理スプレッドシート
- FAQ一覧スプレッドシート
