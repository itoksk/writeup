# デモコード・テンプレート集

## ファイル一覧

### 1. OKRテンプレート_サンプル.csv
ChatGPTで生成した部門OKRのサンプルデータです。
Googleスプレッドシートにインポートして使用できます。

#### 使用方法
1. Googleスプレッドシートを開く
2. 「ファイル」→「インポート」→「アップロード」
3. このCSVファイルを選択
4. インポート設定で「カンマ」を区切り文字に指定
5. 「データをインポート」をクリック

#### カスタマイズ
- 部門名を自社の組織に合わせて変更
- Objectiveを自社の経営方針に合わせて調整
- Key Resultsの数値目標を現実的な値に修正

## Googleスプレッドシートテンプレートの作成方法

### 基本構成

#### シート1：部門別OKR一覧
| 列 | 項目 | 説明 |
|----|------|------|
| A | 部門 | 部門名 |
| B | Objective | 目的（定性的） |
| C | Key Results 1 | 定量指標1 |
| D | Key Results 2 | 定量指標2 |
| E | Key Results 3 | 定量指標3 |
| F | 進捗率 | 全体の進捗（%） |
| G | ステータス | 順調/注意/遅延 |

#### シート2：経営方針
経営方針を記録しておくシートです。

| 列 | 項目 | 説明 |
|----|------|------|
| A | 項目番号 | 1, 2, 3... |
| B | 経営方針 | 具体的な方針内容 |
| C | 担当役員 | 責任者 |
| D | 期限 | 達成期限 |

#### シート3：進捗トラッキング
四半期ごとの進捗を記録します。

| 列 | 項目 |
|----|------|
| A | 部門 |
| B | Key Results |
| C | Q1進捗 |
| D | Q2進捗 |
| E | Q3進捗 |
| F | Q4進捗 |
| G | 年間達成率 |

## スプレッドシートの書式設定

### 色分けルール
- **ヘッダー行**: 濃いグレー背景 + 白文字
- **営業部**: 薄い青背景
- **マーケティング部**: 薄い緑背景
- **開発部**: 薄いオレンジ背景
- **人事部**: 薄い紫背景

### 条件付き書式
進捗率（F列）に応じて自動で色を変更：
- 80%以上: 緑
- 50〜79%: 黄色
- 50%未満: 赤

ステータス（G列）に応じて自動で色を変更：
- 順調: 緑
- 注意: 黄色
- 遅延: 赤

## 数式の例

### 進捗率の自動計算
```
=AVERAGE(進捗データ範囲)
```

### ステータスの自動判定
```
=IF(F2>=80, "順調", IF(F2>=50, "注意", "遅延"))
```

### Key Results達成度の可視化
```
=SPARKLINE(Q1進捗:Q4進捗, {"charttype","line"})
```

## ChatGPTとの連携フロー

1. **経営方針をChatGPTに入力**
   - プロンプト集のプロンプト1を使用
   - 経営方針をコピー&ペースト

2. **OKRを生成**
   - ChatGPTの出力を確認
   - 必要に応じて再生成

3. **スプレッドシートに貼り付け**
   - 表形式のままコピー
   - スプレッドシートのA2セルから貼り付け

4. **書式を整える**
   - 色分けルールを適用
   - 条件付き書式を設定

5. **チームと共有**
   - 「共有」ボタンから共有設定
   - コメント権限を付与

## トラブルシューティング

### Q: ChatGPTの出力が表形式にならない
A: プロンプトに「表形式で出力してください」と明記してください。また、「Markdown形式の表」と指定すると確実です。

### Q: 貼り付けた時に列がずれる
A: ChatGPTの出力をコピーする前に、スプレッドシートの形式を確認してください。必要に応じて「形式を選択して貼り付け」→「値のみ」を使用します。

### Q: 日本語が文字化けする
A: CSVファイルの文字コードを「UTF-8」に設定してください。Googleスプレッドシートは自動的にUTF-8を認識します。
