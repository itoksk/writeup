# 10分で戦略展開！AIが自動生成するOKR・KPI設計＆整合性診断マスター講座

## コース概要

このコースでは、ChatGPTを活用してOKR（Objectives and Key Results）とKPI（Key Performance Indicators）の設計を自動化し、戦略実行を10分で設計可能にする方法を学びます。経営方針から部門OKR、KPIツリー、整合性診断まで、AIがサポートする戦略展開の全プロセスを習得します。

## 想定する現場課題

### 現状の問題
- **各部門が独自にOKRを作り、経営方針とズレてしまう**
  - 経営層の意図が現場に正確に伝わらず、部門ごとに異なる解釈で目標設定
  - 四半期ごとの見直しでズレが発覚し、手戻りが発生

- **目標が抽象的で、行動や数値に落とし込めていない**
  - 「価値を高める」「品質向上」など抽象的な表現が多く、測定不可能
  - KPIが曖昧で、進捗管理ができない

- **会議でOKRをすり合わせるのに時間がかかり、議論が空回り**
  - 1回の会議で2〜3時間かけても合意形成できない
  - 部門間の調整に毎期1週間以上を要する

- **OKRやKPI設計が属人化し、毎期フォーマットが変わる**
  - 担当者が変わると設計品質にバラつき
  - 過去の設計資産が活用できず、毎回ゼロから作成

- **AIを活用したいが、「どうプロンプトを設計すれば正確に出せるか」が分からない**
  - ChatGPTを使っても期待する出力が得られない
  - プロンプトの書き方が分からず、試行錯誤に時間がかかる

### 数値で見る現状
- 部門OKR策定に **2〜3日** かかる
- 整合性チェックに **会議1回あたり2〜3時間**
- 毎期のOKR設計で **担当者ごとに品質差が20〜30%**
- 戦略ズレの修正に **四半期あたり1週間以上**

## 理想の状態（Before / After）

| 項目 | Before | After |
|------|--------|-------|
| **作業時間** | 部門OKR策定に2〜3日 | **10分でOKR→KPI→整合性まで自動生成** |
| **属人化** | 担当者ごとに品質差 | **ChatGPTが統一フォーマットで自動展開** |
| **戦略整合性** | 方針ズレが毎期発生 | **AIが全体整合性を自動診断・可視化** |
| **感情面** | 「どこから手をつければ…」 | **"AIが設計図を描いてくれる"安心感** |

### 実現される効果
- 作業時間を **90%削減**（2〜3日 → 10分）
- 部門整合性の **可視化** による意思決定の高速化
- 設計の **属人化解消** による品質の均一化
- **再現性** のある戦略展開プロセスの確立

## コース構成（全3本）

### 第1回：AIで戦略を10分設計：ChatGPTでOKR→行動計画を自動展開
**目標**: 経営方針からChatGPTを使って部門OKRを自動生成する方法を習得

- ChatGPTを"戦略翻訳装置"として活用
- 経営目標から部門別OKRを10分で自動出力
- Objective（目的）とKey Results（定量指標）の自動生成
- スプレッドシートへの展開と可視化

### 第2回：KPIツリーをAIで即構築：目標を「数値と行動」に落とすプロンプト設計術
**目標**: OKRをKPIツリー（3階層構造）に展開し、行動レベルまで落とし込む

- OKRから「行動化テンプレート」を自動生成
- KPIツリー（3階層構造）の自動構築
- 各Key Resultに対応する具体行動の抽出
- 数値・行動・担当の明確化

### 第3回：ズレを可視化するAI整合性診断：部門OKRを一発チェック
**目標**: AIを使って部門OKR間の整合性を診断し、ズレを可視化

- ChatGPTによる整合性診断レポートの自動生成
- 重複・矛盾・上位目標とのギャップの自動検出
- 整合性スコアの可視化
- 改善提案の自動出力

## 改善方針

### 技術アプローチ
- **使用ツール**: ChatGPT（GPT-4 / GPT-5）、Googleスプレッドシート、Excel、Notion
- **制約**: 各動画10分以内、ChatGPTとスプレッドシート中心の最小構成
- **効率化目標**: 作業時間90%削減（現実的な数値目標）

### 教育設計方針
- **実演重視**: 全ステップが講師のChatGPT画面デモで再現可能
- **プロンプト設計**: テンプレート化されたプロンプト例を提供
- **段階学習**: 設計 → KPI化 → 整合性診断の順で段階的に習得

## 使用ツール・技術要素

### 主要ツール
- **ChatGPT（GPT-4 / GPT-5）**: OKR生成、KPIツリー構築、整合性診断
- **Googleスプレッドシート**: OKR・KPIの可視化、整合性スコア表作成
- **Excel**: データ整理、ツリー構造の管理
- **Notion**: 整合性レポートの共有

### プロンプト設計技術
- 出力形式の指定（表形式、ツリー構造）
- 定量表現の強制
- 階層構造の明示
- 整合性評価基準の設定
