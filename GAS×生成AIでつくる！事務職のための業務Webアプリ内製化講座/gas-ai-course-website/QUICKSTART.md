# クイックスタートガイド

このガイドでは、Webサイトをローカルで起動して確認する手順を説明します。

## 📋 必要なもの

- Node.js 18以上 ([ダウンロード](https://nodejs.org/))

## 🚀 3ステップで起動

### Step 1: ディレクトリに移動

```bash
cd "GAS×生成AIでつくる！事務職のための業務Webアプリ内製化講座/gas-ai-course-website"
```

### Step 2: 依存関係をインストール

```bash
npm install
```

初回のみ実行してください。数分かかる場合があります。

### Step 3: 開発サーバーを起動

```bash
npm run dev
```

### ✅ ブラウザで確認

ブラウザで以下のURLを開いてください：

```
http://localhost:3000
```

サイトが表示されれば成功です！🎉

## 🎨 サイト構造

- **トップページ** (`/`) - 講座の概要と一覧
- **各講座ページ** (`/lesson/1` 〜 `/lesson/5`) - 教材・セリフ・プロンプトをタブで表示

## 📝 次のステップ

1. **ローカルで確認**: サイトが正しく表示されるか確認
2. **内容の確認**: 各講座ページでタブ切り替えが動作するか確認
3. **デプロイ**: `DEPLOYMENT.md` を参照してVercelにデプロイ

## 🐛 トラブルシューティング

### ポート3000が使用中の場合

別のポートで起動:

```bash
PORT=3001 npm run dev
```

### インストールエラーが出る場合

Node.jsのバージョンを確認:

```bash
node -v
```

18以上であることを確認してください。

### データが表示されない場合

親ディレクトリに各動画フォルダ（動画1〜5）が存在することを確認してください。

---

問題が解決しない場合は、`README.md` または `DEPLOYMENT.md` を参照してください。
