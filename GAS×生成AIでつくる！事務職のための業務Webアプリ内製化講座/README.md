# GAS×生成AIでつくる！事務職のための業務Webアプリ内製化講座

---

## 1. 想定する現場課題

**事務職の現場で頻発する非効率な業務プロセス**

- **紙とExcelによる分散管理**: 日報・見積書・請求書などの業務データが紙やExcel、メールに分散し、集計・検索に毎回30分～1時間かかる
- **定型業務の手入力**: 見積書や日報などの定型フォーマット作成に毎回10～15分かかり、転記ミス・記載漏れが月に3～5件発生
- **属人化とブラックボックス化**: 業務手順やフォーマットが担当者ごとに異なり、引き継ぎや代理対応が困難
- **業務改善への壁**: 「プログラミングは無理」「予算がない」「IT部門に頼めない」という思い込みで、改善が進まない
- **AI活用の未着手**: ChatGPTやGeminiの存在は知っているが、具体的に業務でどう使えばいいか分からない

**具体的な数値例**:
- 日報作成：1件あたり平均15分 × 月20日 = 月5時間
- 見積書作成：1件あたり平均20分 × 月10件 = 月3.3時間
- データ集計・検索：1回あたり平均30分 × 月8回 = 月4時間
- **合計：月12.3時間の非効率作業**

## 4. 改善方針

### 技術範囲の明確化

- **使用ツール**: Google Apps Script（GAS）、ChatGPT、Claude、Gemini
- **対象サービス**: Googleフォーム、スプレッドシート、Google Docs、Googleサイト、Google Drive
- **開発環境**: Webブラウザのみ（インストール不要、無料利用範囲内）
- **難易度**: プログラミング未経験者でも、AIの支援を受けながら段階的に習得可能

### 動画制約・構成

- **1動画あたり9～10分以内**（集中力維持、昼休みで視聴可能）
- **全5回の段階学習**: 準備 → 日報アプリ → 公開 → 見積書自動化 → AI比較活用
- **実演中心**: 台本セリフに沿って、画面操作を逐一解説
- **失敗例・注意点を明示**: 権限設定ミス、コード貼り付け位置など、初学者がつまずきやすいポイントを事前に提示

### 現実性の担保

- **過度な期待を排除**: 「完全自動化」「革命的」などの表現は使わず、「20～30%効率化」「転記ミス削減」など現実的な効果を提示
- **段階的な学習**: いきなり複雑なアプリではなく、日報アプリ → 見積書アプリと難易度を上げていく
- **AI活用の現実**: AIの出力をそのまま使わず、必ず業務目的に照らして確認・調整する姿勢を強調

## 9. 使用ツール・技術要素

### 開発環境・ツール

| ツール | 用途 | 費用 |
|--------|------|------|
| **Google Apps Script (GAS)** | 業務アプリの自動化コード作成 | 無料 |
| **Googleフォーム** | 日報入力インターフェース | 無料 |
| **Googleスプレッドシート** | データ蓄積・集計 | 無料 |
| **Google Docs** | 見積書テンプレート | 無料 |
| **Google Drive** | ファイル保存・共有 | 無料 |
| **Googleサイト** | アプリ公開・部内共有 | 無料 |

### 生成AIツール

| AI | 用途 | 費用 |
|----|------|------|
| **ChatGPT** | コード生成、具体的な実装支援 | 無料版で対応可 |
| **Claude** | 長文説明、仕様書作成支援 | 無料版で対応可 |
| **Gemini** | Google Workspace連携、日本語対応 | 無料版で対応可 |

### 技術要素

- **JavaScript（GAS）**: 基本的な関数、変数、配列、オブジェクト操作
- **Google Apps Script API**: SpreadsheetApp、FormApp、DocumentApp、DriveApp
- **イベントトリガー**: onFormSubmit、時刻ベーストリガー
- **テンプレートエンジン**: 文字列置換（{{変数名}}形式）
- **PDF変換**: getAs(MimeType.PDF)

### 学習前提

- **必須**: Webブラウザ操作、Googleアカウント
- **推奨**: Excelの基本操作経験（関数、セル参照など）
- **不要**: プログラミング経験、IT部門との連携、有料ツール

以下のような過度な表現は使用しません：

- ❌ 「革命的」「完全自動化」「誰でも簡単」「AIにすべてお任せ」
