## プロンプト0: 業務課題の深掘りヒアリング（新規追加）

### 使用タイミング
動画視聴前、または自分の業務に応用する前の要件整理時

### プロンプト本文

```
あなたは業務効率化のコンサルタントです。
私の業務課題をヒアリングして、Google Apps Script（GAS）で解決できる可能性があるか、一緒に整理してください。

【私の現状】
業種・職種: [例：営業事務、総務、経理など]
抱えている課題: [例：日報作成に毎日30分かかる、見積書作成でミスが多い、など]

以下の質問に答えてください：
1. この業務の「現在の手順」を教えてください（何をどの順番で行っていますか？）
2. その作業に「どれくらい時間がかかっていますか」？（1日あたり、1週間あたりなど）
3. 「どんな課題やストレス」を感じていますか？（ミスが多い、時間がかかる、属人化している、など）
4. 「理想の状態」はどうなっていれば嬉しいですか？（どこまで自動化したいか、何を残したいか）

この情報をもとに、Google Apps Scriptで実現可能な改善案を一緒に考えましょう。
```

### 期待される回答例

```
【ヒアリング項目の整理】

1. 現在の手順を教えてください
   → 例：「毎日Excelファイルを開く → 手入力 → 上司にメール送信」

2. 時間はどれくらいかかっていますか？
   → 例：「1日15分 × 20日 = 月5時間」

3. どんな課題を感じていますか？
   → 例：「記載漏れが月3件、過去データの検索が困難」

4. 理想の状態は？
   → 例：「フォーム入力で自動集計、検索が簡単にできる」

【改善案】
- Googleフォームで入力を簡略化
- Google Apps Scriptで自動集計
- Googleスプレッドシートで一元管理
- 期待効果：作業時間30%削減、ミス80%削減
```

### 使い方
1. [業種・職種] と [抱えている課題] を自分の状況に置き換える
2. AIの質問に答えながら、業務フローと課題を整理
3. AIが提案する改善案をもとに、次のプロンプトに進む

## プロンプト1: ChatGPTで日報アプリの仕様案を作成

### 使用タイミング
動画内デモ（2:00-5:00）、受講者の実践時

### プロンプト本文

```
あなたは業務効率化のコンサルタントです。
事務職向けの「日報管理アプリ」を、**Google Apps Script（GAS）**と**Google Workspace（Googleフォーム、Googleスプレッドシート）**を使って作りたいと考えています。
以下の要件を満たすアプリの仕様案を、簡潔に箇条書きで提案してください。

【要件】
- 毎日の業務内容を記録する
- 日付、氏名、業務内容、所感を入力できる
- 入力したデータは自動でGoogleスプレッドシートに蓄積される
- 後から検索・集計しやすい形式にする

【使用技術】
- Google Apps Script（GAS）
- Googleフォーム
- Googleスプレッドシート
- Googleサイト（任意）

【制約】
- 無料で利用できるGoogle Workspaceの範囲内
- プログラミング未経験者でも理解・運用できる内容
```

### 期待される回答例

```
【日報管理アプリ仕様案】
1. Googleフォームで入力画面を作成
   - 入力項目: 日付、氏名、業務内容、所感
2. フォーム送信時に自動でスプレッドシートに転記
3. スプレッドシートの列構成:
   - A列: 日付
   - B列: 氏名
   - C列: 業務内容
   - D列: 所感
4. 日付でソート可能にする
5. Googleサイトでチーム内共有
```

### 注意点
- AIの回答は毎回少しずつ異なります
- 自分の業務に合わせて、要件を追加・削除してOKです
- 「Google Apps Scriptで」と明記することで、GASに適した提案が出やすくなります

## プロンプト2: Geminiでフォーム項目の詳細を作成

### 使用タイミング
動画内デモ（2:00-5:00）、受講者の実践時

### プロンプト本文

```
**Googleフォーム**で「業務日報」入力フォームを作成します。
このフォームは**Google Apps Script（GAS）**でGoogleスプレッドシートに自動連携されます。

以下の要件を満たす入力項目を、具体的に列挙してください。

【要件】
- 日付、氏名、業務内容、所感を記録
- 入力しやすく、検索しやすい形式
- 必須項目と任意項目を区別
- **Googleフォーム**の標準機能で実現可能な入力形式を使用

【出力形式】
各項目について、以下の形式で出力してください：
1. 項目名: [具体的な項目名]
2. 入力形式: [記述式/段落/ラジオボタン/チェックボックス/プルダウン/日付/時刻など]
3. 必須/任意: [必須 or 任意]
4. 備考: [入力のヒントや注意事項]
```

### 期待される回答例

```
【業務日報 フォーム項目一覧】
1. 日付
   - 入力形式: 日付選択
   - 必須/任意: 必須
2. 氏名
   - 入力形式: プルダウン（事前登録）
   - 必須/任意: 必須
3. 業務内容
   - 入力形式: 長文テキスト（段落）
   - 必須/任意: 必須
4. 所感・気づき
   - 入力形式: 長文テキスト（段落）
   - 必須/任意: 任意
5. 作業時間
   - 入力形式: 数値入力
   - 必須/任意: 任意
```

### 注意点
- Geminiは日本語の理解が得意です
- 「入力形式」まで指定すると、より具体的な提案が得られます
- 不要な項目（例：作業時間）は削除してOKです

## プロンプト3: AIに業務改善アイデアを聞く（応用編）

### 使用タイミング
受講者が自分の業務に応用する時

### プロンプト本文（テンプレート）

```
あなたは業務効率化のコンサルタントです。
私は [業種・職種] で働いており、以下の業務課題を抱えています。

【現在の課題】
- [具体的な課題1（例：見積書作成に毎回20分かかる）]
- [具体的な課題2（例：過去データの検索に時間がかかる）]
- [具体的な課題3（例：転記ミスが月3件発生）]

この課題を、**Google Apps Script（GAS）**と**Google Workspace（Googleフォーム、Googleスプレッドシート、Google Docs、Google Drive）**、および生成AI（ChatGPT/Gemini/Claude）を活用して解決するアプリのアイデアを、簡潔に3つ提案してください。

【使用可能な技術】
- Google Apps Script（GAS）
- Googleフォーム
- Googleスプレッドシート
- Google Docs
- Google Drive
- Googleサイト

【制約条件】
- 無料で利用できる範囲内
- プログラミング未経験者でも1週間以内に構築可能な内容
- 月の作業時間を20～30%削減できる現実的な提案

【出力形式】
各アイデアについて、以下の形式で出力してください：
1. アプリ名: [分かりやすい名称]
2. 概要: [どんなアプリか、2～3行で説明]
3. 使用技術: [Google Apps Script + 〇〇]
4. 主要機能: [箇条書きで3つ程度]
5. 期待効果: [作業時間〇%削減、ミス〇件削減など、具体的な数値で]
```

### 使い方
- [業種・職種] の部分を自分の状況に置き換えてください
  - 例：「営業事務」「経理部門」「総務」など
- [具体的な課題] には、数値や頻度を入れると良い提案が得られます
  - 良い例：「見積書作成に毎回20分かかる」
  - 悪い例：「見積書が大変」

### 期待される回答例

```
【業務改善アプリ案】
1. 見積書自動生成アプリ
   - 概要: Docsテンプレート×GASで、顧客情報を入力するだけで見積書PDFを自動生成
   - 期待効果: 作成時間75%削減、転記ミスゼロ

2. 過去データ検索システム
   - 概要: スプレッドシートに蓄積したデータを、フォームから検索できる仕組み
   - 期待効果: 検索時間80%削減、即座にデータ取得

3. 承認フロー自動化
   - 概要: フォーム送信時に、自動でメール通知＋Slackへ投稿
   - 期待効果: 承認待ち時間50%削減、見落とし防止
```

## プロンプト活用のコツ

### 1. 具体的な数値を入れる
- ❌ 悪い例：「時間がかかる」
- ⭕ 良い例：「毎回20分かかる」「月10件発生」

### 2. 制約条件を明記する
- 「Google Apps Scriptで」「無料ツールのみで」「10分以内で作れる範囲で」
- 制約を伝えることで、現実的な提案が得られます

### 3. 出力形式を指定する
- 「箇条書きで」「表形式で」「手順を番号付きで」
- 形式を指定すると、見やすい回答が得られます

### 4. 複数AIで比較する
- 同じプロンプトをChatGPT、Gemini、Claudeに投げて比較
- それぞれの得意分野が見えてきます

### 5. 段階的に質問する
- いきなり「コードを書いて」ではなく
- 「仕様案を出して」→「項目を詳しく」→「コードを書いて」と段階的に

## よくある質問

**Q1: AIの回答が期待と違う場合は？**
A1: 追加で「もっと具体的に」「初心者向けに」と指示を出してください。
    または、プロンプトに「例：〇〇のような形式で」と例示を加えると改善します。

**Q2: 同じプロンプトでも毎回回答が違うのはなぜ？**
A2: AIは確率的に回答を生成するため、毎回少し異なります。
    何度か試して、一番良い回答を選ぶのもコツです。

**Q3: どのAIを使えばいいですか？**
A3: 第5回で詳しく解説しますが、迷ったらChatGPTから始めてOKです。
    日本語重視ならGemini、長文説明が欲しい時はClaudeがおすすめです。
