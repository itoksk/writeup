# 動画1 対話型プロンプト - デモ用プロンプト集

## 実演用プロンプト（動画内で使用）

### デモ1：営業部門の業務効率化

```
営業部門の業務効率化について一緒に考えたい。
現状を理解するために、必要に応じて質問してください。
```

**AIからの想定質問と回答例：**

Q: 営業部門の規模と主な業務内容を教えてください。
A: 営業部員は15名。毎日の日報作成と週次の訪問計画作成に時間がかかっています。

Q: 日報作成には1日どれくらい時間がかかっていますか？
A: 平均30分。訪問先、商談内容、次回アクション、所感を記載しています。

Q: 既存のツールやシステムは何を使用していますか？
A: ExcelとWordが中心。CRMは導入していません。

## 練習用プロンプト集

### 1. 新規事業アイデアの深掘り

#### 初回プロンプト
```
シニア向けの新規事業アイデアを一緒に考えたい。
市場理解のため、必要に応じて質問してください。
```

#### 対話の進め方
- ターゲットの年齢層を具体化（65-75歳のアクティブシニア）
- 解決したい課題を明確化（社会的孤立、健康維持）
- 予算規模を設定（初期投資500万円）
- 収益モデルを検討（月額課金、コミュニティ型）

---

### 2. マーケティング戦略立案

#### 初回プロンプト
```
BtoB SaaSのマーケティング戦略を立案したい。
現状分析のため、詳細を質問してください。
```

#### 段階的な深掘りポイント
1. **製品理解**
   - 製品の主要機能
   - 競合との差別化要因
   - 価格帯

2. **顧客理解**
   - ターゲット企業規模
   - 意思決定者の役職
   - 導入障壁

3. **現状課題**
   - リード獲得コスト
   - コンバージョン率
   - 顧客獲得単価

---

### 3. 社内研修プログラム設計

#### 初回プロンプト
```
DX推進のための社内研修プログラムを設計したい。
組織の現状を把握するため、質問してください。
```

#### 重要な確認事項
- 対象者のITリテラシーレベル
- 研修に割ける時間（週何時間）
- 期待する成果（具体的なスキル）
- 予算と期間

## 効果的な対話のコツ

### ✅ 良い回答の特徴

1. **具体的な数値を含む**
   ```
   ❌ 「売上が低い」
   ✅ 「月商500万円で、前年比20%減少」
   ```

2. **背景情報を追加する**
   ```
   ❌ 「効率が悪い」
   ✅ 「手作業でのデータ入力に1日2時間かかり、ミスも月10件程度発生」
   ```

3. **制約条件を明示する**
   ```
   ❌ 「予算がある」
   ✅ 「年間予算300万円、ただし初期投資は100万円まで」
   ```

## 実践演習

### 演習1：問題解決型
```
【初回プロンプト】
売上が伸び悩んでいる飲食店の改善策を検討したい。
状況把握のため、必要な情報を質問してください。

【あなたの準備】
- 立地：駅から徒歩5分
- 客単価：ランチ1,000円、ディナー3,000円
- 現在の集客：1日平均30名
- 課題：リピート率が20%と低い
```

### 演習2：企画立案型
```
【初回プロンプト】
社員の健康増進プログラムを企画したい。
企画に必要な情報を収集するため、質問してください。

【あなたの準備】
- 社員数：200名
- 平均年齢：35歳
- 現状の課題：健康診断でメタボ該当者30%
- 予算：年間100万円
```

### 演習3：改善提案型
```
【初回プロンプト】
ECサイトのカート放棄率を改善したい。
現状分析のため、詳細を質問してください。

【あなたの準備】
- カート放棄率：70%
- 平均カート金額：8,000円
- 主な離脱ポイント：送料確認画面
- 月間訪問者数：10,000人
```

## プロンプトテンプレート

### 汎用テンプレート
```
[タスク/課題]について一緒に検討したい。
[目的]のため、必要に応じて質問してください。

例：
新商品開発について一緒に検討したい。
市場ニーズを正確に把握するため、必要に応じて質問してください。
```

### 段階的深掘りテンプレート
```
ステップ1：現状把握
「現在の[対象]の状況を整理したい。詳細を質問してください。」

ステップ2：課題特定
「把握した現状から、主要な課題を3つ特定してください。」

ステップ3：解決策検討
「特定した課題に対して、実現可能な解決策を提案してください。」
```

## よくある失敗と対処法

### 失敗例1：曖昧な回答
**問題：** AIの質問に対して「普通」「まあまあ」と答える
**対処：** 数値や具体例を使って回答する

### 失敗例2：情報の出し惜しみ
**問題：** 重要な制約条件を後から追加する
**対処：** 最初に主要な制約を全て伝える

### 失敗例3：対話の中断
**問題：** 1-2回のやり取りで満足してしまう
**対処：** 最低3-5回は対話を続ける

## 発展的な使い方

### 1. ロールプレイ型
```
あなたは経営コンサルタントとして、
私の事業計画をレビューしてください。
厳しい質問も歓迎します。
```

### 2. 段階的詳細化
```
まず大枠のアイデアを3つ提示してください。
選んだ1つについて、さらに詳細を詰めていきます。
```

### 3. 比較検討型
```
AプランとBプランのメリット・デメリットを
整理しながら、最適解を導きたい。
それぞれの詳細を質問してください。
```
