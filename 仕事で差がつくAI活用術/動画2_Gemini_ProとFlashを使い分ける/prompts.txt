# Gemini Pro/Flash 使い分けプロンプト集

## モデル選択の判断基準

### Gemini Flash向けタスク（速度優先）
- 大量のデータ分類
- 簡単な要約作成
- 下書きの生成
- アイデアの初期ブレスト
- 定型文書の作成

### Gemini Pro向けタスク（精度優先）
- 詳細な分析レポート
- 企画書・提案書の作成
- 顧客向け文書の作成
- 複雑な問題解決
- 戦略立案

## 実践プロンプト例

### Flash用プロンプト

#### 1. 議事録の要約
```
以下の議事録を3つのポイントで要約してください。
[議事録内容]
```

#### 2. メール分類
```
以下のメール100件を「緊急」「重要」「通常」「情報共有」に分類してください。
[メールリスト]
```

#### 3. アイデア出し
```
新商品のネーミング案を20個提案してください。
条件：若者向け、健康志向、覚えやすい
```

### Pro用プロンプト

#### 1. 市場分析
```
以下のデータを基に、今後3年間の市場動向を分析し、
当社が取るべき戦略を3つ提案してください。
根拠となる数値も含めて説明してください。
[データ]
```

#### 2. 提案書作成
```
以下の条件で、経営層向けのDX推進提案書を作成してください。
予算：2000万円
期間：1年
目標：業務効率30%改善
[現状の課題]
```

#### 3. 顧客対応文書
```
以下のクレーム内容に対して、誠意ある謝罪と
具体的な改善策を含む回答文を作成してください。
[クレーム内容]
```

## 処理時間の目安

### Flash
- 100文字要約：1-2秒
- 1000文字要約：2-3秒
- アイデア20個：3-5秒

### Pro
- 100文字要約：3-4秒
- 1000文字要約：5-8秒
- 詳細分析：10-15秒

## 使い分けのコツ

1. **二段階処理**
   - Flash で下書き → Pro で仕上げ

2. **優先度判断**
   - 社内用 → Flash
   - 社外用 → Pro

3. **コスト意識**
   - 試行錯誤段階 → Flash
   - 最終版作成 → Pro
