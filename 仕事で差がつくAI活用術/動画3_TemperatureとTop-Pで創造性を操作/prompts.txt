# Temperature & Top-P パラメータ活用プロンプト集

## パラメータ設定の基本

### Temperature（創造性の度合い）
- 0.1 - 0.3：堅実・保守的
- 0.4 - 0.6：バランス型
- 0.7 - 0.9：創造的・革新的

### Top-P（選択肢の幅）
- 0.1 - 0.3：限定的
- 0.4 - 0.6：適度
- 0.7 - 1.0：幅広い

## 実践的な調整方法

### ステップ1：中間値から開始
```
初期設定：Temperature: 0.5 / Top-P: 0.5
「新サービスの名称を提案してください」
```

### ステップ2：結果を見て調整
- もっと堅実に → Temperatureを下げる
- もっと創造的に → Temperatureを上げる
- 選択肢を絞る → Top-Pを下げる
- 選択肢を広げる → Top-Pを上げる

### ステップ3：微調整
```
調整後：Temperature: 0.7 / Top-P: 0.6
「先ほどより創造的で、でも現実的な範囲で再提案してください」
```

## 注意事項

### やりすぎ注意
- Temperature 0.9以上：非現実的になりがち
- Temperature 0.1以下：同じような答えばかり
- Top-P 0.1以下：選択肢が狭すぎる
- Top-P 1.0：ノイズが多くなる

### おすすめの組み合わせ
- 安全重視：T=0.2, P=0.3
- バランス型：T=0.5, P=0.5
- 創造性重視：T=0.7, P=0.7
