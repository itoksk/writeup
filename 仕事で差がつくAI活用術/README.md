# 仕事で差がつくAI活用術：プロンプト・モデル選択・創造性コントロール

## 研修概要

AIツールを導入したものの、効果的に活用できていない組織向けの実践的研修プログラムです。
対話型プロンプト、モデル選択、パラメータ調整を段階的に学び、AI活用スキルを向上させます。

## 対象者

- AIツールを使い始めたが、効果を実感できていない方
- ChatGPTやGeminiの基本操作は理解しているが、応用できていない方
- 業務効率を改善したいビジネスパーソン

## 期待効果

### Before（現状）
- AI活用率：15%未満
- プロンプト作成：平均15分
- 対話回数：1往復で終了
- 満足度：30%

### After（研修後）
- AI活用率：70%以上
- プロンプト作成：平均5分
- 対話回数：3-5往復の深掘り
- 満足度：70%
- **作業効率：25%改善**

## カリキュラム構成

### 動画1：対話型プロンプトで答えを深掘り（9分）
AIを「相棒」として活用し、対話を通じて具体的な答えを引き出す方法を学びます。

**学習内容：**
- 対話型プロンプトの3ステップ
- AIに質問させる技術
- 具体的な回答の仕方

### 動画2：Gemini ProとFlashを使い分ける（10分）
処理速度と精度のバランスを考慮し、タスクに応じてモデルを選択する方法を学びます。

**学習内容：**
- Pro/Flashの特性理解
- タスク別の使い分け基準
- 二段階処理の活用法

### 動画3：TemperatureとTop-Pで創造性を操作（9分）
パラメータ調整により、AIの創造性を意図的にコントロールする方法を学びます。

**学習内容：**
- Temperature（創造性）の調整
- Top-P（選択肢の幅）の設定
- タスク別の最適値

### 動画4：プロンプト最適化術総まとめ（10分）
学んだ要素を統合し、実務で成果を出すための総合的なAI活用法を実践します。

**学習内容：**
- 4ステップフレームワーク
- 実践シナリオ演習
- トラブルシューティング

## 各ファイルの説明

### script.md
講師が実際に話すセリフを詳細に記載した台本です。
時間配分、画面表示の指示、デモの手順が含まれています。

### materials.html
インタラクティブなHTMLプレゼンテーション資料です。
- **Fキー**：全画面プレゼンモード開始
- **矢印キー**：スライド移動
- **G**：スライド一覧表示
- **L**：レーザーポインタ
- 編集・並び替え機能付き

### prompts.txt
動画内で使用するプロンプト例と実践的な応用例をまとめています。
コピー＆ペーストですぐに試せる形式で提供。

### demo_code/
デモンストレーション用のサンプルコードやデータを格納。
（必要に応じて追加）

## 研修の進め方

### 準備
1. 各動画のscript.mdを事前に確認
2. materials.htmlをブラウザで開き、動作確認
3. prompts.txtから使用するプロンプトを選択

### 実施
1. materials.htmlを画面共有
2. script.mdに沿って説明
3. 実際のツール（ChatGPT/Gemini）でデモ実演
4. prompts.txtのプロンプトを活用

### フォローアップ
- 受講者にprompts.txtを配布
- 実務での実践を促す
- 質疑応答の時間を設ける
