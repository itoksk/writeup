# 動画4 プロンプト最適化術総まとめ - デモ用プロンプト集

## メインデモ：営業部向け新規事業提案書作成

### ステップ1：対話で背景整理

#### 初回プロンプト
```
営業部向けの新規事業提案を作成したい。
現状理解のため質問してください。
```

#### AIとの対話例
```
AI: 営業部の現在の主力商品と課題を教えてください。

You: BtoB向けSaaSツール。
新規開拓が頭打ちで、既存顧客の単価向上が課題。

AI: 予算規模と期待する投資回収期間は？

You: 初期投資1000万円以内、2年で黒字化希望。

AI: ターゲット顧客の業種や規模は？

You: 中堅企業（従業員100-500名）の製造業・卸売業

AI: 競合他社との差別化ポイントは？

You: カスタマイズ性とサポート体制の充実
```

### ステップ2：モデル選択で草案作成（Gemini Pro使用）

```
プロンプト：
上記の条件で、既存顧客向けのアップセル商品として、
AI活用支援サービスの提案書の構成案を作成してください。

期待される出力：
1. エグゼクティブサマリー
2. 市場分析と機会
3. サービス概要
4. ビジネスモデル
5. 収益予測
6. 実施計画
7. リスクと対策
```

### ステップ3：パラメータ調整でアイデア追加（Temperature=0.8）

```
プロンプト：
この提案に含める、他社と差別化できる
ユニークなサービス要素を5つ提案してください。

結果例：
1. AIによる商談同席サービス
2. 成果連動型の料金体系
3. 顧客企業の従業員向けAI研修
4. 24時間AIサポートボット
5. 競合分析レポートの自動生成
```

## 統合実践シナリオ集

### シナリオ1：マーケティングキャンペーン企画

#### Phase 1：現状分析（対話型）
```
初回プロンプト：
新商品のマーケティングキャンペーンを企画したい。
戦略立案のため、必要な情報を質問してください。

準備する回答：
- 商品：健康管理アプリ
- ターゲット：30-40代ビジネスパーソン
- 予算：500万円
- 期間：3ヶ月
- 目標：ダウンロード数10,000件
```

#### Phase 2：アイデア生成（Flash + 高Temperature）
```
設定：Gemini Flash, Temperature=0.8, Top-P=0.8

プロンプト：
健康管理アプリのプロモーション施策を20個提案してください。
従来にない斬新なアイデアを含めてください。
```

#### Phase 3：詳細企画（Pro + 低Temperature）
```
設定：Gemini Pro, Temperature=0.3, Top-P=0.4

プロンプト：
選定した3つの施策について、
実行計画、予算配分、KPI、スケジュールを含む
詳細な企画書を作成してください。
```

### シナリオ2：業務改善提案

#### Phase 1：課題の深掘り（対話型）
```
初回プロンプト：
経理部門の月次決算を効率化したい。
現状把握のため、詳細を質問してください。

対話の流れ：
- 現在の所要時間：5営業日
- ボトルネック：データ収集と照合作業
- 使用ツール：Excel中心
- 人員：5名
- エラー率：月3-5件
```

#### Phase 2：解決策リスト（Flash）
```
設定：Gemini Flash, Temperature=0.6

プロンプト：
上記の課題を解決する施策を、
実現難易度別（低・中・高）に分類して
15個提案してください。
```

#### Phase 3：実装計画（Pro）
```
設定：Gemini Pro, Temperature=0.3

プロンプト：
選定した施策について、
3ヶ月の段階的導入計画を作成してください。
各月のタスク、必要リソース、期待効果を明記。
```

## 4ステップフレームワーク実践

### 実践例1：新サービス企画

#### 1. 明確化
```
プロンプト：
高齢者向けの新しいITサービスを企画したい。
制約条件：
- 初期投資：500万円
- 開発期間：6ヶ月
- 想定ユーザー数：1万人
```

#### 2. 選択
```
分析フェーズ：Gemini Pro
アイデア出し：Gemini Flash
詳細設計：Gemini Pro
```

#### 3. 調整
```
市場分析：T=0.2, P=0.3（正確性重視）
サービスアイデア：T=0.7, P=0.7（創造性重視）
ビジネスプラン：T=0.4, P=0.5（バランス）
```

#### 4. 対話
```
継続的な質問：
「このサービスの最大の価値は何ですか？」
「競合との差別化要因を3つ挙げてください」
「収益化の具体的な方法は？」
```

## 実務で使える統合プロンプトテンプレート

### テンプレート1：問題解決型
```
【背景整理】
[問題]について解決策を検討したい。
現状把握のため、必要な情報を質問してください。

【アイデア生成】（Flash, T=0.7）
把握した課題に対して、解決策を20個提案してください。

【評価・選定】（Pro, T=0.3）
提案された解決策を実現可能性と効果で評価し、
上位3つを選定してください。

【実行計画】（Pro, T=0.2）
選定した解決策の詳細な実行計画を作成してください。
```

### テンプレート2：企画立案型
```
【コンセプト開発】（対話型）
[対象]向けの新サービスを企画したい。
企画に必要な情報を収集するため、質問してください。

【市場分析】（Pro, T=0.2）
ターゲット市場の規模、成長性、競合状況を分析してください。

【差別化戦略】（Flash→Pro）
差別化要素のアイデアを出し（Flash, T=0.8）、
その後詳細化してください（Pro, T=0.4）。

【事業計画】（Pro, T=0.3）
3年間の事業計画を作成してください。
```
