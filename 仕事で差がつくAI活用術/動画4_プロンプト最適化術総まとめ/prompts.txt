# プロンプト最適化術 総合実践集

## 4ステップフレームワーク

### 1. 明確化（ゴール設定）
### 2. 選択（モデル選定）
### 3. 調整（パラメータ設定）
### 4. 対話（深掘り）

---

## 実践シナリオ1：営業提案書作成

### ステップ1：明確化
```
「営業部向けの新規事業提案書を作成したい。
条件：
- 予算：1000万円以内
- 期間：1年
- 目標：既存顧客の単価20%向上
現状理解のため質問してください。」
```

### ステップ2：選択
**使用モデル：Gemini Pro**（精度重視）

### ステップ3：調整
**Temperature: 0.4 / Top-P: 0.5**（バランス型）

### ステップ4：対話
```
AI: 「既存顧客の業種と現在の平均単価を教えてください」
You: 「製造業中心、平均単価は月50万円」
AI: 「単価向上の障壁となっている要因は何ですか？」
You: 「付加価値サービスの不足」
```

### 最終プロンプト
```
「上記の条件で、製造業向けAI活用支援サービスの
提案書構成を作成してください。
付加価値として、導入支援と運用サポートを含めてください。」
```

## 実践シナリオ2：マーケティング企画

### ステップ1：明確化
```
「若年層向けの新商品マーケティング戦略を立案したい。
ターゲット：20-30代、予算：500万円
目標：認知度30%達成（3ヶ月）
詳細把握のため質問してください。」
```

### ステップ2：選択
**初期案：Gemini Flash**（スピード重視でアイデア出し）
**最終案：Gemini Pro**（詳細戦略の策定）

### ステップ3：調整
**初期案：Temperature: 0.8 / Top-P: 0.8**（創造的）
**最終案：Temperature: 0.3 / Top-P: 0.4**（実務的）

### ステップ4：対話
```
Flash使用：「SNSを活用した斬新なキャンペーンアイデアを20個」
↓
Pro使用：「上記から選んだ3つのアイデアを具体的な実行計画に」
```

---

## 実践シナリオ3：業務改善提案

### ステップ1：明確化
```
「経理部門の月次決算業務を効率化したい。
現在：5営業日→目標：3営業日
AIツールの活用を前提に検討。
現状分析のため質問してください。」
```

### ステップ2：選択
**分析：Gemini Pro**
**解決策リスト：Gemini Flash**
**実装計画：Gemini Pro**

### ステップ3：調整
**分析：T=0.2, P=0.3**（正確性重視）
**解決策：T=0.6, P=0.7**（幅広いアイデア）
**計画：T=0.3, P=0.4**（実現可能性重視）

### ステップ4：対話の流れ
```
1. 現状の問題点を深掘り
2. 解決策を幅広く検討
3. 実現可能な案に絞り込み
4. 具体的な実装計画を策定
```

## プロンプト改善のチェックリスト

### ✅ 明確化
- [ ] ゴールは明確か
- [ ] 制約条件は記載したか
- [ ] 期待する成果は伝えたか

### ✅ 選択
- [ ] タスクの性質を判断したか
- [ ] 速度vs精度の優先度は決めたか
- [ ] 二段階処理が必要か検討したか

### ✅ 調整
- [ ] 創造性のレベルは適切か
- [ ] 選択肢の幅は適切か
- [ ] タスクに合った設定か

### ✅ 対話
- [ ] AIに質問を促したか
- [ ] 具体的に回答したか
- [ ] 追加情報を提供したか

## トラブルシューティング

### 問題：答えが抽象的
→ 解決：具体例を要求、数値を含める指示

### 問題：処理が遅い
→ 解決：Flashに切り替え、分割処理

### 問題：創造性が足りない
→ 解決：Temperature上げる、制約を緩める

### 問題：非現実的な提案
→ 解決：Temperature下げる、実現可能性を条件に追加
