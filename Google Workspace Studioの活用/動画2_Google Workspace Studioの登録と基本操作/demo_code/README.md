# 第2回：Google Workspace Studioの登録と基本操作 - デモファイル

## 概要

第2回は、Studioの画面操作を中心としたハンズオンです。コード類は使用しません。

## デモで使用する操作

### 1. Studioへのアクセス
- URL: https://studio.workspace.google.com
- Googleアカウントでサインイン

### 2. テンプレート選択
- 「Recap unread emails」テンプレートを使用

### 3. エージェント設定画面の操作
- スターターの確認・調整
- Geminiステップの確認
- 通知ステップの確認
- 保存

### 4. テスト実行
- 「Run now」ボタンで即座に実行
- 実行結果の確認
- Chatでの通知確認

### 5. エージェント管理
- My agentsページの確認
- 操作メニューの説明
- 共有機能の説明

## 画面キャプチャ推奨箇所

1. Discoverページ全体
2. テンプレート一覧
3. エージェント設定画面（スターター）
4. エージェント設定画面（Geminiステップ）
5. エージェント設定画面（通知ステップ）
6. 保存完了画面
7. テスト実行中の画面
8. テスト実行結果
9. Chatでの通知メッセージ
10. My agentsページ
11. 共有ダイアログ

## 注意事項

- UIは英語のため、重要なボタン名は英語で読み上げてください
- 実際の未読メールがある状態でデモを行うと、リアルな結果が得られます
- テスト用のGmailに事前にテストメールを送っておくと良いでしょう
