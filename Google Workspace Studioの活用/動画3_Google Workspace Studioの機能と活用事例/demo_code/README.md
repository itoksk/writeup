# 第3回：Google Workspace Studioの機能と活用事例 - デモファイル

## 概要

第3回は、Studioの機能と活用事例の紹介が中心です。各事例のフロー図を視覚的に示すことが重要です。

## デモで使用する資料

### 1. スターター一覧図
- イベントベーススターター（Gmail、Forms、Drive、Chat、Calendar）
- スケジュールベーススターター（時間指定、頻度指定）

### 2. Gemini機能マトリックス
- 要約・抽出
- 分析・分類
- コンテンツ生成
- 意思決定支援

### 3. サードパーティ連携図
- 連携可能なサービス一覧
- 連携フローの例

### 4. 活用事例フロー図（4つ）

#### 事例1：フォーム回答通知
```
[Formに回答送信] → [Geminiで要約・分類] → [Chatに通知] → [Sheetsに記録]
```

#### 事例2：Driveファイル要約
```
[Driveにファイルアップロード] → [Geminiで要約] → [Chatに要約+リンク通知]
```

#### 事例3：会議ブリーフィング
```
[会議30分前] → [参加者・議題収集] → [Geminiで要約] → [Chatに通知]
```

#### 事例4：サポートトリアージ
```
[サポートメール受信] → [Geminiで分析・優先度判定] → [担当者Chatに通知] → [Sheetsにログ]
```
