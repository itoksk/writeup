# 動画1: AppSheetとは？基本画面とデータ連携の仕組み - Demo Code

このディレクトリには、動画1で使用するデモファイルが含まれています。

## ファイル一覧

### 1. 顧客.csv
初回デモで使用する顧客データのサンプルファイルです。AppSheetにアップロードして、アプリの自動生成を体験します。

**含まれるデータ:**
- ID顧客（主キー）
- 顧客名
- カテゴリ（法人/個人事業主）
- 顧客住所
- 顧客電話番号
- 顧客メールアドレス
- 作成日時
- 最終編集日時

**サンプル件数:** 5件

**使い方:**
1. Google Driveを開く
2. 「新規」→「ファイルのアップロード」で「顧客.csv」をアップロード
3. アップロード後、ファイルを右クリック→「アプリで開く」→「Googleスプレッドシート」
4. AppSheetで「Start with your own data」を選択し、このスプレッドシートを選択

### 2. appsheet_tabs_overview.csv
AppSheet Editorの6つの主要タブの役割と機能をまとめたCSVファイルです。

**含まれる情報:**
- タブ名
- 役割
- 主な機能
- 今回の研修での使用頻度

**用途:**
- 動画内で6つのタブを説明する際の参照資料
- 受講者が後で復習する際の早見表

### 3. basic_terms.csv
AppSheetの基本用語（テーブル、ビュー、アクション、オートメーション、スライス）をまとめたCSVファイルです。

**含まれる情報:**
- 用語名（英語）
- 用語名（日本語）
- 説明
- 具体例
- 今後の動画での登場頻度

**用途:**
- 動画内で基本用語を解説する際の参照資料
- 受講者が後で復習する際の用語集
