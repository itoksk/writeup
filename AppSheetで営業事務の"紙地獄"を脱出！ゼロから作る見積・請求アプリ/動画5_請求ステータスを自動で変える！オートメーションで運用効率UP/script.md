# 動画5: アクションとオートメーションで業務を自動化する

## 想定時間: 10分

## 学習目標
- アクション（Action）とオートメーション（Automation）の違いを理解する
- フラグを使ったトリガー設計を学ぶ
- 見積書・請求書発行の自動化フローを構築する
- エラーハンドリングとデバッグ方法を習得する

---

## アクションの設定（3分）

### 見積書発行アクション
1. **Actionsタブ→New Action**
2. **Nameを「見積書発行」に設定**
3. **Do thisで「Data: set the values of some columns in this row」を選択**
4. **「見積書発行フラグ」を「True」に設定**

---

## オートメーションの設定（4分）

### 見積書発行オートメーション
1. **Automationsタブ→Create a new bot**
2. **Eventで「Data Change」を選択**
3. **Tableで「注文」を選択**
4. **Conditionで `[見積書発行フラグ] = TRUE` を設定**
5. **Processで「Create a file from a template」を選択**
6. **最後に「見積書発行フラグ」を「False」に戻す**

---
