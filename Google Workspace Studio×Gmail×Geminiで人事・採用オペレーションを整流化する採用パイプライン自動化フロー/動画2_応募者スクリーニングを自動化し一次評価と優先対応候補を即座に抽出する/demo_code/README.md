# 第2回：応募者スクリーニングエージェント - デモファイル

## エージェント構成

```
[Gmail: 応募メール受信]
        ↓
[Gemini: スクリーニング]
        ↓
[Conditional: 適合度判定]
    ├─ HIGH → [Chat: 優先アラート] + [ラベル: 優先候補]
    ├─ MEDIUM → [Chat: 通知] + [ラベル: 検討中]
    └─ LOW → [ラベル: 不採用候補]
```

## テスト用サンプル応募メール

### サンプル1：適合度HIGH
```
Subject: エンジニア職への応募

採用ご担当者様

株式会社○○でバックエンドエンジニアとして5年間勤務しております。
Python、Java、AWSの経験があり、直近2年間はチームリーダーを担当しています。

履歴書・職務経歴書を添付いたします。
面接の機会をいただければ幸いです。
```

### サンプル2：適合度MEDIUM
```
Subject: 営業職からエンジニア職への転職希望

ご担当者様

現在営業職として3年勤務しておりますが、
プログラミングに興味を持ち、独学でPythonを学んでいます。
実務経験はありませんが、熱意を持って取り組みます。
```

### サンプル3：適合度LOW
```
Subject: アルバイト希望

お世話になります。
週2日程度のアルバイトを希望しています。
プログラミング経験はありません。
```
