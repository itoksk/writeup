# 第3回：面接日程調整の手間をゼロにし、Calendar連携で候補者対応を自動化する

**動画時間**: 10分
**学習目標**: Calendar連携で面接官の空き時間を取得し、候補者に日程候補を自動提案するエージェントを作成する

---

## 1. 導入（0:00-1:00）

### 講師セリフ

「皆さん、こんにちは。採用パイプライン自動化フロー、第3回をお届けします。

前回は、応募者スクリーニングエージェントを作成し、候補者の適合度を自動評価する仕組みを構築しました。

今回は、採用担当者が最も時間を取られる業務の一つ、**面接日程調整**を自動化します。

**（画面：メールのやりとりイメージ）**

面接日程調整って、本当に手間がかかりますよね。

- 面接官のカレンダーを確認
- 候補者に複数の日程候補を送る
- 候補者から返信が来たらカレンダーに登録
- 会議室も予約...

1件あたり15〜30分。月に20人の候補者がいれば、それだけで5〜10時間。

**（画面：自動化後のイメージ）**

このエージェントは、面接官の空き時間を自動で取得し、候補者に日程候補を自動送信。確定後はカレンダーに自動登録してくれます。」

---

## 2. エージェントの全体像（1:00-2:00）

### 講師セリフ

「今回作るエージェントの構成を確認しましょう。

**（画面：エージェントフロー図）**

### フロー

```
[書類通過メール送信] → [Calendar空き取得] → [日程候補生成] → [候補者にメール]
```

### 機能

1. **トリガー**：書類選考通過時に手動で起動 or ラベル付与で起動
2. **Calendar連携**：面接官の空き時間を自動取得
3. **Gemini生成**：日程候補メールを自動作成
4. **Gmail送信**：候補者に日程候補を送信（下書き or 自動送信）」

---

## 3. Calendar連携の設定（2:00-4:00）

### 講師セリフ

「新しいエージェントを作成します。

**（画面：Create → 新規エージェント）**

エージェント名：『面接日程自動調整エージェント』

### スターター設定

今回は手動起動にします。書類選考通過時に、担当者がボタンを押して起動する形です。

または、「採用_優先候補」ラベルが付いたときに自動起動することもできます。

### Calendar空き時間取得

**（画面：Add Step → Google Calendar）**

『Google Calendar』→『Get free/busy times』を選択します。

**（画面：Calendar設定）**

- **Calendar**: 面接官のカレンダーを選択
- **Date range**: 今日から2週間後まで
- **Duration**: 60分（面接時間）
- **Working hours**: 10:00-18:00

これで、面接官の空き時間が取得できます。」

---

## 4. 日程候補メールの生成（4:00-6:00）

### 講師セリフ

「次に、取得した空き時間をもとに、候補者へのメールを生成します。

**（画面：Add Step → Gemini）**

### プロンプト

```
Generate an interview scheduling email in Japanese:

Candidate: {candidate_name}
Position: {position}
Available slots:
{available_times}

Create a professional email that:
- Thanks for passing document screening
- Offers 3-5 time slot options
- Asks candidate to reply with preferred time
- Mentions interview duration (60 min)
- Includes next steps

Format time slots clearly.
```

**（画面：出力例）**

```
件名：【面接日程のご案内】〇〇株式会社 エンジニア職

〇〇様

この度は弊社エンジニア職にご応募いただき、誠にありがとうございます。
書類選考の結果、ぜひ面接にお越しいただきたく、ご連絡いたしました。

つきましては、下記日程のご都合をお聞かせください。

【候補日程】
1. 1月20日（月）14:00〜15:00
2. 1月21日（火）10:00〜11:00
3. 1月22日（水）15:00〜16:00
4. 1月23日（木）11:00〜12:00

面接時間は約60分を予定しております。
ご希望の日程を番号でご返信ください。

何卒よろしくお願いいたします。
```」

---

## 5. メール送信と確定処理（6:00-8:00）

### 講師セリフ

「生成されたメールを候補者に送信します。

**（画面：Add Step → Gmail）**

### Gmail送信設定

- **Action**: Create draft（下書き作成）or Send email（自動送信）
- **To**: {candidate_email}
- **Subject**: 【面接日程のご案内】...
- **Body**: {gemini_email}

安全のため、最初は**下書き作成**をお勧めします。内容を確認してから送信できます。

### 確定後の処理（別エージェント）

候補者から返信が来たら、別のエージェントで：

1. 返信メールを検知
2. 選択された日程を抽出
3. Calendarに予定を自動登録
4. 候補者に確定通知を送信

**（画面：Calendar登録設定）**

```
Google Calendar → Create event
- Title: 面接：{candidate_name}様
- Start: {selected_time}
- Duration: 60分
- Attendees: 面接官、候補者
```」

---

### 講師セリフ
