# 第3回：面接日程自動調整エージェント - デモファイル

## エージェント構成

```
[手動起動 / ラベルトリガー]
        ↓
[Calendar: 空き時間取得]
        ↓
[Gemini: 日程案内メール生成]
        ↓
[Gmail: 下書き作成 / 送信]
```

## 出力例：日程案内メール

```
件名：【面接日程のご案内】〇〇株式会社 エンジニア職

〇〇様

この度は弊社エンジニア職にご応募いただき、誠にありがとうございます。
書類選考の結果、ぜひ面接にお越しいただきたく、ご連絡いたしました。

つきましては、下記日程のご都合をお聞かせください。

【候補日程】
1. 1月20日（月）14:00〜15:00
2. 1月21日（火）10:00〜11:00
3. 1月22日（水）15:00〜16:00
4. 1月23日（木）11:00〜12:00

面接時間は約60分を予定しております。
ご希望の日程を番号でご返信ください。

何卒よろしくお願いいたします。
```
