# 第5回：選考ステータス自動更新 & KPIレポートエージェント - デモファイル

## エージェント1：ステータス自動更新

```
[面接結果メール受信]
        ↓
[Gemini: ステータス抽出]
        ↓
[Google Sheets: 行を更新]
```

## エージェント2：週次KPIレポート

```
[Schedule: 毎週月曜 9:00]
        ↓
[Google Sheets: データ取得]
        ↓
[Gemini: KPI集計・レポート生成]
        ↓
[Google Chat: チームスペースに送信]
```

## 出力例：KPIレポート

```
【採用KPIレポート】2025年1月第3週

■ 今週の応募
新規応募：12件（前週比 +3件）
応募経路：Indeed 5件、LinkedIn 4件、直接応募 3件

■ パイプライン状況
├ 書類選考中：8名
├ 一次面接済：5名
├ 二次面接済：3名
├ 最終面接済：2名
├ 内定：1名
└ 合計アクティブ：19名

■ 転換率
書類通過率：75%（9/12）
一次通過率：60%（3/5）
内定承諾率：100%（1/1）

■ 平均所要日数
書類→一次：3.2日
一次→二次：5.1日

■ 注目ポイント
・エンジニア職の応募が増加傾向
・一次→二次の移行に時間がかかっている
```

## ステータス定義

| ステータス | 説明 |
|---|---|
| 新規応募 | 応募を受け付けた直後 |
| 書類選考中 | 履歴書・職務経歴書を確認中 |
| 一次面接済 | 一次面接完了、結果待ち |
| 二次面接済 | 二次面接完了、結果待ち |
| 最終面接済 | 最終面接完了、内定検討中 |
| 内定 | オファー提示済み |
| 不採用 | 選考終了（不合格） |
| 辞退 | 候補者からの辞退 |
