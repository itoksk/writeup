# プロンプト集：第3回 AIが作るWBRサマリー＆是正案

**対象**: 初心者（ChatGPT無料版でOK）
**使用ツール**: ChatGPT、Google Sheets
**作業方法**: コピー&ペーストのみ、プログラミング不要

---

## プロンプト1：基本的なWBR報告書生成（最も使う）

スプレッドシートのKPIデータ、変動理由、営業日報をコピーして、ChatGPTに以下のプロンプトと一緒に貼り付けます。

```
以下のデータから、週次ビジネスレビュー（WBR）報告書を作成してください。

【今週のKPIデータ】
売上：前週1,480,000円 → 今週1,650,000円（+11.5%）
成約件数：前週26件 → 今週30件（+15.4%）
問い合わせ数：前週118件 → 今週135件（+14.4%）

【変動理由】
✅ プラス要因
- B社プロジェクト（420万円）が成約（2/10）
- 新商品キャンペーンの反響が好調（問い合わせ+14.4%）
- ウェビナー開催（2/8）による新規リード増加（25件）

⚠️ マイナス要因・注意すべき点
- 既存顧客のリピート率は微減（-3%）：要注意

【営業日報（今週の主要案件）】
2024-02-10	B社プロジェクト	4,200,000	成約	新規大口案件
2024-02-12	C社追加発注	800,000	成約	既存顧客リピート

【出力形式】
以下の形式でWBR報告書を作成してください：

## 📊 今週のKPIサマリー
- KPI項目ごとに、前週比を含めて記載

## ✅ 今週のハイライト
【主要成果】
- 具体的な案件名、金額、日付を含めて記載

【注意すべき点】
- リスクや懸念事項があれば記載

## 💡 次週へのアクション
- 具体的で実行可能なアクションプランを3〜5個提案
- 優先順位の高い順に記載
```

**使い方**：
1. Google SheetsでKPIデータ、変動理由、営業日報の今週分をコピー（Ctrl+C / Cmd+C）
2. ChatGPTを開く
3. 上記プロンプトの【データ】部分にコピーしたデータを貼り付け
4. 送信
5. ChatGPTの回答をコピーして、PowerPointやGoogleスライドに貼り付け

## プロンプト3：役員向け版（エグゼクティブサマリー重視）

役員や経営陣向けの簡潔な報告書が欲しい場合：

```
以下のデータから、役員向けの週次ビジネスレビュー（WBR）報告書を作成してください。
簡潔で、戦略的な視点を重視してください。

【今週のKPIデータ】
売上：前週1,480,000円 → 今週1,650,000円（+11.5%）
成約件数：前週26件 → 今週30件（+15.4%）
問い合わせ数：前週118件 → 今週135件（+14.4%）

【変動理由】
✅ プラス要因
- B社プロジェクト（420万円）が成約（2/10）
- 新商品キャンペーンの反響が好調（問い合わせ+14.4%）
- ウェビナー開催（2/8）による新規リード増加（25件）

⚠️ マイナス要因・注意すべき点
- 既存顧客のリピート率は微減（-3%）：要注意

【過去4週間のトレンド】
第1週：売上1,350,000円、成約22件
第2週：売上1,420,000円、成約24件
第3週：売上1,480,000円、成約26件
今週：売上1,650,000円、成約30件

【出力形式】
以下の形式で役員向けWBR報告書を作成してください：

# エグゼクティブサマリー
今週の最重要ポイントを1-2文で簡潔に

## 📊 KPIパフォーマンス
- 売上、成約件数、問い合わせ数の前週比とトレンド（上昇/横ばい/下降）

## ✅ 主要成果と課題
【成果】
- 戦略的に重要な成果を2-3個

【課題・リスク】
- 組織として注意すべき課題を1-2個

## 💡 戦略的アクション
1. 短期（1-2週間）：即座に取り組むべきこと
2. 中期（1ヶ月）：仕組みとして改善すべきこと
3. リソース配分：予算・人員の最適化提案

【注意事項】
- エグゼクティブレベルの視点で
- データだけでなく、戦略的な示唆を含める
- 簡潔に、要点だけに絞る
```

週次データをまとめて月次報告を作りたい場合：

```

【月次KPIデータ】
第1週（2024-02-05）：売上1,350,000円、成約22件、問い合わせ105件
第2週（2024-02-12）：売上1,420,000円、成約24件、問い合わせ110件
第3週（2024-02-19）：売上1,480,000円、成約26件、問い合わせ118件
第4週（2024-02-26）：売上1,650,000円、成約30件、問い合わせ135件
月間合計：売上5,900,000円、成約102件、問い合わせ468件

【前月との比較】
売上：前月5,200,000円 → 今月5,900,000円（+13.5%）
成約件数：前月88件 → 今月102件（+15.9%）

【主要トピックス（各週の変動理由から抽出）】
- 第1週：新商品発表
- 第2週：キャンペーン開始
- 第3週：ウェビナー開催
- 第4週：B社大口案件成約

【出力形式】
以下の形式で月次ビジネスレビュー（MBR）報告書を作成してください：

# 月次ビジネスレビュー（MBR）
対象月：2024年2月

## 📊 月次KPIサマリー
- 売上：XXX円（前月比X% / 目標達成率X%）
- 成約件数：XX件（前月比X% / 目標達成率X%）

## 📈 週次トレンド分析

- 第2週：[状況]
- 第3週：[状況]
- 第4週：[状況]
```

## 応用編：GPTsで専用アシスタントを作る

**ChatGPT Plus（月20ドル）が必要です。**

GPTs（カスタムGPT）を作ると、毎回同じプロンプトを書く必要がなくなります。

### GPTs Instructions（コピーして使用）

```
あなたはビジネスレポート作成の専門家です。
ユーザーがKPIデータ、変動理由、営業日報を貼り付けたら、週次ビジネスレビュー（WBR）報告書を作成してください。

【役割】
1. 提供されたデータを分析し、WBR報告書を作成
2. KPIサマリー、ハイライト、次週へのアクションの3つに分けて整理
3. 具体的な案件名、金額、日付、数値を含めて記載

【出力形式】
## 📊 今週のKPIサマリー
- KPI項目ごとに前週比を含めて記載
- プラス/マイナスを明確に表示

## ✅ 今週のハイライト
【主要成果】
- 具体的な成果を3〜5個記載
- 案件名、金額、日付を必ず含める

【注意すべき点】
- リスクや課題があれば記載
- 数値の根拠を示す

## 💡 次週へのアクション
- 具体的で実行可能なアクションを3〜5個提案
- 優先順位の高い順に記載
- 期待される効果も簡潔に記載

【注意事項】
- 推測ではなく、提供されたデータに基づいた分析のみ行う
- 固有名詞（案件名、顧客名、イベント名）と数値（金額、件数、日付）を必ず含める
- 簡潔に、要点を絞って記載
- 次週へのアクションは具体的かつ実行可能なものに限定
- ビジネス報告として適切なトーンを維持
```

### GPTs作成手順

1. ChatGPT Plusにログイン
2. 『Explore GPTs』をクリック
3. 『Create a GPT』をクリック
4. 『Configure』タブを開く
5. 『Instructions』欄に上記のInstructionsをコピー&ペースト
6. 名前を『WBRレポート作成アシスタント』などに設定
7. 保存

これで、毎週データをコピー&ペーストするだけで、専用アシスタントがWBR報告書を作成してくれます。

## よくある質問（FAQ）

### Q1: ChatGPT無料版でも使えますか？
**A**: はい、無料版（GPT-3.5）で十分使えます。より高精度を求める場合は有料版（GPT-4）を使うとさらに良いですが、WBR報告書作成程度なら無料版で問題ありません。

### Q2: データをコピーする際、どの範囲を選択すればいいですか？
**A**: 以下のデータをコピーしてください：
- KPIデータ（今週と前週の数値、前週比）
- 変動理由（前回作成した分析結果）
- 営業日報（今週の主要案件）
- イベント情報（あれば）

### Q3: ChatGPTの回答をどうやってスライドに貼り付けますか？
**A**: ChatGPTの回答をそのままコピーして、PowerPointやGoogleスライドのテキストボックスに貼り付けてください。Markdown形式で出力されるので、そのまま使えます。

### Q4: 報告書の内容が期待と違います。どうすればいいですか？
**A**: プロンプトを調整してください。例：
- 『もっと簡潔に』
- 『もっと詳しく』
- 『アクションプランをもっと具体的に』
- 『役員向けにもっと戦略的な視点で』

### Q5: 毎週同じプロンプトを書くのが面倒です。
**A**: プロンプトをテキストファイルやメモ帳に保存しておき、毎週【データ】部分だけ差し替えて使いましょう。または、GPTsを作成すると、さらに楽になります。

### Q6: 他のKPI項目（顧客満足度、解約率など）も含めた報告書を作れますか？
**A**: はい、プロンプトの【KPIデータ】部分を変更すれば、どんな指標でも対応できます。

### Q7: 月次報告（MBR）や四半期報告（QBR）にも使えますか？
**A**: はい、「プロンプト5」を使えば、月次や四半期の報告書も作成できます。週次データをまとめて入力してください。

## トラブルシューティング

### 問題1: ChatGPTが報告書を作成してくれない

**症状**: プロンプトを送っても、説明だけ返ってくる

**解決策**: プロンプトの最後に「必ず上記の形式で報告書を出力してください」と追加
```
【出力形式】
以下の形式で、必ずWBR報告書を出力してください：
## 📊 今週のKPIサマリー
...
```

### 問題2: アクションプランが抽象的すぎる

**症状**: 『売上向上』『顧客満足度改善』など、抽象的なアクションしか出てこない

**解決策**: プロンプトに以下を追加
```
【注意事項】
- アクションプランは具体的で実行可能なものに限定
- 『誰が、何を、いつまでに』がわかるレベルで記載
- 例：『B社類似企業5社へ、営業チームが来週中にアプローチ』
```

### 問題3: 報告書が長すぎる

**症状**: ChatGPTの出力が長すぎて、報告会議で使えない

**解決策**: プロンプトで文字数制限を指定
```
【注意事項】
- 簡潔に、要点だけに絞る
- 各セクションは3〜5個の項目に限定
- 1項目は1〜2文で記載
```

### 問題4: データの鮮度が古い

**症状**: ChatGPTが古いデータを参照している

**解決策**: ChatGPTは提供されたデータのみを使用します。必ず最新のデータをコピーして貼り付けてください。

## 活用のコツ

### コツ1: プロンプトをテンプレート化
一度プロンプトを作ったら、テキストファイル（.txt）やメモアプリに保存しておきます。
毎週、【データ】部分だけ差し替えれば、5分で完了します。

### コツ2: ChatGPTに「確認」を求める
報告書の内容に自信がない場合は、ChatGPTに追加質問：
```
この報告書の内容は、提供したデータに基づいていますか？推測している部分があれば教えてください。
```

### コツ3: フィードバックを与えて改善
ChatGPTの出力が期待と違う場合は、フィードバックを与えて改善：
```
もっと簡潔にしてください。各項目は1文で記載してください。
```

### コツ4: 複数バージョンを作る
役員向け、チーム向け、取引先向けなど、複数のバージョンのプロンプトを用意しておくと便利です。
