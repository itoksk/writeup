# プロンプト集：第1回 KPI台帳をAIが自動で書き換える仕組みをつくる

**対象**: 初心者（ChatGPT無料版でOK）
**使用ツール**: ChatGPT、Google Sheets
**作業方法**: コピー&ペーストのみ、プログラミング不要

---

## プロンプト1：基本的な前週比計算（最も使う）

スプレッドシートのデータをコピーして、ChatGPTに以下のプロンプトと一緒に貼り付けます。

```
以下のスプレッドシートデータから、前週比（%）を計算してください。

【データ】
2024-02-05	1480000	26	118
2024-02-12	1650000	30	135

【計算対象】
- 売上（2列目）
- 成約件数（3列目）
- 問い合わせ数（4列目）

【計算式】
前週比（%）= ((今週の値 - 前週の値) / 前週の値) × 100

【出力形式】
以下の形式で、小数点第1位まで表示してください：
- 売上前週比：XX.X%
- 成約件数前週比：XX.X%
- 問い合わせ数前週比：XX.X%
```

**使い方**：
1. Google Sheetsで最新2行（日付、売上、成約件数、問い合わせ数）を選択してコピー（Ctrl+C / Cmd+C）
2. ChatGPTを開く
3. 上記プロンプトの【データ】部分にコピーしたデータを貼り付け
4. 送信
5. ChatGPTの回答をコピーして、スプレッドシートの前週比列（E〜G列）に貼り付け

## プロンプト2：複数週分をまとめて計算

過去4週分のデータを一度に計算したい場合：

```
以下のスプレッドシートデータから、各週の前週比（%）を計算してください。

【データ】
2024-01-22	1350000	22	105
2024-01-29	1420000	24	110
2024-02-05	1480000	26	118
2024-02-12	1650000	30	135

【計算対象】
- 売上（2列目）
- 成約件数（3列目）
- 問い合わせ数（4列目）

【計算式】
前週比（%）= ((今週の値 - 前週の値) / 前週の値) × 100

【出力形式】
各週について、以下の形式で出力してください：

■ 2024-01-29の前週比
- 売上前週比：XX.X%
- 成約件数前週比：XX.X%
- 問い合わせ数前週比：XX.X%

■ 2024-02-05の前週比
- 売上前週比：XX.X%
- 成約件数前週比：XX.X%
- 問い合わせ数前週比：XX.X%

■ 2024-02-12の前週比
- 売上前週比：XX.X%
- 成約件数前週比：XX.X%
- 問い合わせ数前週比：XX.X%
```

**使い方**：
- 過去4週分のデータを選択してコピー
- プロンプトの【データ】部分に貼り付けて送信

## プロンプト3：計算過程も確認したい場合

計算が正しいか確認したい場合は、計算過程も表示してもらいます：

```
以下のスプレッドシートデータから、前週比（%）を計算してください。
計算過程も詳しく教えてください。

【データ】
2024-02-05	1480000	26	118
2024-02-12	1650000	30	135

【計算対象】
- 売上（2列目）
- 成約件数（3列目）
- 問い合わせ数（4列目）

【計算式】
前週比（%）= ((今週の値 - 前週の値) / 前週の値) × 100

【出力形式】
以下の形式で、計算過程と結果を表示してください：

■ 売上前週比
計算：(1,650,000 - 1,480,000) ÷ 1,480,000 × 100 = XX.X%
結果：+XX.X%

■ 成約件数前週比
計算：(30 - 26) ÷ 26 × 100 = XX.X%
結果：+XX.X%

■ 問い合わせ数前週比
計算：(135 - 118) ÷ 118 × 100 = XX.X%
結果：+XX.X%
```

**使い方**：
- 計算ミスが心配な場合や、上司に根拠を説明する必要がある場合に使用

## プロンプト4：前週のデータが0の場合の対処

前週のデータが0の場合、前週比は計算できません。その場合の指示：

```
以下のスプレッドシートデータから、前週比（%）を計算してください。
前週の値が0の場合は「N/A」と表示してください。

【データ】
2024-02-05	1480000	0	118
2024-02-12	1650000	30	135

【計算対象】
- 売上（2列目）
- 成約件数（3列目）
- 問い合わせ数（4列目）

【計算式】
前週比（%）= ((今週の値 - 前週の値) / 前週の値) × 100
※ただし、前週の値が0の場合は「N/A」と表示

【出力形式】
- 売上前週比：XX.X%
- 成約件数前週比：N/A （前週が0のため）
- 問い合わせ数前週比：XX.X%
```

【出力形式】

| 項目 | 前週 | 今週 | 前週比（%） |

- 問い合わせ数前週比：+XX.X% または -XX.X%

## よくある質問（FAQ）

### Q1: ChatGPT無料版でも使えますか？
**A**: はい、無料版（GPT-3.5）で十分使えます。より高精度を求める場合は有料版（GPT-4）を使うとさらに良いですが、前週比計算程度なら無料版で問題ありません。

### Q2: データをコピーする際、どの範囲を選択すればいいですか？
**A**: 最新2行の以下の列を選択：
- A列：日付
- B列：売上
- C列：成約件数
- D列：問い合わせ数

見出し行は含めなくてOKです。

### Q3: ChatGPTの回答をスプレッドシートに貼り付けるとき、うまく貼り付けられません。
**A**: ChatGPTの回答から数値部分だけをコピーして、一つずつセルに貼り付けてください。
例：「売上前週比：+11.5%」→ 「11.5」だけコピーして貼り付け

### Q4: 計算結果が間違っている気がします。確認方法は？
**A**: 「プロンプト3」を使って、計算過程も表示してもらいましょう。手計算で確認できます。

### Q5: 毎週同じプロンプトを書くのが面倒です。
**A**: プロンプトをテキストファイルやメモ帳に保存しておき、毎週【データ】部分だけ差し替えて使いましょう。

### Q6: 他のKPI項目（顧客満足度、解約率など）も計算できますか？
**A**: はい、プロンプトの【計算対象】部分を変更すれば、どんな指標でも計算できます。
例：
```
【計算対象】
- 顧客満足度（2列目）
- 解約率（3列目）
- NPS（4列目）
```

## トラブルシューティング

### 問題1: ChatGPTが計算してくれない

**症状**: プロンプトを送っても、計算せずに説明だけ返ってくる

**解決策**: プロンプトの最後に「必ず計算結果を数値で出力してください」と追加
```
【出力形式】
以下の形式で、必ず計算結果を数値で出力してください：
- 売上前週比：XX.X%
- 成約件数前週比：XX.X%
- 問い合わせ数前週比：XX.X%
```

### 問題2: 小数点以下の桁数がバラバラ

**症状**: 「11.5%」「15.43%」「14%」のように、桁数が揃わない

**解決策**: プロンプトで桁数を明確に指定
```
【出力形式】
小数点第1位まで必ず表示してください（例：11.5%、15.4%、14.0%）
```

### 問題3: 日付のフォーマットがおかしい

**症状**: コピーしたデータの日付が数値（例：45329）になってしまう

**解決策**: スプレッドシートで日付セルを選択 → 右クリック → 「表示形式」→ 「書式なしテキスト」に変更してからコピー

または、ChatGPTに以下のように指示：
```
【データ】
※1列目は日付です（シリアル値で表示されている場合があります）
45329	1480000	26	118
45336	1650000	30	135
```

## 活用のコツ

### コツ1: プロンプトをテンプレート化
一度プロンプトを作ったら、テキストファイル（.txt）やメモアプリに保存しておきます。
毎週、【データ】部分だけ差し替えれば、5分で完了します。

### コツ2: ChatGPTに「確認」を求める
計算結果に自信がない場合は、ChatGPTに追加質問：
```
計算結果は合っていますか？もう一度確認してください。
```

### コツ3: 複数週分まとめて処理
月末など、まとめて処理したい場合は「プロンプト2」を使って4週分を一度に計算。
時間がさらに短縮されます。

### コツ4: スプレッドシートに貼り付ける際の工夫
ChatGPTの回答から数値部分だけを抜き出して、以下のように一列で貼り付け：
```
11.5
15.4
14.4
```
こうすると、E列、F列、G列に一度に貼り付けられます。

## 応用編：他の用途にも使える

### 応用1: 前月比の計算
「前週比」を「前月比」に変更するだけで、月次データにも使えます：
```
【計算式】
前月比（%）= ((今月の値 - 前月の値) / 前月の値) × 100
```

### 応用2: 予算比の計算
予算と実績の比較にも使えます：
```
【データ】
予算	1500000	28	120
実績	1650000	30	135

【計算式】
予算比（%）= ((実績 - 予算) / 予算) × 100
```

### 応用3: 目標達成率の計算
```
【データ】
目標	2000000	35	150
実績	1650000	30	135

【計算式】
目標達成率（%）= (実績 / 目標) × 100
```
