# Google Workspace Studio バックオフィス業務ゼロ化講座

## コース概要
Google Workspace Studioを活用して、経費申請・勤怠管理・タスク管理などのバックオフィス業務を完全自動化する実践講座です。5つのレッスンで、従業員からの申請受付から承認処理、データ集計まですべてを自動化し、人手を介さないゼロタッチオペレーションを実現します。

## 対象者
- 経費精算や勤怠管理に多くの時間を費やしている経理・人事担当者
- バックオフィス業務の効率化を検討している管理部門マネージャー
- Google Workspaceを導入している企業の総務・経理担当者
- ルーティンワークを自動化したい業務改善担当者

## 前提知識
- Google Workspaceの基本操作（Gmail、スプレッドシート、カレンダー）
- プログラミング知識は不要（完全ノーコード）

## 達成目標
- 経費申請の受付から承認・集計までの完全自動化
- 勤怠データの自動収集と異常検知
- タスク割り当てと進捗追跡の自動化
- 月次レポートの自動生成と配信

## カリキュラム（全5回）

### 第1回：レシート撮影だけで経理完了！AIが金額・科目を自動抽出する経費申請フロー
**フォーム受付→レシート解析→自動仕分け**

- **学習目標**: 経費申請をGoogleフォームで受け付け、Geminiでレシート画像を解析し、科目を自動分類する
- **内容**:
  - Formsスターターでの経費申請受付設定
  - Geminiによるレシート画像の自動読み取り
  - 経費科目の自動分類ロジック
  - スプレッドシートへの自動記録
  - 申請者への受付完了自動通知
- **作成するエージェント**: 経費申請受付エージェント

### 第2回：5,000円未満は即承認！金額で振り分けるスマート承認フロー
**金額判定→承認ルーティング→自動承認**

- **学習目標**: 金額に応じた承認者への自動振り分けと、少額経費の自動承認を実装する
- **内容**:
  - 金額に基づく承認ルートの条件分岐
  - 少額経費（例：5,000円未満）の自動承認
  - 承認者へのSlack/Chat DM通知
  - 承認待ちリストのリアルタイム更新
  - 承認期限切れアラートと自動エスカレーション
- **作成するエージェント**: 経費承認ルーティングエージェント

### 第3回：36協定違反を未然に防ぐ！残業時間をリアルタイム監視するアラートシステム
**カレンダー連携→勤怠集計→アラート発報**

- **学習目標**: Googleカレンダーから勤怠情報を自動収集し、異常パターンを検知してアラートを出す
- **内容**:
  - カレンダーイベントからの勤務時間自動算出
  - 残業時間の自動計算と36協定チェック
  - 有給取得率の自動モニタリング
  - 異常勤怠パターンの検知とアラート
  - 週次勤怠サマリーの自動生成
- **作成するエージェント**: 勤怠モニタリングエージェント

### 第4回：期限切れゼロを実現！3日前・前日・当日の段階リマインド自動化
**タスク割り当て→進捗追跡→自動リマインド**

- **学習目標**: チームのタスク管理を自動化し、期限切れ前のリマインドを自動送信する
- **内容**:
  - スプレッドシートベースのタスク管理
  - 担当者への自動タスク通知
  - 期限3日前・前日の自動リマインド
  - 未完了タスクの自動エスカレーション
  - タスク完了率のダッシュボード連携
- **作成するエージェント**: タスクリマインダーエージェント

### 第5回：経営層への報告を完全自動化！AIが作る月次インサイトレポート
**データ集約→レポート生成→自動配信**

- **学習目標**: 月末に各種データを集約し、経営レポートを自動生成・配信する
- **内容**:
  - 経費・勤怠・タスクデータの月次集計
  - Geminiによる傾向分析とコメント生成
  - スライド形式の月次レポート自動作成
  - 経営層への自動配信スケジュール
  - 前月比較と異常値ハイライト
- **作成するエージェント**: 月次レポート自動生成エージェント

## 技術スタック
- Google Workspace Studio
- Gemini（画像解析・テキスト生成）
- Googleフォーム
- Googleスプレッドシート
- Googleカレンダー
- Google Chat / Slack連携

## 主要なステップ・機能

### スターター
| スターター | 用途 |
|-----------|------|
| Formsスターター | 経費申請・各種申請の受付 |
| スケジュールスターター | 定期的なデータ収集・レポート生成 |
| Gmailスターター | メール経由での申請受付 |

### ステップ
| ステップ | 用途 |
|---------|------|
| Geminiステップ | レシート解析、異常検知、レポート生成 |
| スプレッドシート連携 | データの読み書き、集計 |
| 条件分岐 | 金額判定、承認ルーティング |
| Chat DMステップ | 承認依頼、リマインド送信 |
| Driveステップ | レポートファイル操作 |

## 自動化される業務フロー

```
経費申請フロー：
フォーム申請 → レシート解析 → 科目分類 → 金額判定 →
承認者振り分け → 承認依頼 → 承認/却下 → 結果通知 → 月次集計

勤怠管理フロー：
カレンダー収集 → 勤務時間計算 → 残業チェック →
異常検知 → アラート発報 → 月次レポート

タスク管理フロー：
タスク登録 → 担当者通知 → 期限監視 →
リマインド → エスカレーション → 完了報告
```

## 達成できること
- 経費精算業務：90%削減
- 勤怠集計時間：週2時間→5分
- 月次レポート作成：8時間→0分（完全自動化）
- 承認処理の遅延：72時間→24時間以内

## 注意事項
- 外部APIキーは不要（Google Workspace内で完結）
- 対応エディション: Business Standard以上推奨
- 最初はテスト環境で動作確認を推奨

## 必要条件
- Google Workspace アカウント（Workspace Studio へのアクセス権）
- 経費管理用スプレッドシート
- 勤怠管理用スプレッドシート
- タスク管理用スプレッドシート
