# Google Workspace Studio セールスオペレーション効率化講座

## コース概要
Google Workspace Studioを活用して、商談管理・見積作成・顧客フォローなど営業業務を完全自動化する実践講座です。5つのレッスンで、リード獲得から受注後フォローまでを自動化し、営業担当者が本来の商談に集中できる環境を実現します。

## 対象者
- 商談管理やフォローアップに多くの時間を費やしている営業担当者
- 見積作成・送付の効率化を検討している営業マネージャー
- 営業活動の可視化・分析を自動化したい営業企画担当者
- CRMを導入していないがGoogle Workspaceを活用している中小企業

## 前提知識
- Google Workspaceの基本操作（Gmail、カレンダー、スプレッドシート）
- プログラミング知識は不要（完全ノーコード）

## 達成目標
- リード情報の自動収集と優先度判定
- 商談進捗の自動追跡とフォローアップリマインド
- 見積書の自動生成と送付
- 受注・失注分析の自動レポート生成

## カリキュラム（全5回）

### 第1回：問い合わせ即対応！AIがリードを自動スコアリングして優先順位をつける仕組み
**フォーム受信→リード分析→スコアリング→担当者通知**

- **学習目標**: Webフォームからのリード情報をGeminiで自動分析し、優先度をスコアリングして担当者に通知する
- **内容**:
  - Formsスターターでの問い合わせ受付設定
  - Geminiによるリード情報の自動分析（企業規模・予算・緊急度）
  - スコアリングロジックの構築
  - 高スコアリードの即時アラート
  - リード情報のスプレッドシート自動記録
  - 担当者への自動アサインと初回アプローチ依頼
- **作成するエージェント**: リードスコアリングエージェント

### 第2回：商談を取りこぼさない！ステージ別に自動フォローが飛ぶ進捗管理システム
**商談ステージ監視→フォロー判定→リマインド送信→エスカレーション**

- **学習目標**: 商談ステージに応じた自動フォローアップと、停滞商談のエスカレーションを実装する
- **内容**:
  - 商談管理シートからのステータス監視
  - ステージ別フォロー期限の自動計算
  - 期限前リマインドの自動送信（3日前・当日）
  - 停滞商談（7日以上未更新）の自動検知
  - マネージャーへのエスカレーション通知
  - 次アクション提案の自動生成
- **作成するエージェント**: 商談フォローアップエージェント

### 第3回：見積作成を10分から1分に！テンプレート×自動計算で即レスポンス
**見積依頼→商品選択→金額計算→PDF生成→送付**

- **学習目標**: 見積依頼から見積書作成・送付までを自動化し、顧客への即レスポンスを実現する
- **内容**:
  - フォームでの見積依頼受付
  - 商品マスタからの価格自動取得
  - 数量・割引を含めた金額自動計算
  - 見積書テンプレートへの自動反映
  - PDF形式での見積書生成
  - 顧客へのメール自動送付と担当者への控え送信
- **作成するエージェント**: 見積自動作成エージェント

### 第4回：受注率を見える化！商談データを自動集計する営業ダッシュボード
**データ収集→KPI計算→ダッシュボード更新→週次レポート配信**

- **学習目標**: 商談データを自動集計し、営業KPIをリアルタイムで可視化するダッシュボードを構築する
- **内容**:
  - 商談データの自動収集と集計
  - 受注率・商談数・売上予測の自動計算
  - 担当者別・商品別・業種別の分析
  - Geminiによる傾向コメント自動生成
  - 週次営業レポートの自動作成
  - 経営層・営業チームへの自動配信
- **作成するエージェント**: 営業ダッシュボード更新エージェント

### 第5回：失注から学ぶ！AIが原因を分析して次の提案に活かすフィードバックループ
**失注検知→原因分析→パターン抽出→改善提案→ナレッジ蓄積**

- **学習目標**: 失注案件をGeminiで自動分析し、失注原因のパターン化と次回提案への改善点を抽出する
- **内容**:
  - 失注ステータスの自動検知
  - 失注理由フォームの自動送信
  - Geminiによる失注原因の分類と分析
  - 競合・価格・タイミング別のパターン抽出
  - 類似案件への改善提案の自動生成
  - 失注ナレッジベースへの自動蓄積
- **作成するエージェント**: 失注分析フィードバックエージェント

## 技術スタック
- Google Workspace Studio
- Gemini（リード分析・失注分析・コメント生成）
- Gmail
- Googleスプレッドシート
- Googleフォーム
- Google Drive（見積書保存）
- Google Chat / Slack連携

## 主要なステップ・機能

### スターター
| スターター | 用途 |
|-----------|------|
| Formsスターター | リード受付・見積依頼・失注理由収集 |
| スケジュールスターター | 定期的な商談チェック・レポート生成 |
| Gmailスターター | メールでの問い合わせ受付 |

### ステップ
| ステップ | 用途 |
|---------|------|
| Geminiステップ | リード分析、失注分析、コメント生成 |
| スプレッドシート連携 | 商談・見積・売上データ管理 |
| 条件分岐 | スコア判定、ステージ分岐 |
| Gmail送信ステップ | 見積送付・フォローアップ送信 |
| Chat DMステップ | 担当者への通知・アラート |

## 自動化される業務フロー

```
リード管理フロー：
問い合わせ受付 → AI分析 → スコアリング → 担当者アサイン →
初回アプローチ → フォローアップ → 商談化

商談管理フロー：
商談登録 → ステージ監視 → フォロー判定 → リマインド →
停滞検知 → エスカレーション → 次アクション提案

見積・受注フロー：
見積依頼 → 自動作成 → 承認 → 送付 → 受注/失注 →
分析 → ナレッジ蓄積
```

## 達成できること
- リード対応時間：問い合わせから初回連絡まで24時間→1時間以内
- 商談フォロー漏れ：月5件→0件
- 見積作成時間：10分→1分
- 営業レポート作成：週2時間→0分（完全自動化）

## 注意事項
- 外部APIキーは不要（Google Workspace内で完結）
- 対応エディション: Business Standard以上推奨
- 商談金額など機密情報を扱うため、適切なアクセス権限設定を推奨

## 必要条件
- Google Workspace アカウント（Workspace Studio へのアクセス権）
- 商談管理スプレッドシート
- 商品マスタスプレッドシート
- 見積テンプレート（Googleドキュメント）
