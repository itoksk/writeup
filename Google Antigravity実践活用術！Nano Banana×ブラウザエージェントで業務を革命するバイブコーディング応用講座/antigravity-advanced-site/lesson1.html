無料で画像生成し放題！Nano Banana Proで開発素材を量産する方法

内蔵画像生成AIでビジュアル制作を効率化

このレッスンのゴール

Google Antigravityに内蔵されている画像生成AI「Nano Banana Pro」を使って、開発に必要なアイコン、イラスト、UIモックアップを無料で生成できるようになります。API不要・追加料金なしで、高品質な画像素材を量産する方法を習得しましょう。

Nano Banana Proとは？

Gemini 3 Pro Imageベースの画像生成AI

Nano Banana Proは、Googleの最新画像生成モデル「Gemini 3 Pro Image」をベースにした内蔵AIです。2025年3月のGoogleブログで発表された「日本語テキスト描画精度95%」を実現する驚異の画像生成技術を、Antigravity内で無料で利用できます。

従来の画像制作
  • Photoshop/Illustrator等のスキルが必要
  • 素材サイトで有料ダウンロード
  • 画像生成AIは別サービスで課金
  • 日本語テキストは文字化けしがち
Nano Banana Pro
  • チャットで「〇〇の画像を作って」と指示するだけ
  • Antigravity内で完結、追加料金なし
  • プレビュー期間中は無料で使い放題
  • 日本語テキストも95%の精度で正確描画

Nano Banana Proの起動方法

1

Playgroundを開く

Antigravityの画面右側にあるPlayground（チャット画面）を表示します。まだ開いていない場合は、メニューから「View > Playground」を選択。

2

モデルを確認

Playground上部のモデル選択が「Gemini 3 Pro」または「Gemini 3 Pro Image」になっていることを確認。これがNano Banana Proのベースです。

3

画像生成を依頼

チャットに「〇〇のアイコンを作成して」「〇〇のイラストを生成して」と入力するだけ。AIが自動で画像を生成します。

// 画像生成の基本的な指示例 「青いグラデーションの丸いアイコンを作成して」 「シンプルなフラットデザインのユーザーアイコンを作って」 「このアプリのロゴを作成して。モダンでミニマルなデザインで」

開発で使える4つの画像生成パターン

パターン1

アイコン生成

アプリ用のカスタムアイコンを作成。フラットデザイン、グラデーション、アウトラインなど様々なスタイルに対応。

パターン2

UIモックアップ

ワイヤーフレームやデザイン案を画像として生成。「ログイン画面のモックアップを作って」のように指示。

パターン3

イラスト・ヒーロー画像

ランディングページ用のヒーロー画像や説明用イラストを生成。「SaaS紹介用のビジネスイラスト」など。

パターン4

日本語テキスト入り画像

看板、ポスター、バナーなど日本語テキストが入った画像も高精度で生成。「『新規登録はこちら』と書かれたボタン画像」など。

実践：アイコンセットを作成する

一貫したデザインのアイコンを量産

ここでは、Webアプリ用のアイコンセットを作成してみましょう。スタイルを統一することで、プロ品質のUIを実現できます。

1

スタイルを指定

まず、アイコン全体のスタイルを決めます。

// Playgroundへの指示 「これから複数のアイコンを作成します。以下のスタイルで統一してください： - フラットデザイン - 青のグラデーション（#3B82F6 → #1D4ED8） - 角丸の正方形ベース（128x128px） - 白いシンボル まず、ホームアイコンを作成してください」
2

順番に生成

スタイルが確定したら、必要なアイコンを順番に依頼します。

「同じスタイルで設定アイコンを作成して」 「同じスタイルでユーザープロフィールアイコンを作成して」 「同じスタイルで通知ベルのアイコンを作成して」 「同じスタイルで検索虫眼鏡のアイコンを作成して」
3

画像を保存

生成された画像はPlayground内に表示されます。右クリック→「名前を付けて画像を保存」、またはプロジェクトフォルダに自動保存するよう依頼できます。

「生成した全てのアイコンを /images/icons フォルダに保存して」

ポイント

「同じスタイルで」と指示することで、AIは前回の設定を引き継ぎます。統一感のあるアイコンセットが簡単に作れます！

実践：ヒーロー画像付きランディングページ

1

ヒーロー画像を生成

まずランディングページ用のメインビジュアルを作成します。

「SaaSプロダクトのランディングページ用ヒーロー画像を作成してください。 - モダンでプロフェッショナルなビジネスイメージ - 青と白を基調としたグラデーション - 抽象的なテクノロジー感のあるデザイン - 16:9の横長アスペクト比 - テキストオーバーレイ用に中央を少し暗くして」
2

ランディングページを生成

ヒーロー画像ができたら、それを使ったHTMLを生成します。

「今作成したヒーロー画像を使って、シンプルなランディングページを作成して。 - ヒーロー画像を背景に配置 - 中央に白文字で「AI時代の業務効率化」というタイトル - その下に「今すぐ始める」ボタン - モダンなデザインで」

画像生成からコード生成まで一気通貫

Nano Banana Proで画像を作り、そのままAntigravityでHTMLに組み込む。従来なら複数のツールを行き来する作業が、全てチャット1つで完結します。

日本語テキスト画像の生成

他の画像生成AIとの大きな違い

多くの画像生成AIは日本語テキストが文字化けしますが、Nano Banana Pro（Gemini 3 Pro Image）は95%の精度で日本語を正確に描画できます。これは2025年3月のGoogleブログで発表された大きな進化です。

// 日本語テキスト入り画像の指示例 「『無料相談受付中』と書かれた赤いバナー画像を作成して」 「『田中商店』と書かれた和風の看板画像を作成して」 「『SALE 50% OFF 期間限定』と書かれたセールバナーを作成して」

注意点

100%ではないため、生成結果は必ず確認しましょう。複雑な漢字や特殊なフォントの場合は、何度か再生成が必要になることもあります。「もう一度生成して」と言えばOK。

生成画像の活用方法

方法1

直接HTMLに組み込み

「この画像をプロジェクトに保存して、HTMLに埋め込んで」と依頼すれば、画像保存からimg要素の追加まで自動で行われます。

方法2

CSSの背景画像として

「この画像をヒーローセクションの背景に設定して」と言えば、CSSのbackground-imageプロパティとして適用されます。

方法3

外部書き出し

右クリックで画像を保存し、他のツール（Figma、Canvaなど）で加工することも可能です。
