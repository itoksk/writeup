# Google Antigravity バイブコーディング応用講座

## コース概要
Google Antigravityの真の力を引き出す応用テクニックを学ぶ講座です。内蔵画像生成AI「Nano Banana Pro」でのビジュアル制作、ブラウザエージェントによる自動操作・マニュアル作成、MCP連携による外部ツール接続まで、バイブコーディングを実務で活用するための実践スキルを習得します。

## 対象者
- 基礎講座を修了した方、またはAntigravityの基本操作ができる方
- 画像生成AIを開発に組み込みたい方
- ブラウザ操作の自動化やマニュアル作成を効率化したい方
- 外部ツール（GitHub、Notion、Slack等）と連携したい方
- 本格的なアプリ開発にバイブコーディングを活用したい方

## 前提知識
- Google Antigravityの基本操作（起動・チャット・コード生成）
- バイブコーディングの基本概念
- **プログラミング知識は不要**（ただし基礎講座修了推奨）

## 達成目標
- Nano Banana Proで開発に必要な画像を自動生成できる
- ブラウザエージェントでUI検証を自動化できる
- Walkthroughで操作マニュアルを自動作成できる
- MCPでGitHub・Notion等と連携できる
- 本格的なWebアプリを一人で開発できる

## カリキュラム（全5回）

### 第1回：無料で画像生成し放題！Nano Banana Proで開発素材を量産する方法
**内蔵画像生成AIでビジュアル制作を効率化**

- **学習目標**: Nano Banana Proを使って、開発に必要な画像素材を生成できるようになる
- **内容**:
  - Nano Banana Pro（Gemini 3 Pro Image）とは
  - Antigravity内での画像生成の始め方
  - UIモックアップの生成
  - アイコン・イラストの作成
  - 日本語テキスト入り画像の生成（高精度）
  - 生成画像のプロジェクトへの組み込み
  - API不要・無料で使える仕組み
- **作成するもの**: カスタムアイコンセット、ヒーロー画像付きランディングページ

### 第2回：AIがブラウザを勝手に動かす！ブラウザエージェントで動作確認を完全自動化
**Chrome連携で開発・テストを革命する**

- **学習目標**: ブラウザエージェントを使って、UI検証とテストを自動化できる
- **内容**:
  - ブラウザエージェントとChrome拡張機能のセットアップ
  - 「ブラウザでテストして」の一言で何が起こるか
  - クリック・入力・スクロールの自動操作
  - エラー検知と自動修正のループ
  - スクリーンショットの自動取得
  - 複数ページにまたがる操作の自動化
  - Gemini 2.5 UI Checkpointの役割
- **作成するもの**: フォーム入力を自動テストするエージェント

### 第3回：スクショ撮影からマニュアル作成まで！Walkthroughで操作説明書を自動生成
**ドキュメント作成の自動化**

- **学習目標**: Walkthrough機能を使って、操作マニュアルやドキュメントを自動生成できる
- **内容**:
  - Walkthroughとは（変更サマリー＋検証ステップ）
  - タスク完了後の自動ドキュメント生成
  - スクリーンショット付き操作説明の作成
  - 画面録画の活用
  - Markdownドキュメントへのエクスポート
  - チームへの共有方法
  - Artifactsの種類と活用法
- **作成するもの**: アプリの操作マニュアル（スクショ付き）

### 第4回：GitHubもNotionも思いのまま！MCP連携で開発ワークフローを革命する
**外部ツールとの連携で生産性を最大化**

- **学習目標**: MCPを使って外部サービスと連携し、開発ワークフローを自動化する
- **内容**:
  - MCP（Model Context Protocol）とは
  - MCPサーバーのインストール方法
  - GitHub連携（Issue確認・PR作成・コードプッシュ）
  - Notion連携（ドキュメント作成・更新）
  - Slack連携（通知・メッセージ送信）
  - カスタムMCPサーバーの設定
  - mcp_config.jsonの編集方法
- **作成するもの**: GitHub連携した開発プロジェクト

### 第5回：総合実践！日本語だけで作る本格Webアプリ開発
**全ての技術を組み合わせた実践プロジェクト**

- **学習目標**: 学んだ全てのスキルを活用して、本格的なWebアプリを開発する
- **内容**:
  - プロジェクト要件の整理と計画
  - Nano Bananaでのビジュアル素材作成
  - 段階的な機能実装
  - ブラウザエージェントでの動作検証
  - Walkthroughでのドキュメント作成
  - GitHub連携でのバージョン管理
  - 完成品のデプロイ方法
- **作成するもの**: フル機能のWebアプリ（例：タスク管理、予約システム、ポートフォリオなど）

## 技術スタック
- Google Antigravity
- Gemini 3 Pro / Flash

- ブラウザエージェント（Chrome連携）
- MCP（Model Context Protocol）
- GitHub / Notion / Slack（外部連携）

## 主要な機能・概念

### Nano Banana Pro機能
| 機能 | 説明 |
|------|------|
| UIモックアップ生成 | ワイヤーフレームやデザイン案を画像で生成 |
| アイコン生成 | アプリ用のカスタムアイコンを作成 |
| 日本語テキスト描画 | 看板やポスターの日本語を95%の精度で正確に描画 |
| システム図生成 | アーキテクチャ図や説明図を作成 |

### ブラウザエージェント機能
| 機能 | 説明 |
|------|------|
| 自動操作 | クリック、入力、スクロールを自動実行 |
| エラー検知 | 動作エラーを自動で検知・報告 |
| 自動修正 | エラー時にロールバックして修正 |
| スクリーンショット | 操作過程を自動キャプチャ |

### MCP連携サービス
| サービス | できること |
|----------|-----------|
| GitHub | Issue管理、PR作成、コードプッシュ |
| Notion | ドキュメント作成・更新 |
| Slack | 通知送信、メッセージ投稿 |
| Jira | チケット管理、ステータス更新 |
| Firebase | デプロイ、データベース操作 |

## 達成できること
- 開発用の画像素材を無料で大量生成できる
- ブラウザ操作のテストを完全自動化できる
- 操作マニュアルを自動で作成できる
- GitHub等の外部ツールと連携した開発ができる
- 本格的なWebアプリを一人で開発・デプロイできる

## 注意事項
- 基礎講座の内容を理解していることが前提
- Nano Banana Proの画像生成には無料枠の制限あり
- MCP連携には各サービスのアカウントが必要
- ブラウザエージェントはChrome拡張機能のインストールが必要

## 必要条件
- Google Antigravity（インストール済み）
- Googleアカウント（個人用）
- Chrome ブラウザ
- GitHubアカウント（第4回〜）
