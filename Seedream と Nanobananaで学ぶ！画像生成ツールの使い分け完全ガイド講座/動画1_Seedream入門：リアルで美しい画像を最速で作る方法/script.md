## 設定実演（詳細操作）（5:30 - 7:30）

**【画面】**: Seedreamの実際の操作画面

**【講師セリフ】**:
```
それでは、具体的な操作手順を一緒に見ていきましょう。

【ステップ1：ログイン】
まず、Seedreamの公式サイトにアクセスします。
すでにアカウントをお持ちの方は、ログインしてください。

【画面操作：ログイン画面 → ダッシュボード】

ログインすると、このようなダッシュボードが表示されます。

【ステップ2：新規画像生成画面を開く】
左側のメニューから「New Generation」をクリックします。

【画面操作：New Generationをクリック】

はい、画像生成画面が開きました。

【ステップ3：プロンプト入力】
ここに、生成したい画像の内容を英語で記述します。

効果的なプロンプトの構成は：
1. 主題（何を描くか）
2. 環境・背景（どこで、どんな状況か）
3. スタイル指示（photorealistic、high detail など）

例えば、さっき使ったプロンプトはこうなります：

「A serene mountain lake at sunrise, with mist rising from the water,
surrounded by pine trees, photorealistic, high detail」

ポイントは、
- 具体的な描写を含める
- 雰囲気を表す形容詞を使う（serene、majestic など）
- スタイル指示を明記する

【ステップ4：解像度とモデル選択】
次に、パラメータを設定します。

- 解像度：1024x1024 が標準です。用途に応じて調整してください。
- モデル：Seedream v3 が最新です。写実性が最も高いモデルです。
- スタイル：「Photorealistic」を選択します。

【画面操作：各パラメータを設定】

【ステップ5：生成実行】
すべての設定が完了したら、「Generate」ボタンをクリックします。

【画面操作：Generateボタンをクリック】

生成には通常10〜30秒程度かかります。
この間、プログレスバーで進行状況を確認できます。

【生成完了】

はい、完成しました。

【ステップ6：気になる部分を微調整】
もし、気になる部分があれば、「補正指示」機能を使います。

例えば、「空の色をもう少し鮮やかに」といった指示を追加できます。

【画面操作：補正指示の入力例を表示】

このように、部分的な調整も簡単にできるのがSeedreamの特徴です。
```

## 注意点・活用ポイント（7:30 - 8:30）

**【画面】**: 注意点のスライド表示

**【講師セリフ】**:
```
ここで、Seedreamを使う際の注意点と活用ポイントをお伝えします。

【注意点1：写実性を追う時の注意】
写実性を追求する際、特に気をつけるべきポイントが3つあります：

1. ノイズの発生
   - 高解像度で生成すると、時々ノイズが入ることがあります
   - その場合は、解像度を一段階下げて再生成してみてください

2. 顔・手の破綻
   - 人物の顔や手の描写は、AIが苦手とする部分です
   - 複数枚生成して、最も自然なものを選ぶのが実践的です

3. 細部の不自然さ
   - 拡大すると不自然な部分が見つかることがあります
   - 出力後にトリミングや補正を併用することで、品質が向上します

【活用ポイント1：プロンプトの工夫】
効果的なプロンプトのコツ：

- 長さ：50〜100単語程度が適切
- 語彙：具体的な形容詞を使う（例：majestic、serene、dramatic）
- 構成：主題 → 環境 → スタイル指示の順番

【活用ポイント2：バッチ生成】
1つのプロンプトで複数枚（4枚など）同時生成すると、
良い結果が得られる確率が高まります。

【活用ポイント3：参照画像の活用】
既存の画像をアップロードして、「このような雰囲気で」と指示すると、
より意図に近い結果が得られます。

これらのポイントを押さえれば、Seedreamをより効果的に使いこなせます。
```
