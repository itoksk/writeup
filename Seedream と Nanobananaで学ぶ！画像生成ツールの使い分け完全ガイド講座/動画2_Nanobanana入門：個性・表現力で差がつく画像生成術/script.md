## 設定実演（詳細操作）（5:30 - 7:30）

**【画面】**: Nanobananaの実際の操作画面

**【講師セリフ】**:
```
それでは、具体的な操作手順を一緒に進めていきましょう。

【ステップ1：環境を開く】
Nanobananaの公式サイトにアクセスし、ログインします。

【画面操作：ログイン画面 → ダッシュボード】

ダッシュボードが表示されました。
Seedreamとは少し雰囲気が違いますね。

【ステップ2：基本プロンプト作成】
まず、何を描きたいかの基本プロンプトを作ります。

Seedreamと同様に：
1. 主題
2. 環境・背景
3. 基本的な描写

例：
「A young woman in a coffee shop, reading a book, cozy atmosphere,
afternoon light through window」

ここまではSeedreamと同じです。

【ステップ3：スタイルキーワード追加】
ここからがNanobananaの特徴です。

スタイルキーワードを追加します。
主なスタイルキーワード：

- illustrated style（イラストスタイル）
- watercolor painting（水彩画風）
- anime style（アニメ風）
- pop art（ポップアート風）
- minimalist design（ミニマリストデザイン）
- vintage aesthetic（ヴィンテージ風）

今回は「illustrated style, warm colors, artistic」を追加します。

完成したプロンプト：
「A young woman in a coffee shop, reading a book, cozy atmosphere,
afternoon light through window, illustrated style, warm colors, artistic」

【画面操作：プロンプト入力】

【ステップ4：スタイルモード選択】
Nanobananaには、プリセットの「スタイルモード」があります。

画面右側のスタイル選択パネルから：
- Illustrative（イラスト的）
- Artistic（芸術的）
- Stylized（様式化）
- Expressive（表現主義的）

今回は「Illustrative」を選択します。

【画面操作：スタイルモード選択】

【ステップ5：生成実行】
「Generate」ボタンをクリックします。

【画面操作：Generateボタンをクリック】

生成には15〜40秒程度かかります。
Seedreamより少し時間がかかることもあります。

【生成完了】

はい、完成しました。

イラスト風で、温かみのある色使い、
独特の雰囲気が出ていますね。

【ステップ6：トーン・質感調整】
気になる部分があれば、調整します。

Nanobananaでは：
- Color intensity（色の強さ）
- Style strength（スタイルの強さ）
- Detail level（ディテールレベル）

を後から調整できます。

【画面操作：調整パネルを表示】

例えば、「色をもう少し鮮やかに」したい場合、
Color intensityを上げます。

このように、生成後の微調整もスムーズです。
```

## 注意点・活用ポイント（7:30 - 8:30）

**【画面】**: 注意点のスライド表示

**【講師セリフ】**:
```
ここで、Nanobananaを使う際の注意点と活用ポイントをお伝えします。

【注意点1：表現の破綻】
スタイル性を強く出しすぎると、破綻しやすくなります。

対策：
- Style strengthを適度に（70〜85%程度）
- プロンプトに「balanced」「harmonious」を追加
- 複数枚生成して、最もバランスの良いものを選ぶ

【注意点2：スタイルの固定化】
同じスタイルキーワードを使い続けると、
表現がワンパターンになりがちです。

対策：
- 定期的に違うスタイルキーワードを試す
- 複数のスタイルを組み合わせる（例：watercolor + minimalist）
- テンプレートに頼りすぎない

【注意点3：過度な補正】
生成後の調整で、やりすぎると不自然になります。

対策：
- 調整は1〜2項目に絞る
- 元の生成結果を保存しておく
- 調整前後を比較しながら進める

【活用ポイント1：スタイルの組み合わせ】
複数のスタイルキーワードを組み合わせることで、
ユニークな表現が生まれます。

例：
- watercolor + vintage（水彩画 + ヴィンテージ）
- anime + cinematic（アニメ + 映画的）
- minimalist + pop art（ミニマル + ポップアート）

【活用ポイント2：参照画像の活用】
「このような雰囲気で」と画像をアップロードすると、
より意図に近い表現ができます。

【活用ポイント3：ムードボード作成】
複数のスタイルで生成し、
プロジェクトの「ムードボード」を作るのに便利です。
```

## まとめ（8:30 - 9:30）

**【画面】**: まとめスライド表示

**【講師セリフ】**:
```
それでは、今回のポイントをまとめましょう。

【今回学んだこと】

1. **Nanobananaの強み**
   - 表現力と個性を重視
   - アーティスティックでスタイリッシュな画像生成
   - リアルさより「印象」を大切にする

2. **基本的な使い方**
   - 基本プロンプト + スタイルキーワード
   - スタイルモードの選択
   - トーン・質感の調整

3. **注意点**
   - 表現を強く出しすぎると破綻しやすい
   - スタイルの固定化に注意
   - 過度な補正を避ける

4. **活用のコツ**
   - 複数のスタイルを組み合わせる
   - 参照画像を活用
   - ムードボード作成に便利

Nanobananaは「表現の幅」と「個性を出す力」が魅力です。

適切に使えば、目を引くビジュアルを生み出せます。

特に、
- キャラクターイラスト
- ブランドイメージ
- SNS用のビジュアル
- 印象的なプレゼン資料

などを作る場合は、Nanobananaが活躍します。

【Seedream vs Nanobanana の違い】

ここで、2つのツールの違いを整理しておきましょう：

| 項目 | Seedream | Nanobanana |
|------|----------|------------|
| 重視する点 | 写実性・リアルさ | 表現力・個性 |
| 得意な用途 | 商品写真、風景、プレゼン背景 | イラスト、キャラクター、ブランディング |
| 生成スピード | 速い（10〜30秒） | やや遅い（15〜40秒） |
| 調整のしやすさ | 微調整が簡単 | スタイル調整が豊富 |

このように、両者は「補完関係」にあります。
次回は、この使い分けのルールを詳しく学びます。
```
