# Seedream と Nanobananaで学ぶ！画像生成ツールの使い分け完全ガイド講座

## 1. 想定する現場課題

各部門・職種で「**画像生成AIを使いたいけど、どのツールを使えばいいか迷っている**」「**生成画像のクオリティを安定させたい**」「**表現の幅を拡げたい**」というモヤモヤがある状況です。

具体的な課題：

- **ツール選定に時間がかかる**: 複数の画像生成ツール（Seedream、Nanobanana など）を導入しても、「併用ルール」「得意場面」が明文化されず、担当者によってばらばらな運用になる
- **意思決定の遅延**: ツール選定・切り替え判断に時間を取られ、1画像あたり平均3～5分もかかる
- **試行錯誤の多さ**: 適切なツールを選べないため、10回以上の試行錯誤が必要
- **属人化の問題**: 「このツールはよいけど、こういう表現が苦手」という暗黙知が属人化しており、ノウハウが共有されていない
- **導入失敗リスク**: 結果、画像生成AI活用が点在的になり、「使ってみたが使いこなせない／結局使われない」失敗が起こりやすい

## 2. 理想の状態（Before / After）

| 指標 | Before | After |
|------|--------|-------|
| **画像生成にかかる意思決定時間**<br>（どのツールを使うか迷う時間） | 1画像あたり平均 3～5分 | 平均 30秒以内 |
| **画像生成の試行錯誤回数** | 10回以上 | 3〜5回程度で満足できるクオリティに到達 |
| **属人化度／運用ばらつき** | 担当者ごとに使い方・質が差が出る | 運用ガイドラインに沿って誰でも安定した使い方 |
| **ユーザーの心理状態** | 「使ってみたいけど不安」<br>「選び方が分からない」 | 「場面に応じて使い分けできそう」<br>「自信を持って使える」 |

**感覚ベースでいうと**、「ツール選びで迷って時間を無駄にするストレス」が**半分以下に減り**、表現の幅と精度が向上し、業務で活用できる自信が持てる状態を目指します。

## 7. 各動画の概要

### 動画1：Seedream入門：リアルで美しい画像を最速で作る方法

**目的**: Seedreamの基本操作と、リアルで美麗な画像を効率的に生成する方法を学ぶ

**主な内容**:
- Seedreamの強み（写実性・スピード）の理解
- 基本的なプロンプト作成とパラメータ調整
- 実際の操作デモ（ログイン〜画像生成〜微調整）
- 注意点（ノイズ、顔・手の破綻への対処）

### 動画2：Nanobanana入門：個性・表現力で差がつく画像生成術

**目的**: Nanobananaの特徴と、アーティスティックな画像を生成する方法を学ぶ

**主な内容**:
- Nanobananaの強み（表現力・個性）の理解
- スタイルテンプレートとプロンプトの併用
- 実際の操作デモ（スタイル選択〜トーン調整）
- 注意点（表現の破綻を避けるバランス感）

### 動画3：Seedream vs Nanobanana 最適使い分けルールと実践判断基準

**目的**: 場面に応じた適切なツール選択ができるようになる

**主な内容**:
- ツール選択の判断基準（リアル重視 vs スタイル重視）
- 場面別デモ（商品写真／キャラクターイラスト／背景風景）
- 使い分けチャートの使い方
- ハイブリッド利用の応用例
