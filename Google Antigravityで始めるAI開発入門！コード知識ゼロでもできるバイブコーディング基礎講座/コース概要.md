# Google Antigravity バイブコーディング基礎講座

## コース概要
2025年11月にGoogleが発表した次世代AIエディタ「Antigravity」を使って、プログラミング知識ゼロからアプリ開発ができる「バイブコーディング」を学ぶ入門講座です。日本語で指示するだけでAIがコードを自動生成する新時代の開発手法を、5つのレッスンで完全マスターします。

## 対象者
- プログラミング経験がないが、自分用のツールやアプリを作りたい方
- AIを活用した開発に興味があるビジネスパーソン
- ノーコード/ローコードツールを使ったことがある方
- エンジニアに依頼せず自分でプロトタイプを作りたい企画担当者
- 話題のバイブコーディングを試してみたい方

## 前提知識
- パソコンの基本操作（ファイルの保存・開く等）
- Webブラウザの基本操作
- Googleアカウントを持っている
- **プログラミング知識は一切不要**

## 達成目標
- Google Antigravityのインストールと日本語化ができる
- バイブコーディングの基本概念と考え方を理解する
- 日本語の指示だけでWebページを作成できる
- AIエージェントと対話しながら開発を進められる
- 簡単な自分用ツールを作れるようになる

## 技術スタック
- Google Antigravity（エージェントファーストIDE）
- Gemini 3 Pro（AI言語モデル）
- HTML / CSS / JavaScript（AIが自動生成）
- ブラウザエージェント（動作確認用）

## 主要な機能・概念

### Antigravityの3つの画面
| 画面 | 役割 |
|------|------|
| Agent Manager | エージェントの管理・監視（Mission Control） |
| Editor | コードの確認・手動編集（VSCode風） |
| Playground | AIとの対話・タスク指示 |

### 会話モード
| モード | 用途 |
|--------|------|
| Planning | 複雑なタスク・調査・計画立案 |
| Fast | シンプルなタスク・素早い実行 |

### AIモデル
| モデル | 特徴 |
|--------|------|
| Gemini 3 Pro | 標準モデル・バランス型 |
| Gemini 3 Flash | 高速・簡単なタスク向け |
| Claude Sonnet 4.5 | 複雑な推論・長文対応 |
| GPT-OSS 120B | オープンソースモデル |

## 注意事項
- 個人Googleアカウント（@gmail.com）が必要（Workspaceアカウントは未対応）
- Mac: Apple Silicon（M1/M2/M3/M4）が必須、Intel Macは非対応
- 無料枠には利用制限あり（約5時間ごとにリセット）
- 生成されたコードは完璧ではない場合があり、調整が必要なことも

## 必要条件
- Google Antigravity（無料ダウンロード）
- Googleアカウント（個人用）
- 対応OS（macOS / Windows / Linux）
- インターネット接続
