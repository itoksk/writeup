5分で完了！Antigravityインストールから日本語化まで完全ガイド

セットアップと基本画面の理解

このレッスンのゴール

Google Antigravityをダウンロード・インストールし、日本語化して使える状態にします。基本画面の構成を理解し、最初の「Hello World」を表示させるところまで到達しましょう。

動作環境の確認

事前に確認してください

Antigravityを使うには、いくつかの条件があります。特にMacユーザーはApple Silicon（M1/M2/M3/M4）が必須です。

項目 要件
Mac Apple Silicon（M1/M2/M3/M4）必須。Intel Macは非対応
Windows Windows 10以降、64bit
Linux Ubuntu 20.04以降、Debian系
Googleアカウント 個人用（@gmail.com）のみ対応。Workspaceアカウントは未対応
インターネット 常時接続が必要（AIモデルはクラウドで動作）

Macのチップ確認方法

メニューバーの「」→「このMacについて」で「チップ」の項目を確認。「Apple M1」「Apple M2」などと表示されていればOK。「Intel」と表示されている場合は非対応です。

ダウンロードとインストール

1

公式サイトにアクセス

ブラウザで https://antigravity.google/download にアクセスします。

2

OSを選択してダウンロード

Mac / Windows / Linux から自分の環境を選択し、「Download」ボタンをクリック。ダウンロードが開始されます。

3

インストール（Mac）

ダウンロードした .dmg ファイルを開き、Antigravityアイコンを「Applications」フォルダにドラッグ＆ドロップ。

4

インストール（Windows）

ダウンロードした .exe ファイルをダブルクリックし、インストーラーの指示に従って進めます。

5

初回起動

アプリケーションからAntigravityを起動。初回はセキュリティの確認が出る場合があります。「開く」を選択してください。

Googleアカウントでサインイン

1

サインイン画面が表示される

初回起動時、Googleアカウントでのサインインを求められます。

2

ブラウザでログイン

ブラウザが開き、Googleアカウントの選択画面が表示されます。個人用のGmailアカウントを選択。

3

認証を完了

必要な権限を許可し、認証を完了します。成功すると自動的にAntigravityに戻ります。

Workspaceアカウントは未対応

会社や学校で使用しているGoogle Workspace（@会社名.com など）のアカウントでは現在ログインできません。個人用の@gmail.comアカウントを使用してください。

日本語化の設定

デフォルトは英語表示

Antigravityは初期状態では英語表示です。日本語パックをインストールして日本語化しましょう。

1

コマンドパレットを開く

Mac: Cmd + Shift + P
Windows: Ctrl + Shift + P

2

言語設定を検索

「Configure Display Language」と入力し、表示されたコマンドを選択します。

3

日本語を選択

言語一覧から「日本語 (ja)」を選択します。

4

再起動

「Restart」をクリックしてAntigravityを再起動。メニューが日本語になっていれば成功！

別の方法：拡張機能から

左サイドバーの「拡張機能」アイコンをクリック → 「Japanese Language Pack for Visual Studio Code」を検索 → インストール → 再起動でも日本語化できます。

基本画面の構成

画面1

Agent Manager

起動時に表示されるダッシュボード。複数のエージェントを管理・監視。タスクの進捗状況が一覧で確認できます。

画面2

Editor

VSCodeと同じエディタ画面。ファイルツリー、コードエディタ、ターミナルなど。AIが生成したコードを確認・編集。

画面3

Playground

AIとチャット形式で対話する画面。ここに日本語で指示を入力すると、AIがタスクを実行します。

Tips

画面の切り替え方

左サイドバーのアイコンで画面を切り替えられます。一番上がAgent Manager、その下がExplorer（ファイル）、さらに下がPlayground用のチャットアイコンです。

無料枠と利用制限

プレビュー期間は無料

現在、Antigravityはパブリックプレビューとして無料で提供されています。ただし、利用制限があります。

項目 内容
利用制限 一定量の利用後、約5時間のクールタイムが発生
モデル別制限 Gemini 3 Pro、Claude、GPT-OSSはそれぞれ個別に制限
制限の回避 複数のモデルを切り替えて使うことで継続利用可能
有料プラン Google One経由で月額約3,000円のProプランあり（制限緩和）

制限に達したら

「Rate limit exceeded」と表示されたら、別のAIモデルに切り替えるか、約5時間待つとリセットされます。
