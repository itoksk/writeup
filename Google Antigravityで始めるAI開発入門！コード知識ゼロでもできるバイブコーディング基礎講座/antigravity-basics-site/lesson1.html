AIが勝手にコードを書く時代到来！Google Antigravityとバイブコーディングの衝撃

エージェントファーストIDEの革命を理解する

このレッスンのゴール

2025年11月にGoogleが発表した「Google Antigravity」とは何か、そして「バイブコーディング」という新しい開発手法がなぜ革命的なのかを理解します。プログラミング経験がなくてもアプリ開発ができる時代が来たことを実感しましょう。

バイブコーディングとは何か？

2025年2月に提唱された新概念

OpenAI共同創設者のアンドリュー・カーシュ氏が提唱した「バイブコーディング」。英語の「vibe（雰囲気）」に由来し、コードを書くのではなく「雰囲気で伝えてAIに作らせる」という革命的な手法です。

従来のプログラミング
  • プログラミング言語を習得する（数ヶ月〜数年）
  • 文法やルールを正確に書く
  • エラーを自分で特定・修正する
  • 人間がコードを書き、AIが補助
バイブコーディング
  • 日本語で「作りたいもの」を伝える
  • AIがコードを自動生成
  • エラーもAIが自動で修正
  • AIが主体でコードを書き、人間が監督
// 従来のプログラミング：HTMLを書く必要があった <!DOCTYPE html> <html> <head> <title>自己紹介</title> <style> body { font-family: sans-serif; } .profile { padding: 20px; } </style> </head> <body> <div class="profile"> <h1>田中太郎</h1> <p>Webデザイナーです</p> </div> </body> </html>

バイブコーディングでの指示

「シンプルな自己紹介ページを作って。名前は田中太郎、職業はWebデザイナー。」

→ これだけでAIが上記のコードを全て自動生成！

Google Antigravityの正体

2025年11月18日発表

GoogleがGemini 3と同時に発表したAIエディタ。VSCodeをベースに、AIエージェントを「主役」にした革新的な開発環境です。

特徴1

エージェントファーストIDE

従来のIDEは「人がコードを書く場所」でしたが、Antigravityは「AIエージェントがタスクを実行し、人が監督する」構造。

特徴2

VSCodeベース

世界で最も使われているエディタVSCodeのフォーク。拡張機能も使え、操作感も同じ。

特徴3

Gemini 3搭載

Googleの最新AIモデル「Gemini 3 Pro」を標準搭載。複雑な推論やコード生成が可能。

特徴4

複数AI対応

Gemini以外にも、Claude Sonnet 4.5やGPT-OSS 120Bなど複数のAIモデルを選択可能。

特徴5

ブラウザ連携

Chrome拡張機能と連携し、作ったアプリの動作確認やテストを自動化。

特徴6

無料で利用可能

プレビュー期間中は完全無料。プレミアモデルも無料枠で使用可能。

Antigravityの3つの画面

Agent Manager

ミッションコントロール

起動時に最初に表示される画面。複数のAIエージェントを管理・監視するダッシュボード。タスクの進捗状況が一目でわかります。

Editor

コードエディタ

VSCodeと同じ操作感のエディタ画面。AIが生成したコードを確認したり、必要に応じて手動編集することも可能。

Playground

AIとの対話画面

チャット形式でAIに指示を出す画面。「〇〇を作って」と日本語で伝えるだけで、AIがタスクを実行します。

なぜ「Antigravity」という名前？

重力に逆らう＝従来の常識を覆す

「Antigravity（反重力）」という名前には、「開発の常識を覆す」「コードを書くハードルを無重力にする」という意味が込められています。

Google Gravityとのジョーク

Googleには「Google Gravity」という、画面の要素が重力で落ちるイースターエッグがありました。Antigravityはその逆、つまり「開発の重荷から解放される」というユーモアも含まれています。

Antigravityの強み

Googleのエコシステム（Gmail、Drive、Docs等）との連携、Nano Banana Proによる画像生成、ブラウザエージェントによる自動テストなど、他にはない機能が多数。

非エンジニアでも使える理由

1

日本語で指示できる

「ToDoリストを作って」「ボタンの色を青にして」など、普通の日本語で伝えるだけ。プログラミング言語の知識は一切不要。

2

AIが全て自動化

コード生成、ファイル作成、エラー修正まで全てAIが担当。あなたは「何を作りたいか」を伝えるだけ。

3

動作確認も自動

ブラウザエージェントが作ったアプリを自動で開いて動作確認。問題があればAIが自動で修正。

4

エラーを恐れなくていい

「うまくいかない」と伝えるだけで、AIが原因を特定して修正。試行錯誤のストレスがゼロ。

バイブコーディングで作れるもの

自己紹介ページ、ポートフォリオ、ランディングページなど

例2

業務ツール

例3

計算ツール

見積計算、日付計算、単位変換など

例4

データ管理

名刺管理、読書記録、アイデアメモなど

例5

Chrome拡張機能

ブックマーク管理、ページ翻訳補助など

例6

プロトタイプ

新規サービスのモックアップ、UI案など

実際の声

「2〜3日かかっていたプロトタイプ開発が、わずか2時間で完成した」という報告も。フロントエンド、バックエンド、テストコードまで一貫して生成されます。
