# テスト用メール文章集
## 営業パイプライン自動化フロー 動作確認用

このファイルには、各エージェントの動作をテストするためのサンプルメール文章を収録しています。
自分宛てにこれらのメールを送信して、エージェントが正しく動作するか確認してください。

---

## 使い方

1. 以下のメール文章をコピー
2. 自分のGmailから自分宛て（または同僚）に送信
3. エージェントが起動するか確認
4. 出力結果が期待通りか検証

> **ヒント**: テスト時はフィルターの `from:` 条件を一時的に外すか、自分のメールアドレスを追加してください。
>
> **注意**: `from:`演算子ではドメイン指定（@xxx.co.jp）は使用できません。メールアドレスを個別に指定してください。

## Lesson 2: 見込み顧客メール要約エージェント

### テストメール①：緊急度HIGH（見積もり依頼）

**件名**: 【至急】見積書の再調整のお願い

**本文**:
```
お世話になっております。
株式会社ABCの山田太郎です。

先日ご提示いただいた見積書について、社内で検討いたしました。

具体的には、以下の点についてご検討いただければ幸いです：
- 初期費用の分割払い対応
- 年間保守費用の見直し

来週の役員会議で最終決定したいと考えておりますので、
できれば今週中にご回答いただけると助かります。

何卒よろしくお願いいたします。

株式会社ABC
営業部 山田太郎
TEL: 03-1234-5678
```

**期待される出力**:
- 緊急度: HIGH
- 案件ステージ: 見積もり
- 推奨アクション: 本日中に見積もり再調整の連絡

社内での検討がスムーズに進みます。

株式会社GHI

## Lesson 4: 添付ファイル自動保存エージェント

### テストメール⑥：提案書添付

**件名**: 【ご提案】業務効率化システムのご提案書

**本文**:
```
お世話になっております。
株式会社JKLの渡辺です。

先日ご要望いただいた業務効率化システムについて、
提案書を作成いたしましたのでお送りいたします。

添付ファイル：
- JKL社向け_業務効率化システム提案書_v1.pdf

ご不明な点がございましたら、お気軽にお問い合わせください。
来週、オンラインでのご説明の機会をいただければ幸いです。

株式会社JKL
ソリューション営業部 渡辺美香
```

**添付ファイル**: 適当なPDFファイルを添付（テスト用に空のPDFでも可）

**期待される動作**:
- 添付ファイルが「営業資料」フォルダに自動保存される
- Chat DMで保存完了通知が届く

## トラブルシューティング

### エージェントが起動しない場合
1. フィルター条件を確認（`from:`の設定）
2. テスト用に自分のメールアドレスを追加（ドメイン指定は不可）
3. エージェントが「Enabled」になっているか確認

### 期待通りの出力にならない場合
1. プロンプトの変数が正しく設定されているか確認
2. Geminiの出力をテスト実行で直接確認
3. 条件分岐の文字列一致（大文字/小文字）を確認

### 通知が届かない場合
1. Chat DMステップの宛先設定を確認
2. Google Chatアプリの通知設定を確認
3. エージェントの実行ログを確認
