# 第4回：営業資料を自動で蓄積＆可視化！
## 添付ファイル自動保存 × 週次レポートで営業活動を見える化

---

## この回の目標

メールに添付された提案書・見積書・契約書などを自動でDriveに保存し、週次でGeminiが分析・レポート化します。**2つのシンプルなエージェント**で営業活動の可視化を実現します。

---

## Before / After

### BEFORE：受講前
- 添付ファイルがメールに埋もれる
- 「あの見積書どこ？」と探す時間
- 今週何をやったか振り返れない
- 営業活動が見えない

### AFTER：受講後
- 添付ファイルが自動でDriveに蓄積
- Driveで即座に検索可能
- 週次レポートで活動を振り返り
- どの顧客と活発にやり取りしているか一目瞭然

---

## 作成するエージェント（2つ）

| エージェント | スターター | 役割 |
|-------------|-----------|------|
| 添付ファイル自動保存 | Gmailスターター（has:attachment） | ファイルをDriveに蓄積 |
| 週次営業活動レポート | スケジュールスターター（毎週月曜9時） | Geminiで分析・レポート化 |

---

## 週次レポートから分かること

- **活発な商談**：今週どの顧客と多くやり取りしたか
- **営業活動量**：提案書・見積書の送受信数
- **案件の進捗**：提案→見積もり→契約の流れが見える
- **注目案件**：動きが活発な顧客をGeminiが抽出

---

## エージェント1：添付ファイル自動保存

添付ファイル付きのメールを受信したら、自動でGoogle Driveの指定フォルダに保存します。

### フロー

```
Gmailスターター（has:attachment）
    ↓
Driveステップ（指定フォルダに保存）
    ↓
Chat DMステップ（保存完了を通知）
```

### 作成手順

| Step | 内容 | 詳細 |
|------|------|------|
| 01 | Gmailスターター設定 | Gmail - Email received / Filter: `has:attachment` |
| 02 | Driveステップ（保存） | Drive - Upload file / 「営業資料」フォルダに保存 |
| 03 | Chat DMステップ（通知） | 保存完了をChat DMで通知 |

> **事前準備**
> Google Driveに「営業資料」フォルダを事前に作成してください。エージェントはこのフォルダに添付ファイルを保存します。

### 通知テンプレート

```
📁 添付ファイルを保存しました

【ファイル名】{attachment_name}
【送信者】{sender_name} ({sender_email})
【件名】{email_subject}
【保存先】営業資料フォルダ

📂 Driveで確認: {drive_link}
```

### フィルターの追加（オプション）

全ての添付ファイルを保存すると量が多くなる場合は、重要顧客からのメールに絞ることもできます。

```
has:attachment (from:yamada@a-corp.co.jp OR from:tanaka@b-inc.com)
```

## エージェント2：週次営業活動レポート

毎週月曜日に、蓄積された営業資料をGeminiが分析し、営業活動のサマリーレポートを自動生成します。

### フロー

```
スケジュールスターター（毎週月曜9時）
    ↓
Driveステップ（ファイル一覧取得）
    ↓
Geminiステップ（営業活動を分析）
    ↓
Chat DMステップ（レポート送信）
```

### 作成手順

| Step | 内容 | 詳細 |
|------|------|------|
| 01 | スケジュールスターター設定 | Scheduled / 毎週月曜 9:00 AM / Asia/Tokyo |
| 02 | Driveステップ（一覧取得） | Drive - List files / 「営業資料」フォルダのファイル一覧を取得 |
| 03 | Geminiステップ（分析） | ファイル名から顧客・資料タイプを分析 |
| 04 | Chat DMステップ（送信） | 週次レポートをChat DMで送信 |

### Geminiプロンプト

```
以下は今週保存された営業資料のファイル一覧です。
この情報を分析して、営業活動の週次レポートを作成してください。

ファイル一覧:
{drive_files}

以下の形式で出力してください：

📊 今週の営業活動レポート

【今週の概要】
- 保存されたファイル数と主な種類（提案書、見積書など）を要約

【活発な商談】
- ファイル名から推測される顧客名をリストアップ
- 複数のファイルがある顧客は特に注目

【資料の種類】
- 提案書: X件（推定）
- 見積書: X件（推定）
- 契約書: X件（推定）
- その他: X件

【注目ポイント】
- 見積書や契約書がある場合は、商談が進んでいる可能性を指摘
- 今週フォローすべき顧客を提案

ファイルがない場合は「今週は新規ファイルはありません」と出力してください。
```

### 週次レポートの出力例

```
📊 今週の営業活動レポート

【今週の概要】
今週は5件のファイルが保存されました。
提案書2件、見積書2件、契約書1件と、複数の商談が活発に動いています。

【活発な商談】
- 株式会社ABC：提案書、見積書の2件 → 商談が進行中
- XYZ株式会社：見積書1件 → 検討段階
- DEF株式会社：契約書1件 → 契約間近の可能性

【資料の種類】
- 提案書: 2件
- 見積書: 2件
- 契約書: 1件
- その他: 0件

【注目ポイント】
⭐ DEF株式会社は契約書が届いています。契約締結に向けた最終確認を優先してください。
⭐ 株式会社ABCは見積もり段階です。今週中にフォローの連絡を入れることをお勧めします。
```

## 運用のコツ

### フォルダの定期整理
ファイルが溜まってきたら、月1回程度で古いファイルを別フォルダに移動するか、削除しましょう。Geminiの分析精度も向上します。

### ファイル名での検索
Driveの検索機能を使えば、顧客名や「見積」などのキーワードで簡単に資料を見つけられます。週次レポートと組み合わせると効果的です。

### レポートの活用
週次レポートは週明けのタスク整理に最適です。「注目ポイント」を参考に、今週のフォロー優先順位を決めましょう。
