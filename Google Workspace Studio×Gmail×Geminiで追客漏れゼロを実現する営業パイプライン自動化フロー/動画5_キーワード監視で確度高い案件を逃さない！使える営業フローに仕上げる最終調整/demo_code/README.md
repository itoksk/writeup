# 第5回：確度高案件監視エージェント - デモファイル

## 概要

営業キーワードを監視して確度高い案件を優先アラート。複数エージェントを統合して運用可能な営業フローを完成させます。

## エージェント構成

```
[Gmail: キーワード含むメール受信]
        ↓
[Gemini: 確度判定]
        ↓
[Conditional: 確度HIGH?]
    ├─ YES
    │   ├─ [Chat: 優先アラート]
    │   └─ [Gmail: ラベル「確度HIGH」]
    └─ NO
        └─ [Conditional: 確度MEDIUM?]
            ├─ YES → [Gmail: ラベル「検討中」]
            └─ NO → 終了
```

## デモで作成するエージェント

### エージェント名
```
確度高案件監視エージェント
```

### Starter設定
```
Type: Gmail - Email received
Filter: from:(@重要顧客) (契約 OR 発注 OR 見積 OR 提案 OR 検討)
```

### Step 1: Gemini確度判定
```
Action: Generate content
Prompt: [prompts.txt参照]
Output variable: probability_analysis
```

### Step 2: 条件分岐（HIGH）
```
Type: Conditional
Condition: {probability_analysis} contains "HIGH"
```

### Step 2a（True）: 優先アラート
```
Action: Google Chat - Send message
Message: [prompts.txt参照]
```

### Step 2b（True）: ラベル付与
```
Action: Gmail - Add label
Label: 確度HIGH
```

## 全エージェント連携図

```
[メール受信]
    │
    ├─→ [見込み顧客メール要約] ────→ 毎朝9時の要約通知
    │     (Scheduled)
    │
    ├─→ [追客漏れゼロ] ──────────→ ラベル「要対応」+ 次アクション
    │     (Email received)
    │
    ├─→ [営業資料自動整理] ───────→ Drive保存 + 通知
    │     (has:attachment)
    │
    └─→ [確度高案件監視] ─────────→ 優先アラート
          (キーワード含む)

[週次]
    └─→ [営業資料週次レポート] ───→ 週次レポート送信
          (毎週月曜9時)
```

## 監視キーワードリスト

## テスト用サンプルメール

### サンプル1：確度HIGH
```
From: tanaka@a-corp.co.jp
Subject: 契約書の件

山田様

お世話になっております。
先日ご提案いただいた件について、社内稟議が通りましたので、
正式に契約を進めさせていただきたく存じます。

契約書のご準備をお願いいたします。
```

期待される出力:
```
【検出キーワード】契約、稟議通りました、正式に、お願いいたします
【買いシグナル】ポジティブ
【確度】HIGH
【確度スコア】9
【根拠】稟議が通り正式契約の依頼。契約書準備の明確な依頼がある。
【推奨アクション】即座に契約書を準備し送付する
【対応優先度】1
```

### サンプル2：確度MEDIUM
```
From: suzuki@b-inc.com
Subject: 見積もりのお願い

山田様

現在検討を進めておりまして、
正式な見積書をいただけますでしょうか。

予算申請に必要となりますので、
来週中にいただければ幸いです。
```

期待される出力:
```
【検出キーワード】見積もり、検討、予算
【買いシグナル】ニュートラル
【確度】MEDIUM
【確度スコア】6
【根拠】検討段階で見積もり依頼。予算申請用のため前向き。
【推奨アクション】来週中に見積書を作成・送付
【対応優先度】2
```

### サンプル3：確度LOW
```
From: sato@c-group.jp
Subject: 情報収集のお願い

山田様

将来的に導入を検討するため、
参考までに資料をいただけますでしょうか。

まだ具体的な時期は決まっておりません。
```

期待される出力:
```
【検出キーワード】検討、将来的に
【買いシグナル】ニュートラル
【確度】LOW
【確度スコア】2
【根拠】将来的な検討で具体的な時期は未定。情報収集段階。
【推奨アクション】資料を送付し、定期的にフォロー
【対応優先度】5
```

## 導入効果まとめ

### Before / After
```
| 項目 | Before | After | 効果 |
|------|--------|-------|------|
| メール確認時間 | 1時間/日 | 12分/日 | 80%削減 |
| 追客漏れ | 月1-2件 | 0件 | 100%解消 |
| 資料探索時間 | 30分/回 | 即座 | 95%削減 |
| 重要案件見落とし | 時々発生 | 0件 | 100%解消 |
```

## 作成したエージェント一覧

```
1. 見込み顧客メール要約エージェント
   - 毎朝9時に未読メールを要約
   - 重要顧客を優先表示
   - Chatに通知

2. 追客漏れゼロエージェント
   - メール受信時に要対応判定
   - 「要対応」ラベルを自動付与
   - 次アクションを提案

3. 営業資料自動整理エージェント
   - 添付ファイルをDriveに自動保存
   - 案件フォルダに分類
   - 保存完了を通知

4. 営業資料週次レポート
   - 毎週月曜に資料一覧をレポート
   - 統計サマリーを作成
   - Chatに送信

5. 確度高案件監視エージェント
   - 営業キーワードを監視
   - 確度を判定
   - 優先アラートを送信
```
