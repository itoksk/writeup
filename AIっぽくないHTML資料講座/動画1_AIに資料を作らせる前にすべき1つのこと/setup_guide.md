# セットアップガイド

## 事前準備
- ChatGPTまたはClaudeのアカウントを用意し、Web版にログインしておく。
- 研修テーマに関する現状資料や過去の要件書があれば手元に置く。
- Googleドキュメント、Notionなどメモできるツールを開いておく。

## ワークスペースの整え方
1. ブラウザでAIチャットとメモ用アプリを2画面表示にする。
2. この動画で紹介する「要件シート」テンプレートをコピーしておく。
3. ヒアリングの回答をそのまま記録できるよう、メモ側に見出しを用意する。

## AIへの指示準備
- 最初のプロンプトでAIに「質問しながら整理して」と依頼するための文章を準備する。
- 自分の頭の中の情報を出しやすいよう、ターゲット・ゴール・成果物のキーワードを箇条書きにしておく。

## 動画内デモを再現するためのチェックリスト
- [ ] AIチャットが日本語で応対できる状態か
- [ ] 画面共有デモで示したプロンプトをコピーしたか
- [ ] 回答をまとめるテンプレートを複製したか

## トラブルシューティング
- AIが質問してくれない → プロンプトに「質問形式で進めて」と明示する。
- 回答が冗長 → 端的に答えられるよう事前に自分の考えを整理してから入力する。
- 会話履歴が長くなりすぎた → 新しいチャットを開き、要件の最終結果だけを貼り直す。
