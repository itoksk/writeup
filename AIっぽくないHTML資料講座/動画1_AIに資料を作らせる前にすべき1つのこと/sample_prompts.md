# サンプルプロンプト集

## 1. 要件ヒアリングをAIに依頼する
```
あなたは社内研修の教材設計をサポートするアシスタントです。
これから資料を作るので、以下の順番で私に質問しながら要件を整理してください。
1. ターゲット（所属部署/スキルレベル/人数）
2. 資料の利用シーン（会議/オンデマンド視聴など）
3. ゴールと測りたい成果指標
4. 希望する成果物の形式（HTML/スライド/PDFなど）
一問ずつ質問し、私の回答をまとめて最後に要件リストを提示してください。
```

## 2. ヒアリング結果を要約する
```
これまでの会話をもとに、以下のテンプレートを埋めて要件シートを作成してください。
- ターゲット：
- 利用シーン：
- 資料の目的：
- 成果物の形式：
- 必ず盛り込みたいメッセージ：
- 想定課題と解決策：
不足があれば質問してください。
```

## 3. 次の準備ステップを提案させる
```
この要件リストを踏まえて、資料を整えるために私が今日やるべき3つのタスクを提案してください。優先度順に並べてください。
```
