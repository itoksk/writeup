# セットアップガイド

## 必要なアカウントと権限
- Google Workspaceアカウント（サイト作成・社内公開が許可されていること）
- HTMLファイルが置かれているGitHub、社内ファイルサーバ、またはGoogleドライブの公開設定

## Googleサイトでの準備
1. Googleサイトにアクセスし、社内ポータル向けのテンプレートを選択。
2. サイトのテーマ設定でブランドカラーを指定し、背景がHTMLのデザインと馴染むように調整。
3. 「ページ」タブで新規ページを作成し、ページ名前を「HTML資料デモ」に変更。

## HTML埋め込み手順
1. 画面右側の「埋め込む」をクリック。
2. URL埋め込みの場合は、ホストしたHTMLのURLを貼り付け「挿入」。
3. コード埋め込みの場合は、以下のサンプルコードを貼り付ける。
```html
<iframe src="https://example.com/materials.html"
        style="border:0; width:100%; height:90vh; overflow:hidden;"
        allowfullscreen></iframe>
```
4. 埋め込みコンテンツを選択し、ツールバーの「サイズ変更 → 幅を100%にする」を設定。

## 公開設定
- 画面右上の「共有 → 公開設定」で「特定のユーザー」または「ドメイン内のユーザー」を選択。
- プレビューでPC・タブレット・スマホ表示を切り替え、崩れがないか確認。
- 必要に応じてバージョン管理のコメント欄に更新理由を残す。

## トラブルシューティング
- iframeが表示されない：`https://`で始まる安全なURLか、社内ネットワークからアクセスできるか確認。
- スクロールバーが二重：HTML側の`body`に`overflow-y: hidden;`を追加するか、iframeの高さを`100vh`→`90vh`に調整。
- ダークモードが効かない：埋め込み先のCSSが上書きしていないか確認し、`prefers-color-scheme`メディアクエリをHTML側に追加。
- Googleサイトで保存できない：フォームやスクリプトが含まれている場合は一部ブロックされるため、静的HTMLを用意する。
