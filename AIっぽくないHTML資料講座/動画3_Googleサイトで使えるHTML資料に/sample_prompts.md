# サンプルプロンプト集

## 1. 埋め込み用のiframeコード生成
```
以下のHTML資料をGoogleサイトに埋め込むための`<iframe>`コードを生成してください。
条件：
- 幅は100%、高さは90vh
- ダークモード切り替えに合わせて背景色を自動で反転するJSは不要
- スクロールバーが二重にならないよう`style="border:0; overflow:hidden;"`を指定
- 下部に「資料を新しいタブで開く」リンクを表示
```

## 2. 公開前チェックリストの生成
```
GoogleサイトでHTML資料を公開する前に確認すべき項目を7つ挙げてください。
分類：デザイン / 権限 / 動作検証 の3カテゴリに分け、各カテゴリごとにリスト化。
```

## 3. トラブル時のサポート手順を提示
```
Googleサイトに埋め込んだHTMLが真っ白になる場合の原因調査フローを作成してください。
- 原因候補を3つ
- 各原因に対する確認方法
- 対処方法
```
