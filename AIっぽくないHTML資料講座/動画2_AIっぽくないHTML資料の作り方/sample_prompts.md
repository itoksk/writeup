# サンプルプロンプト集

## 1. デザイン要件を含めたHTML生成
```
あなたは雑誌風のHTML資料をデザインするUIデザイナー兼フロントエンドエンジニアです。
以下の要件を必ず満たしながら、literary-editorial-templateのトーンでHTMLを生成してください。
- ターゲット：事業企画部の新任マネージャー向け社内研修
- テーマ：落ち着いた紙媒体の雑誌風。アクセントカラーは#00F0FF
- セクション構成：導入 / Before-After / 3ステップ解説 / チェックリスト / 実践タスク
- ナビゲーション：右側縦ドット（IntersectionObserver実装）
- コードブロックには`white-space: pre-wrap;`を指定
- Google Fonts「Zen Kaku Gothic New」「Space Grotesk」を読み込む
- レスポンシブ対応とダークモード切り替えをCSS変数で実装する
```

## 2. 生成されたHTMLのレビュー依頼
```
以下のHTMLについて、AIらしさが残っている箇所を指摘し、改善のための書き換え例を提示してください。
評価ポイント：
1. 配色の一貫性（Primary/Secondary/Accent）
2. 情報のメリハリ（見出し・本文・強調テキスト）
3. アニメーションやインタラクションの自然さ
```

## 3. デザインバリエーションの追加オーダー
```
完成したHTMLを基に、教育現場向けの柔らかいトーンへ変換する提案を3案ください。
- 各案で変えるべきフォント・カラー・UIアクセントを箇条書き
- 差し替え用のGoogle Fontsとアイコン指示も含める
```
