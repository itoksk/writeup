# ノーコードで実務がここまで変わる！Google Workspace Flowsで通知・要約・共有を"全自動化"する

## 1. 想定する現場課題

### 現状の問題点

* Google Workspace内の業務自動化を進めたいが、**GAS（Apps Script）やAPI連携の知識がハードル**になっている
* 「Gemini連携の自動化ツール（Google Workspace Flows）」を触っても、**Chatスペース連携・トリガー設定でつまづく人が多い**
* 現場では、フォーム回答の整理・共有、ファイルアップロード時の通知などを**手作業で行っており、1件あたり5〜10分の時間が奪われている**
* 自動化の恩恵を感じられず、「AI使ってみたけど、結局難しい…」という心理的離脱が発生している

### 具体的な課題シーン

* フォーム回答が来るたびに、担当者が手動でChatに通知を投稿（毎日10〜15件で合計1時間以上）
* Driveにアップロードされた資料を確認し、要点を整理してチームに共有（1件10分×週5件=週50分）
* 自動化ツールを試みたが、Webhook設定やスペース連携でエラーが頻発し、挫折

---

## 2. 理想の状態（Before / After）

| 項目 | Before（現状） | After（研修後） | 改善効果 |
|------|---------------|----------------|----------|
| **業務時間** | 毎日20〜30分の通知作業・整理作業 | 自動化により作業時間を2〜3分に短縮 | **90%削減** |
| **スキル面** | GASコードやWebhookに苦手意識 | ノーコードでAIエージェント構築可能 | コーディング不要 |
| **心理面** | 「AIは難しい」「自分には無理」 | 「できた！」という実感と達成感 | 自己効力感向上 |
| **チーム運用** | 属人化・情報伝達ミス多発 | 自動通知でチーム全員にリアルタイム共有 | 情報共有の均一化 |
| **即応性** | 通知までに数時間〜半日かかる | フォーム送信・ファイルアップロード後、即座に通知 | リアルタイム対応 |

## 4. 改善方針

### 基本方針

* **ノーコード**：GASやプログラミング知識は一切不要。クリック操作のみで完結
* **10分構成**：各動画を10分以内にまとめ、実務の合間に学習可能
* **現実的な効率化**：「完全自動化」ではなく、「20〜30%の時間削減」を目標にした現実解を提示
* **つまづきポイント対策**：最も多いエラー（Chatスペース連携）を1動画まるごと使って徹底解説

### 技術範囲

* **使用ツール**：Google Workspace Flows、Gemini for Workspace、Google Chat、Google Drive、Google Forms
* **前提条件**：Google Workspace（Business/Enterprise）アカウント、Gemini for Workspaceの有効化
* **制約事項**：Educationアカウントでは一部機能が使用不可
