# 第3回：Gemini要約×Chat連携 - プロンプト集

## Gemini要約プロンプトの基本形

```
以下のファイルを要約してください。
要点を3つの箇条書きでまとめ、カジュアルで寄り添うトーンで書いてください。
```

## プロンプトのバリエーション

### パターン1：ビジネスライクなトーン
```
以下のファイルの内容を簡潔に要約してください。
重要なポイントを3〜5つの箇条書きでまとめてください。
```

### パターン2：詳細な要約
```
以下のファイルを要約してください。
・主要なポイント（3つ）
・注意すべき点（2つ）
・次のアクション（1つ）
上記の形式で、わかりやすくまとめてください。
```

### パターン3：特定の観点での要約
```
以下のファイルを、マーケティング施策の観点から要約してください。
特に、ターゲット層、施策内容、期待効果に焦点を当ててまとめてください。
```

### パターン4：超シンプル版
```
以下のファイルを3行で要約してください。
```

## 変数の設定

### 要約対象
- **Link to the file**：ファイル全体へのリンクを渡す（推奨）
- **File content**：ファイルの内容そのものを渡す

### 出力先
- Chat通知のメッセージ内容に、Geminiの出力を含める

## 使い方のコツ

1. **トーン指定**：「カジュアル」「ビジネスライク」「専門的」など
2. **文量指定**：「3行で」「5つの箇条書きで」など
3. **観点指定**：「マーケティングの観点から」「技術的な観点から」など

## 次回以降の応用例

- 条件分岐：ファイル形式に応じて、異なる要約方法を適用
- 複数アクション：要約結果をChatとスプレッドシートの両方に記録
