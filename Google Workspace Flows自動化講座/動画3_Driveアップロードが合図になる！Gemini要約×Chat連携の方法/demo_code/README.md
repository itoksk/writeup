# デモ用資料・参考情報

## テスト用Driveフォルダの準備

### フォルダ名
「資料共有フォルダ」（サンプル）

### テストファイルの準備
以下のようなファイルを用意してください：

1. **PDFファイル**：マーケティング施策資料（2〜3ページ）
2. **Googleドキュメント**：会議議事録
3. **Googleスプレッドシート**：売上データ

## デモ実演の流れ

1. Flowsで「Driveファイル要約」テンプレートを選択
2. トリガー：「資料共有フォルダ」を選択
3. Gemini要約ステップを追加
4. プロンプトを設定
5. Chat通知を設定
6. テスト実行
7. 実際にファイルをアップロードして動作確認

## 必要なスクリーンショット

1. Driveフォルダ選択画面
2. Gemini要約ステップの設定画面
3. プロンプト入力欄
4. 変数選択（Link to the file）
5. Chat通知に表示された要約結果
