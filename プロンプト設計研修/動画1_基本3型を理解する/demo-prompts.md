# 動画1：基本3型デモプロンプト集

## 1. ゼロショットプロンプト

### 基本例
```
新商品説明会の案内メールを書いてください。
日時：3月15日14:00-16:00
場所：本社3F会議室
```

### 実務応用例
```
会議の議事録を作成してください。
参加者：営業部5名、開発部3名
議題：新サービスのローンチ計画
```

## 2. ワンショットプロンプト

### メール作成例
```
以下の例を参考に、新商品説明会の案内メールを書いてください。

[例]
件名：【ご案内】〇〇セミナー開催について
お客様各位
平素より格別のご高配を賜り、厚く御礼申し上げます。
この度、〇〇セミナーを開催する運びとなりましたので、
下記の通りご案内申し上げます。

記
日時：〇月〇日（〇）14:00-16:00
場所：〇〇
内容：〇〇について
申込：メールにてご返信ください

[作成依頼]
日時：3月15日14:00-16:00
場所：本社3F会議室
内容：新商品説明会
```

### レポート作成例
```
以下の形式で売上レポートを作成してください。

[例]
2024年1月売上報告
・総売上：500万円（前月比110%）
・主要因：キャンペーン効果により新規顧客が増加
・課題：リピート率の低下（15%→12%）
・対策：フォローアップメールの強化

[データ]
2024年2月
売上：550万円
前月比：110%
新規顧客：150名
リピート率：13%
```

## 3. フューショットプロンプト

### 顧客対応メール例
```
以下の例を参考に、状況に応じた適切なメールを作成してください。

[例1：お礼]
件名：ご来店ありがとうございました
山田様
本日はご来店いただき、誠にありがとうございました。
お選びいただいた商品がお客様のお役に立てることを願っております。

[例2：お詫び]
件名：配送遅延のお詫び
田中様
この度は商品の配送が遅れ、大変ご迷惑をおかけしております。
現在、配送状況を確認中でございます。

[例3：ご案内]
件名：新サービス開始のご案内
佐藤様
平素よりご愛顧いただき、ありがとうございます。
この度、新しいサービスを開始することとなりました。

[作成依頼]
状況：注文キャンセルの受付
顧客名：鈴木様
内容：キャンセル手続き完了の連絡
```

### 日報作成例
```
以下の形式で本日の日報を作成してください。

[例1]
日付：2024/2/1
業務内容：
・A社訪問（新規提案）
・資料作成（3件）
・チーム会議参加
成果：A社から前向きな回答
課題：資料作成の効率化が必要

[例2]
日付：2024/2/2
業務内容：
・B社とのWeb会議
・見積書作成（2件）
・市場調査
成果：B社と契約締結
課題：競合他社の動向把握

[例3]
日付：2024/2/3
業務内容：
・社内研修参加
・C社フォローアップ
・週次レポート作成
成果：研修で新スキル習得
課題：C社の反応が鈍い

[本日の内容]
・D社訪問（既存顧客）
・プレゼン資料作成
・部署ミーティング
・E社から問い合わせ対応
```

## 使い分けガイド

### ゼロショットが適している場面
- 単発のタスク
- 創造性を重視する場合
- 素早い回答が必要な場合

### ワンショットが適している場面
- 特定のフォーマットに従う必要がある場合
- 文体を統一したい場合
- 品質と効率のバランスを取りたい場合

### フューショットが適している場面
- 定型業務
- 高い再現性が必要な場合
- 複雑なパターンを学習させたい場合
