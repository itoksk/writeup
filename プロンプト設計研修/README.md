# プロンプト設計研修
## 用途にあわせた最適なプロンプトとは？Freeform vs Structured Prompt

---

## 研修概要

本研修は、AIを活用したメール作成において、プロンプト設計の基本から応用まで体系的に学ぶ実践的なプログラムです。全4回の動画（各10分）で、業務効率を40%改善する具体的な手法を習得できます。

### 対象者
- AIツール（ChatGPT/Claude等）を業務で活用したい方
- メール作成の品質と効率を向上させたい方
- チーム全体のAI活用レベルを向上させたい管理職

### 期待される成果
- メール作成時間：40%削減
- 品質のばらつき：70%改善
- チーム生産性：30%向上

---

## カリキュラム構成

### 📚 第1回：ゼロショット vs ワンショット vs フューショット
**プロンプト指示の基本3型を理解する（10分）**

- プロンプトの基本3型の特徴と違い
- 各手法の使い分け基準
- 実践的な比較デモ

**学習ポイント：**
- ゼロショット：速いが不安定
- ワンショット：バランス型
- フューショット：高品質・高再現性

---

### 💬 第2回：「感情を伝える」Freeformプロンプト
**ニュアンスで伝える文面設計術（10分）**

- 感情表現を重視した指示方法
- トーン調整の3つのコツ
- 感謝・お詫び・挨拶メールの実践

**学習ポイント：**
- 自然な言葉でAIに指示
- 具体的な形容詞の組み合わせ
- 3行以内の簡潔な指示

---

### 📋 第3回：「品質を揃える」Structuredプロンプト
**テンプレートで再現性を高める方法（9分）**

- 構造化による品質管理
- テンプレートの作成と管理
- チーム共有の実践方法

**学習ポイント：**
- 入力と出力の構造化
- 再利用可能なテンプレート設計
- 人間味を加えるトーン指定

---

### 🎯 第4回：プロが使い分ける！
**メール作成におけるプロンプト設計の最適解とは？（10分）**

- 状況別の最適手法選択
- ハイブリッド型の実践
- チーム展開とプロンプト管理

**学習ポイント：**
- 3ステップアプローチ
- 最強の組み合わせパターン
- 継続的改善の仕組み

## ファイル説明

### 📝 script.md
各動画の詳細な講師用台本です。以下が含まれます：
- 時間配分（分単位）
- 講師セリフ（そのまま読める形式）
- 画面操作指示
- デモ実演の手順

### 🖥️ materials.html
受講者が手元で確認できる教材です：
- スライド形式の解説
- 比較表・図解
- 実践例とサンプル
- チェックリスト

### 📂 demo_prompts/
実際の研修で使用するプロンプト例：
- 基本型のサンプル
- 実践的なテンプレート
- 成功パターン集

## 使用方法

### 講師の方へ
1. 各動画フォルダの`script.md`を事前に確認
2. `materials.html`をブラウザで開いて画面共有
3. `demo_prompts`内のファイルを使って実演
4. 時間配分に注意しながら進行

### 受講者の方へ
1. `materials.html`を手元で開く
2. デモを見ながら一緒に実践
3. `demo_prompts`のファイルを参考に練習
4. 自社の業務に合わせてカスタマイズ

## 実践課題

各回の終了後に以下を実践してください：

### 第1回後
- 自分の業務メール1つを3つの手法で作成
- 品質と時間を比較

### 第2回後
- 感情表現が重要なメールをFreeformで作成
- トーン調整を3パターン試す

### 第3回後
- 定型メールのテンプレート作成
- チームメンバーと共有

### 第4回後
- 業務メール10種類の分類
- 各メールに最適な手法を選定
- プロンプト管理シート作成
