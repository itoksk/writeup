## シーン別適用マトリクス（1:00〜4:00）

### 使い分けマトリクスの提示（1:00〜2:30）

#### 講師セリフ
「まず、業務シーン別の最適な手法を整理しました。」

#### マトリクス表示
| 業務シーン | 基本手法 | プロンプト型 | 理由 |
|------------|----------|--------------|------|
| クレーム対応 | フューショット | Structured | 一貫性と品質の両立 |
| 初回挨拶 | ワンショット | Freeform | 親しみやすさ重視 |
| 定例報告 | フューショット | Structured | 効率性重視 |
| イベント案内 | ワンショット | Freeform + Structured | バランス型 |
| 緊急連絡 | ゼロショット | Structured | スピード重視 |
| お礼・感謝 | ワンショット | Freeform | 感情表現重視 |

### 判断基準の解説（2:30〜4:00）

#### 講師セリフ
「クレーム対応なら、フューショット＋Structuredの組み合わせ。過去の成功例を学習させつつ、構造も統一します。」

#### 判断フローチャート
```
スタート
   ↓
時間的余裕は？
├─ なし → 緊急度で判断
│         ├─ 超緊急 → ゼロショット
│         └─ 通常急ぎ → ワンショット
└─ あり → 用途で判断
          ├─ 定型業務 → フューショット + Structured
          ├─ 感情重視 → Freeform
          └─ バランス → ハイブリッド
```

## ハイブリッド実演（4:00〜8:00）

### タスク説明（4:00〜4:30）

#### 講師セリフ
「同じ『セミナー参加お礼メール』を3つの方法で作成し、違いを確認しましょう。」

#### シナリオ
- 宛先：セミナー参加者（100名）
- 内容：参加への感謝と次回案内
- 要件：温かみと情報の正確さの両立

### 実演1：Freeformのみ（4:30〜5:30）

#### プロンプト
```
セミナーに参加いただいた方へのお礼メールを書いてください。
・心からの感謝が伝わる温かい文面
・また参加したくなるような前向きな締めくくり
・親しみやすいトーン
```

#### 結果分析
- ✓ 温かみがある
- ✗ 構成にばらつき
- ✗ 必要情報の漏れ

### 実演2：Structuredのみ（5:30〜6:30）

#### プロンプト
```
【入力情報】
イベント名：AI活用セミナー
参加者数：100名
次回予定：3月20日

【出力構成】
1. 件名
2. 挨拶
3. 参加への感謝
4. セミナーの振り返り
5. 次回案内
6. 締めの言葉
```

#### 結果分析
- ✓ 構成が整っている
- ✓ 情報漏れなし
- ✗ やや機械的

### 実演3：ハイブリッド型（6:30〜7:30）

#### プロンプト
```
セミナー参加者へのお礼メールを作成してください。

【基本情報】
イベント：AI活用セミナー
参加者：100名
次回：3月20日

【構成】
1. 件名：感謝の気持ちが伝わるもの
2. 導入：温かみのある挨拶
3. 本文：
   - 参加への心からの感謝（Freeform的表現で）
   - セミナーのハイライト（3点を箇条書き）
   - 次回案内（期待感を持たせる表現）
4. 締め：また会えることを楽しみにする言葉

【トーン】
プロフェッショナルでありながら親しみやすく、
次回も参加したくなるような前向きな雰囲気
```

#### 結果分析
- ✓ 構成が整っている
- ✓ 自然な表現
- ✓ 感情と情報のバランス

#### ステップ詳細
```
Step 1：用途を判断
・誰に送るか（社内/社外）
・何のためか（連絡/依頼/感謝）

・品質 → Freeform/Structured

Step 3：カスタマイズ
・必要に応じて要素を追加
```
