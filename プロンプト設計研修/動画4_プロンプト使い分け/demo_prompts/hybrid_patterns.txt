# ハイブリッドプロンプトパターン集

## 1. 最強の組み合わせ：Structured + Freeform + Few-shot

```
セミナー参加者へのお礼メールを作成してください。

【基本情報】← Structured要素
イベント：AI活用セミナー
参加者：100名
次回：3月20日

【構成】← Structured要素
1. 件名：感謝の気持ちが伝わるもの
2. 導入：温かみのある挨拶 ← Freeform要素
3. 本文：
   - 参加への心からの感謝（Freeform的表現で）
   - セミナーのハイライト（3点を箇条書き）
   - 次回案内（期待感を持たせる表現）
4. 締め：また会えることを楽しみにする言葉

【トーン】← Freeform要素
プロフェッショナルでありながら親しみやすく、
次回も参加したくなるような前向きな雰囲気

【参考例】← Few-shot要素
前回の成功例：「皆様の熱心な姿勢に、私たちも多くを学ばせていただきました」
```

## 2. クレーム対応：Few-shot + Structured + Freeform

```
クレーム対応メールを作成してください。

【状況】
クレーム内容：商品の初期不良
お客様の感情：非常に怒っている
対応方針：交換対応

【成功例を参考に】← Few-shot
例1：「この度は、ご期待に沿えず誠に申し訳ございません」
例2：「お客様のお気持ちを考えると、心が痛みます」

【構成】← Structured
1. 謝罪（心からの）← Freeform
2. 状況の確認と共感
3. 具体的な対応策
4. 再発防止への取り組み
5. 今後の関係への言及

【感情表現】← Freeform
・深い反省と真摯な謝罪
・お客様の立場に立った共感
・誠実で前向きな解決姿勢
```

## 3. 新規開拓：One-shot + Hybrid

```
新規顧客への提案メールを作成。

【参考例】← One-shot
件名：貴社の〇〇にお役立てできることがあります
書き出し：突然のご連絡失礼いたします...

【構造と感情の融合】← Hybrid
1. 自己紹介（簡潔に、でも印象的に）
2. 連絡の理由（相手のメリット中心に）
3. 提案内容（構造的に、でも熱意を込めて）
4. 次のステップ（押しつけがましくなく）

【バランス指定】
・ビジネスライク 60% : パーソナル 40%
・情報提供 70% : 営業色 30%
```

## 4. 重要通知：Structured + トーン調整

```
システムメンテナンスの通知

【必須情報】← Structured
日時：〇月〇日 〇時～〇時
影響：〇〇システムが使用不可
対象：全社員

【構成＋各パートのトーン】
1. 件名：【重要】を付けて目立たせる
2. 要点（最初に結論、分かりやすく）
3. 詳細情報（構造的に整理）
4. 影響と対策（不安を与えないよう配慮）← Freeform
5. お詫び（形式的でなく誠実に）← Freeform
6. 問い合わせ先（親切な対応を約束）
```

## 5. 状況別使い分けマトリクス

### 緊急度 × 重要度での選択

```
【緊急・重要】
→ Structured（速さと正確性）
例：障害報告、緊急会議招集

【緊急・通常】
→ Zero/One-shot（スピード重視）
例：日程変更連絡、簡単な確認

【通常・重要】
→ Few-shot + Structured（品質重視）
例：重要顧客への提案、契約関連

【通常・通常】
→ Freeform or One-shot（効率重視）
例：定例連絡、社内調整
```

## 6. 段階的アプローチ

### Step 1：基本判断
```
判断ポイント：
□ 時間的余裕：あり/なし
□ 繰り返し頻度：高/低
□ 品質要求：高/中/低
□ 感情要素：必要/不要
```

### Step 2：手法選択
```
基本型：Zero / One / Few
構造：Freeform / Structured / Hybrid
```

### Step 3：カスタマイズ
```
追加要素：
+ トーン指定
+ 文字数制限
+ 必須項目チェック
+ 成功例の参照
```

## 8. 実践演習課題

以下のシナリオで最適なハイブリッドプロンプトを設計：

1. **新サービスローンチの社内告知**
   - 全社員の興味を引く
   - 詳細情報を正確に伝える
   - 参加意欲を高める

2. **重要顧客への年末挨拶**
   - 1年の感謝
   - 来年への期待
   - パーソナルな要素

3. **プロジェクト遅延の報告**
   - 事実を正確に
   - 謝罪と対策
   - 信頼回復への配慮
