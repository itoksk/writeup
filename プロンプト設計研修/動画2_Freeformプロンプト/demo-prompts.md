# 動画2：Freeformプロンプトデモ集

## 感謝・お礼メールのプロンプト

### 長期取引先への感謝
```
5年間お取引いただいているお客様に感謝メールを書いてください。
・温かみがあり、心からの感謝が伝わる文面
・ビジネスマナーは守りつつ、機械的でない表現
・今後も末永いお付き合いをお願いする気持ちを込めて
```

### イベント参加へのお礼
```
セミナーに参加してくださった方へのお礼メールを書いてください。
・参加への感謝と、熱心に聞いていただいたことへの感動を表現
・堅苦しくない、親しみやすいトーン
・次回のイベントにも期待していただけるような前向きな締めくくり
```

### サポートへの感謝
```
プロジェクトを支援してくれた他部署のチームへの感謝メールを作成してください。
・チームメンバー全員への感謝が伝わる
・具体的な貢献への言及を含む
・今後の協力関係への期待を自然に表現
```

## お詫び・謝罪メールのプロンプト

### 納期遅延のお詫び
```
納期が遅れたことへのお詫びメールを書いてください。
・誠実な謝罪の気持ちが伝わる
・言い訳がましくない素直な表現
・今後の改善への決意を込めて
・相手の立場に立った共感的な文面
```

### 不具合対応の謝罪
```
システムの不具合でご迷惑をおかけした顧客への謝罪メールを作成してください。
・技術的な説明は最小限に
・お客様の困った状況への理解と共感を示す
・単なる定型文ではない、心のこもった謝罪
・信頼回復への真摯な姿勢
```

## 初回挨拶・関係構築のプロンプト

### 新規顧客への挨拶
```
初めてお会いするお客様への挨拶メールを書いてください。
・緊張感を与えない親しみやすさ
・プロとしての信頼感は保つ
・次回お会いするのが楽しみになる締めくくり
```

### 新任の挨拶
```
新しく担当になった際の挨拶メールを作成してください。
・前任者への敬意を示しつつ
・自分らしさが伝わる自己紹介
・これからの抱負を前向きに
・相手に安心感を与える温かいトーン
```

## 依頼・お願いメールのプロンプト

### 協力依頼
```
他部署に協力をお願いするメールを書いてください。
・相手の忙しさに配慮した丁寧な依頼
・押し付けがましくない柔らかい表現
・協力していただけることへの感謝を先に伝える
・win-winの関係性を意識した内容
```

### アンケート回答依頼
```
顧客満足度アンケートへの回答をお願いするメールを作成してください。
・回答の手間を申し訳なく思う気持ち
・でも、ぜひご意見を聞きたいという熱意
・堅苦しくない、話しかけるような文体
・回答することの価値を感じてもらえる説明
```

## トーン調整の実践例

### フォーマル度の調整
```
【高フォーマル版】
重要な取締役への報告メールを、最高レベルの敬語と格式で書いてください。

【中フォーマル版】
通常の取引先への連絡メールを、ビジネスマナーを守りつつ読みやすく書いてください。

【低フォーマル版】
社内の親しいチームメンバーへの連絡を、カジュアルだけど失礼にならない程度で書いてください。
```

### 感情の温度調整
```
【温かい表現】
長年の顧客への年末挨拶を、心温まる感謝の気持ちを込めて書いてください。

【中立的表現】
新規見込み客への製品紹介を、プロフェッショナルかつ親切に書いてください。

【クールな表現】
契約条件の確認メールを、事実を正確に、感情を抑えて書いてください。
```

## 改善指示の追加例

### 初回生成後の改善
```
前の文章をベースに、以下を追加してください：
・この5年間で特に印象的だったエピソード
・担当者として個人的に感じている感謝
・来年度への具体的な期待
```

### パーソナライズの追加
```
もう少し個人的なタッチを加えてください：
・相手の名前を自然に織り込む
・過去の具体的なやり取りへの言及
・相手の成功や成長への言及
```

### トーンの微調整
```
もう少し親しみやすくしてください：
・敬語は維持しつつ、硬い表現を和らげる
・読み手が笑顔になるような要素を追加
・締めの言葉をより温かく
```

## 使い分けのガイドライン

### Freeformが適している場面
✅ 感情表現が重要な場面
✅ 相手との関係性を深めたい場合
✅ 定型文では伝わらない想いがある時
✅ 個別対応が必要な状況

### Freeformを避けるべき場面
❌ 法的文書や契約関連
❌ 大量送信が必要な定型連絡
❌ 厳格なフォーマットが決まっている報告書
❌ チーム全体で統一が必要な文書

## 実践演習

### 演習1：感情の種類を変える
同じ状況で、異なる感情を込めたメールを作成：
- 状況：プロジェクト完了の連絡
- バージョン1：達成感と喜び
- バージョン2：安堵と感謝
- バージョン3：次への期待と意欲

### 演習2：相手に応じた調整
同じ内容を異なる相手に送る場合の調整：
- 内容：新サービスの案内
- 相手1：長年の信頼関係がある顧客
- 相手2：初めて接触する見込み客
- 相手3：社内の協力部署

### 演習3：段階的な改善
初回生成から理想の文章への改善プロセス：
1. 基本的な要件でまず生成
2. 感情的な要素を追加
3. 具体的なエピソードを追加
4. 最終的なトーン調整
