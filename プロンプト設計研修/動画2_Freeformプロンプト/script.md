### 講師セリフ
「『申し訳ございません』と『心よりお詫び申し上げます』、どちらが相手に響くでしょうか？ビジネスメールでは、言葉のニュアンスが印象を大きく左右します。今回は自然な言葉でAIに感情を伝えるFreeformプロンプトを学びます。」

- 2つの謝罪表現を並列表示

## Freeformの特徴（1:00〜3:00）

### 基本概念の説明（1:00〜2:00）

#### 講師セリフ
「Freeformプロンプトは、日常会話のような自然な文章でAIに指示する方法です。『温かみのある』『親しみやすい』といった感覚的な表現を使えるのが特徴です。」

#### 画面表示
```
Freeform例：
「お客様への感謝の気持ちを、丁寧だけど堅苦しくない、
親しみやすい言葉で表現したメールを書いてください」
```

### 従来型との違い（2:00〜3:00）

#### 講師セリフ
「従来の指示が『件名、本文、署名を含むメール』という構造指定なら、Freeformは『心からの感謝が伝わるメール』という感情指定です。」

#### 比較表示
| 従来型 | Freeform |
|--------|----------|
| 構造重視 | 感情重視 |
| 形式的 | 自然体 |
| 再現性高 | 柔軟性高 |

## 実演：感謝メール作成（3:00〜7:00）

### シナリオ設定（3:00〜3:30）

#### 講師セリフ
「実際に、長期契約をいただいたお客様への感謝メールを作ってみます。」

#### 状況説明
- 顧客：ABC商事様
- 関係：5年間の継続取引
- 目的：年度末の感謝表明
- 要望：形式的でない心のこもった内容

### 初回プロンプト入力（3:30〜4:30）

#### プロンプト
```
5年間お取引いただいているお客様に感謝メールを書いてください。
・温かみがあり、心からの感謝が伝わる文面
・ビジネスマナーは守りつつ、機械的でない表現
・今後も末永いお付き合いをお願いする気持ちを込めて
```

#### 画面操作
1. プロンプト入力
2. 生成実行
3. 初回出力の確認

### 出力分析と改善（4:30〜5:30）

#### 講師セリフ
「1回目の出力を見てみましょう。少し硬いですね。では『もう少しパーソナルな要素を入れて』と追加指示してみます。」

#### 改善プロンプト
```
前の文章をベースに、以下を追加してください：
・この5年間で特に印象的だったエピソード
・担当者として個人的に感じている感謝
・来年度への具体的な期待
```

### 最終出力の確認（5:30〜6:30）

#### 出力例
```
件名：5年間のご愛顧に心から感謝申し上げます

ABC商事 山田様

いつも大変お世話になっております。
株式会社〇〇の田中です。

早いもので、貴社とお取引を始めさせていただいてから
5年という月日が経ちました。

初めてお会いした際、山田様が「一緒に成長できる
パートナーを探している」とおっしゃった言葉が
今でも心に残っています。

この5年間、様々なプロジェクトでご一緒させていただき、
時には厳しいご指摘もいただきながら、
私たちも大きく成長することができました。

特に昨年の新システム導入の際は、
深夜まで一緒に対応していただいたこと、
本当に感謝しております。

来年度も、さらなる価値をご提供できるよう
チーム一同、全力で取り組んでまいります。

今後とも変わらぬご指導ご鞭撻のほど、
よろしくお願い申し上げます。

心からの感謝を込めて

株式会社〇〇
営業部 田中
```

### ポイント解説（6:30〜7:00）

#### 講師セリフ
「パーソナルな要素を加えることで、テンプレート感がなくなり、本当に心がこもった文章になりました。」

## トーン調整のコツ（7:00〜9:00）

### 3つの重要ポイント（7:00〜8:00）

#### 講師セリフ
「Freeformのコツは3つあります。」

#### ポイント1：具体的な形容詞
- NG：「丁寧な」
- OK：「丁寧かつ親しみやすい」
- Best：「プロフェッショナルでありながら温かみのある」

#### ポイント2：状況説明の追加
- 相手との関係性
- これまでの経緯
- 特別な事情や背景

#### ポイント3：簡潔な指示
- 3行以内にまとめる
- 焦点を明確にする
- 矛盾する要求を避ける

### 実践的な組み合わせ例（8:00〜9:00）

#### 講師セリフ
「これらを組み合わせた実践例を見てみましょう。」

#### 良い例
```
初めてお会いするお客様への挨拶メール：
・緊張感を与えない親しみやすさ
・プロとしての信頼感は保つ
・次回お会いするのが楽しみになる締めくくり
```

#### 悪い例
```
メールを書いてください。丁寧に。
相手は重要顧客です。失礼のないように。
でも堅くならないように。形式的でもダメ。
親しみやすく、でもなれなれしくない...（続く）
```
