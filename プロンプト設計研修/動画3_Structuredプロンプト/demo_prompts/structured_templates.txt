# Structuredプロンプトテンプレート集

## 1. 社内研修案内テンプレート

```
以下の情報を使って社内研修案内メールを作成してください。

【入力情報】
件名：[研修名]
対象：[参加対象者]
日時：[開催日時]
場所：[開催場所]
申込期限：[締切日]
備考：[その他特記事項]

【出力構成】
1. 件名（【重要】を付ける）
2. 宛名（各位）
3. 研修の目的（1-2文）
4. 開催詳細（箇条書き）
   - 日時
   - 場所
   - 対象者
   - 所要時間
5. 申込方法（具体的手順）
6. 注意事項（2-3点）
7. 問い合わせ先（担当部署と内線）

【トーン】
丁寧でありながら親しみやすい
```

## 2. 会議招集メールテンプレート

```
以下の構成で会議招集メールを作成してください。

【入力項目】
会議名：[会議の名称]
日時：[開催日時]
場所：[会議室/オンライン]
参加者：[必須参加者/任意参加者]
議題：[主要議題3点]
準備物：[事前準備が必要なもの]

【出力形式】
1. 件名：[会議名] 開催のご案内
2. 宛名
3. 会議開催の背景（簡潔に）
4. 会議詳細
   □ 日時
   □ 場所/接続先
   □ 参加者
5. 議題（番号付きリスト）
6. 事前準備のお願い
7. 締めの言葉
```

## 3. 承認通知テンプレート

```
申請承認の通知メールを作成してください。

【基本情報】
申請者：[氏名]
申請内容：[申請の種類]
申請日：[申請受付日]
承認日：[承認日]
承認者：[承認者名]
条件：[承認条件があれば]

【メール構成】
1. 件名：【承認】[申請内容]について
2. 申請者への挨拶
3. 承認の通知（明確に）
4. 承認詳細
   - 申請内容の確認
   - 承認条件（あれば）
   - 有効期限（あれば）
5. 次のアクション
6. 注意事項
7. 問い合わせ先
```

## 4. イベント告知テンプレート

```
社内イベントの告知メールを作成。

【イベント情報】
イベント名：[名称]
目的：[開催目的]
日程：[開催日時]
会場：[場所]
定員：[人数]
費用：[参加費]
申込方法：[申込手順]

【出力構造】
1. 件名（キャッチーに）
2. イベント概要（魅力的に3行で）
3. 開催詳細（見やすく整理）
4. プログラム（時間割）
5. 参加メリット（3点）
6. 申込方法（ステップで）
7. FAQ（よくある質問2-3個）
8. 締切と連絡先

【雰囲気】
楽しそうで参加したくなる雰囲気
```

## 5. 業務連絡テンプレート

```
業務変更の連絡メールを作成。

【変更内容】
対象業務：[業務名]
変更内容：[具体的な変更点]
開始日：[変更適用日]
影響範囲：[影響を受ける部署/人]
理由：[変更の背景]

【構成指定】
1. 件名：【業務連絡】[対象業務]の変更について
2. 変更の要点（最初に結論）
3. 変更詳細
   ■ 現在の方法
   ■ 新しい方法
   ■ 変更理由
4. 実施スケジュール
5. 影響と対応
6. サポート体制
7. 質問受付先
```

## 6. 定期報告テンプレート

```
月次業務報告メールを作成。

【報告項目】
期間：[報告対象期間]
主要成果：[成果3点]
課題：[課題2点]
来月予定：[予定3点]
特記事項：[あれば]

【レポート形式】
1. 件名：[部署名]月次報告（[月]）
2. サマリー（3行で要約）
3. 成果報告
   ✓ 成果1（数値含む）
   ✓ 成果2（数値含む）
   ✓ 成果3（数値含む）
4. 課題と対策
5. 来月の計画
6. 補足事項
```

## 7. カスタマイズ例

### 基本構造
```
【入力】
項目1：
項目2：
項目3：

【出力】
1. セクション1
2. セクション2
3. セクション3

【オプション】
トーン：[指定]
文字数：[制限]
```

### トーン追加の例
```
【トーン】
- 親しみやすく前向きな雰囲気
- プロフェッショナルかつ温かい
- 簡潔で分かりやすい
```

## 8. 活用のポイント

1. **構造の明確化**
   - 入力項目を明確に定義
   - 出力形式を具体的に指定
   - 階層構造を活用

2. **再利用性の確保**
   - 汎用的な項目名を使用
   - 変更しやすい構造
   - チーム共有を前提に

3. **品質の安定化**
   - 必須項目の明示
   - 文字数や行数の指定
   - チェック項目の組み込み

4. **人間味の追加**
   - トーン指定を忘れない
   - 適度な柔軟性を残す
   - 状況に応じた調整余地

## 実践演習

以下のテンプレートを使って実際にメールを作成してみましょう：

1. 自部署の定例会議の招集
2. 研修参加の承認通知
3. システムメンテナンスの告知
4. 四半期業績の報告

各テンプレートをカスタマイズして、
自社の文化に合わせて調整してください。
