# Google Workspace Studio 人事業務ゼロタッチ講座

## コース概要
Google Workspace Studioを活用して、採用プロセス・オンボーディング・評価面談など人事業務を完全自動化する実践講座です。5つのレッスンで、候補者への連絡から入社手続き、評価サイクルまでを自動化し、人事担当者の工数を大幅に削減します。

## 対象者
- 採用業務の連絡・調整に多くの時間を費やしている採用担当者
- オンボーディングのチェックリスト管理に追われている人事担当者
- 評価面談のスケジュール調整を効率化したい人事マネージャー
- 社員情報の更新・管理を自動化したい総務・人事チーム

## 前提知識
- Google Workspaceの基本操作（Gmail、カレンダー、スプレッドシート）
- プログラミング知識は不要（完全ノーコード）

## 達成目標
- 採用プロセスの候補者コミュニケーション自動化
- 新入社員オンボーディングの進捗管理自動化
- 評価面談のスケジュール調整と事前準備の自動化
- 社員情報の定期更新リマインドと異動処理の自動化


## カリキュラム（全5回）

### 第1回：採用連絡を80%削減！履歴書AI要約から結果通知まで一気通貫フロー
**応募受付→書類選考→面接調整→結果連絡**

- **学習目標**: 採用エントリーから内定通知まで、候補者への連絡を自動化する
- **内容**:
  - Formsスターターでのエントリー受付
  - Geminiによる履歴書・職務経歴書の自動要約
  - 書類選考通過/不通過の自動連絡
  - 面接日程調整リンクの自動送信
  - 面接リマインドの自動送信
  - 選考ステータスの自動更新
- **作成するエージェント**: 採用候補者連絡エージェント

### 第2回：面接終了から1日で評価完了！自動フォーム配信と即時集計システム
**評価フォーム送信→回答収集→スコア計算→レポート生成**

- **学習目標**: 面接終了後に評価フォームを自動送信し、評価を集計してレポートを生成する
- **内容**:
  - 面接カレンダーイベント終了をトリガーに評価依頼
  - 面接官向け評価フォームの自動送信
  - 評価スコアの自動計算と可視化
  - 全面接官の評価が揃ったタイミングでの通知
  - Geminiによる評価コメントの要約
  - 採用判定会議用レポートの自動生成
- **作成するエージェント**: 面接評価収集エージェント

### 第3回：入社日から自動スタート！関係者全員にタスクが飛ぶオンボーディング自動化
**入社日トリガー→タスク配信→進捗追跡→完了確認**

- **学習目標**: 入社日を起点に、新入社員と関係部門へのタスク配信・進捗管理を自動化する
- **内容**:
  - 入社日スケジュールからの自動トリガー
  - 新入社員向けオンボーディングタスクの自動配信
  - 情シス・総務・配属先への準備依頼自動送信
  - Day1, Week1, Month1のチェックポイント自動確認
  - 未完了タスクの自動リマインド
  - オンボーディング完了報告の自動生成
- **作成するエージェント**: オンボーディング管理エージェント

### 第4回：100名分の面談調整が一瞬で完了！カレンダー連携スケジューリング
**評価期開始→日程調整→事前資料準備→リマインド送信**

- **学習目標**: 評価期に合わせて面談日程を自動調整し、事前準備資料を自動配布する
- **内容**:
  - 評価期開始トリガーでの面談調整開始
  - 上司・部下の空き時間自動検索
  - 面談日程の自動提案と確定
  - 事前自己評価フォームの自動送信
  - 過去評価データの自動取得と準備
  - 面談2日前のリマインド自動送信
- **作成するエージェント**: 評価面談調整エージェント

### 第5回：異動・退職の連絡漏れゼロ！関係部門へ自動通知する情報連携フロー
**定期チェック→更新依頼→変更検知→関係システム通知**

- **学習目標**: 社員情報の定期更新を促し、異動・退職情報を関係部門に自動連携する
- **内容**:
  - 四半期ごとの社員情報更新依頼自動送信
  - 住所・緊急連絡先変更の自動収集
  - 異動辞令発令時の自動通知フロー
  - 退職者への手続きチェックリスト自動送信
  - IT・総務・経理への連携事項自動通知
  - 組織図の自動更新トリガー
- **作成するエージェント**: 社員情報管理エージェント

## 技術スタック
- Google Workspace Studio
- Gemini（履歴書解析・評価要約・文章生成）
- Gmail
- Googleカレンダー
- Googleスプレッドシート
- Googleフォーム
- Google Chat / Slack連携

## 主要なステップ・機能

### スターター
| スターター | 用途 |
|-----------|------|
| Formsスターター | 採用エントリー・評価フォーム受信 |
| スケジュールスターター | 定期的なリマインド・更新依頼 |
| カレンダースターター | 面接終了・入社日トリガー |

### ステップ
| ステップ | 用途 |
|---------|------|
| Geminiステップ | 履歴書要約、評価集計、文章生成 |
| スプレッドシート連携 | 候補者・社員データ管理 |
| カレンダー連携 | 日程調整・空き時間検索 |
| Gmail送信ステップ | 候補者連絡・リマインド送信 |
| Chat DMステップ | 関係者への通知 |

## 自動化される業務フロー

```
採用フロー：
エントリー受付 → 書類要約 → 選考判定 → 結果連絡 →
面接日程調整 → リマインド → 評価収集 → 内定連絡

オンボーディングフロー：
入社日検知 → タスク配信 → 進捗監視 → リマインド →
チェックポイント確認 → 完了報告 → フォローアップ

評価フロー：
評価期開始 → 日程調整 → 事前準備 → リマインド →
面談実施 → 評価入力 → 結果通知
```

## 達成できること
- 採用候補者への連絡時間：80%削減
- 面接評価収集期間：5日→1日
- オンボーディングタスク管理工数：週3時間→30分
- 評価面談調整時間：1人あたり30分→0分

## 注意事項
- 外部APIキーは不要（Google Workspace内で完結）
- 対応エディション: Business Standard以上推奨
- 個人情報を扱うため、適切なアクセス権限設定を推奨

## 必要条件
- Google Workspace アカウント（Workspace Studio へのアクセス権）
- 採用管理スプレッドシート
- 社員マスタスプレッドシート
- オンボーディングチェックリストスプレッドシート
