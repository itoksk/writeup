# 動画1：デモプロンプト集
## Excelに頼らず！ChatGPTで売上CSVを"読ませる"初めの一歩

### 1. プロ級：データフレームを使った高精度分析
```
このExcelファイルをpandasのデータフレームとして読み込み、以下の分析を実行してください：

1. データ構造の確認
   - shapeで行数、列数を表示
   - dtypesで各列のデータ型を確認
   - head(10)で最初の10行を表示

2. 記述統計量の算出
   - describe()で基本統計量を出力
   - groupbyを使って商品別売上を集計
   - agg関数で複数の統計量を同時計算

3. データ品質チェック
   - isnull().sum()で欠損値をカウント
   - duplicated().sum()で重複データを検出
   - IQR法で外れ値を特定

結果は表形式とグラフで可視化してください。
```

### 2. 時系列分析のプロフェッショナルプロンプト
```
このデータをpandasデータフレームとして、時系列分析を実行してください。

分析要件：
1. 時系列データの前処理
   - pd.to_datetimeで日付列をdatetime型に変換
   - set_indexで日付をインデックスに設定
   - resampleで日別/週別/月別に集計

2. トレンド分析
   - rollingで移動平均を計算（7日、30日）
   - 線形回帰でトレンドラインを描画
   - 前月比、前年同月比をpct_changeで計算

3. 季節性分析
   - seasonal_decomposeでトレンド・季節性・残差を分解
   - 曜日別集計で週次パターンを検出

4. 可視化
   - matplotlibで折れ線グラフを作成
   - seabornでヒートマップを描画

結果はインタラクティブなグラフで表示してください。
```

### 5. データクリーニングプロンプト
```
売上CSVデータに以下の前処理を施してください：

1. 欠損値の検出と適切な補完（0埋め、平均値、線形補間等）
2. 異常値の特定（売上額が異常に高い/低いデータ）
3. データ型の統一（日付形式、数値形式の正規化）
4. 重複データの除去
5. 処理前後の件数・品質比較

処理済みデータをCSVファイルとして保存可能な状態にしてください。
```
