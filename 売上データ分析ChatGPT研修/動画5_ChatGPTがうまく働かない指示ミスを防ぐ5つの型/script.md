# 第5回：ChatGPTがうまく働かない？"指示ミス"を防ぐ5つの型（8分）

## オープニング（0:00-0:30）

### 導入フレーズ
「みなさん、こんにちは。売上データ分析×ChatGPT研修、いよいよ最終回です。

これまで4回にわたって、ChatGPTを使った分析技術を学んできました。

今日のテーマは『AIのせいじゃない。多くの誤解は"指示の曖昧さ"から生まれます』です。

8分間で、ChatGPTを確実に動かす『5つの指示型』をマスターしていただきます。」

## 1. なぜ指示が重要なのか（0:30-1:30）

### よくある失敗例
「まず、よくある失敗例を見てみましょう。

失敗例1：
質問：『このデータを分析して』
回答：一般的な統計情報の羅列...求めていた答えじゃない

失敗例2：
質問：『売上を予測して』
回答：どの期間？どの精度で？根拠は？...情報不足

失敗例3：
質問：『グラフを作って』
回答：棒グラフ？線グラフ？何を強調？...意図が不明確

これらはすべて、指示の曖昧さが原因です。」

### 良い指示の効果
「一方、明確な指示を出すと：

良い例：
『2023年の月別売上データから、季節変動を考慮した2024年1月の売上を予測し、その根拠を3点で説明してください』

このような指示なら、ChatGPTは的確に：
- 予測値：545万円
- 根拠1：前年同月比+5%の成長トレンド
- 根拠2：1月の季節指数0.95
- 根拠3：直近3ヶ月の移動平均からの推計

期待通りの回答が得られます。」

## 2. 5つの指示型の詳細解説（1:30-5:30）

### 第1の型：目的明示型
「第1の型は『目的明示型』です。

なぜそれが必要なのかを明確にします。

悪い例：『データをまとめて』
良い例：『経営会議で売上トレンドを説明するため、このデータから重要なポイントを3つ抽出してください』

ポイント：
- 用途を明確にする
- 相手（誰に見せるか）を示す
- 期待する成果を具体化する

実践例：
『投資家向けプレゼンのため、成長性を示す指標を5つ選んで、それぞれ1行で説明してください』」

### 第2の型：形式指定型
「第2の型は『形式指定型』です。

出力フォーマットを具体的に指定します。

悪い例：『分析結果を教えて』
良い例：『分析結果を以下の形式で出力してください：
1. 要約（3行以内）
2. 詳細データ（表形式）
3. 推奨アクション（箇条書き3点）』

実践テンプレート：
- 表形式で出力
- JSON形式で構造化
- Markdown形式で階層化
- CSV形式でエクスポート可能に

これで、そのまま使える形で結果が得られます。」

### 第3の型：粒度指定型
「第3の型は『粒度指定型』です。

どの程度詳しく説明してほしいかを指定します。

レベル1（要約）：『要点を3行で』
レベル2（標準）：『主要なポイントを段落ごとに』
レベル3（詳細）：『ステップバイステップで、初心者にも分かるように』

実践例：
『売上予測モデルについて、
- 経営層向け：結果のみ1行で
- 管理職向け：手法と結果を5行で
- 実務担当向け：実装方法を含めて詳細に』

相手に応じて粒度を変えることで、効率的なコミュニケーションが可能です。」

### 第4の型：プロセス指定型
「第4の型は『プロセス指定型』です。

どのような手順で考えてほしいかを指定します。

基本形：『ステップバイステップで』
応用形：『まず前提を確認し、次に分析を行い、最後に結論を述べてください』

実践例：
『売上データの異常値を検出するため、
1. まず基本統計量を確認
2. 次に標準偏差の2倍を超える値を特定
3. それらの発生月と要因を分析
4. 最後に対処法を提案
この順番で分析してください』

プロセスを指定することで、論理的で追跡可能な分析が得られます。」

### 第5の型：エラー修正型
「第5の型は『エラー修正型』です。

うまくいかなかった時の対処法です。

基本形：
『以下のエラーが出ました：[エラーメッセージ]
これを修正してください』

応用形：
『先ほどの回答は求めていたものと違いました。
期待していたのは〇〇です。
その観点で再度回答してください』

実践例：
『生成されたコードで "NameError: name 'pd' is not defined" というエラーが出ました。必要なインポート文を追加して、完全なコードを提供してください』

エラーを恐れず、修正依頼を明確にすることが重要です。」

## 3. 組み合わせ技と実践演習（5:30-7:00）

### 複合型プロンプトの作成
「実務では、5つの型を組み合わせて使います。

究極のプロンプト例：

『【目的】月次役員会議での売上報告のため
【形式】エグゼクティブサマリー形式で
【粒度】要点3つを各2行以内で
【プロセス】現状分析→課題抽出→改善提案の順で
【データ】添付の2023年売上データを使用して

上記条件で、1月の売上レビューを作成してください』

これで、そのまま会議で使える資料が生成されます。」

### よくある場面での応用
「場面別の実践例：

日報作成：
『今日の活動内容を基に、【形式】定型フォーマットで【粒度】各項目3行以内で日報を作成』

予算申請：
『【目的】IT投資の承認を得るため【プロセス】現状課題→解決策→投資効果の順で【粒度】A4用紙1枚に収まる分量で提案書を作成』

顧客報告：
『【形式】グラフ付きレポートで【粒度】専門用語を避けて【プロセス】結論先行型で今月の成果を報告』」

## 4. ChatGPTとの継続的な対話術（7:00-7:45）

### 文脈を保持した追加指示
「一度で完璧な答えを求める必要はありません。

対話の流れ：
1回目：『売上データを分析してください』
2回目：『その中で特に12月の異常値について詳しく』
3回目：『その原因と対策を3つずつ挙げてください』

段階的に深掘りすることで、より精度の高い分析が可能です。」

### フィードバックループ
「ChatGPTの回答を改善する方法：

『良い部分：グラフが見やすい
改善点：縦軸の単位が不明確
修正依頼：縦軸に"百万円"と単位を追加してください』

具体的なフィードバックで、どんどん改善されます。」

## 5. まとめと今後の学習指針（7:45-8:00）

### 5つの型の振り返り
「最後に、5つの型をもう一度確認しましょう：

1. 目的明示型：なぜ必要かを伝える
2. 形式指定型：どんな形で欲しいかを示す
3. 粒度指定型：どの程度詳しくかを決める
4. プロセス指定型：どういう順番でかを指定
5. エラー修正型：うまくいかない時の対処

これらを使いこなせば、ChatGPTは最強の相棒になります。」

### 継続的な改善
「生成AIを"答えさせる"のではなく、"伴走させる"。その鍵は、問いの力です。

今日から実践できること：

2. 慣れたら組み合わせる
3. チームでプロンプトを共有する

あなたの業務で、ぜひこの5つの型を使ってみてください。」
