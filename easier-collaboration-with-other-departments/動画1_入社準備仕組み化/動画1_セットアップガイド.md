# 動画1: Google Workspace完結型セットアップガイド

## 🎯 動画1の目標

Google Workspace内で完結する自動化システムの全体像を体感し、外部サービス依存の問題を解決する方法を学ぶ。

---

## 🛠️ 必要な準備

### **前提条件**
- Google Workspace アカウント（Business Standard以上推奨）
- Google Apps Script 利用権限
- ChatGPT API キー（オプション：AI機能用）

### **使用するGoogleサービス**
- ✅ Googleフォーム（情報収集）
- ✅ Googleスプレッドシート（データ管理）
- ✅ Google Apps Script（自動化エンジン）
- ✅ Gmail API（メール送信）
- ✅ Google Calendar API（予定管理）
- ✅ Google Chat API（チーム通知）

---

## 📋 セットアップ手順

### **ステップ1: Googleフォーム作成**

1. Google Drive で「新規」→「Googleフォーム」を選択
2. フォームタイトル：「内定者情報入力フォーム」
3. 以下の質問を追加：

```
質問1: 氏名 (記述式・必須)
質問2: メールアドレス (記述式・必須・メール形式検証)
質問3: 入社日 (日付・必須)
質問4: 配属部署 (選択式：営業部/開発部/マーケティング部/人事部)
質問5: 職種 (記述式・必須)
質問6: 雇用形態 (選択式：正社員/契約社員/パート)
```

4. 「回答」タブ→スプレッドシートにリンク→「新しいスプレッドシートを作成」

### **ステップ2: Google Apps Script設定**

1. 作成されたスプレッドシートを開く
2. 「拡張機能」→「Apps Script」を選択
3. 提供コード（`動画1_Google_Apps_Script実装.gs`）をコピー＆ペースト
4. プロジェクト名を「AI人事執事システム」に変更

### **ステップ3: API有効化**

1. Apps Script エディタで「サービス」の「+」をクリック
2. 以下のサービスを追加：
   - Gmail API
   - Google Calendar API
   - Google Chat API

### **ステップ4: トリガー設定**

1. Apps Script エディタで「トリガー」をクリック
2. 「トリガーを追加」を選択
3. 設定：
   - 実行する関数：`onFormSubmit`
   - イベントソース：「スプレッドシートから」
   - イベントタイプ：「フォーム送信時」

### **ステップ5: プロパティ設定**

1. Apps Script エディタで「プロジェクトの設定」
2. 「スクリプトプロパティ」に以下を追加：

```
OPENAI_API_KEY: your-chatgpt-api-key
IT_CHAT_WEBHOOK: your-it-chat-webhook-url
GENERAL_AFFAIRS_CHAT_WEBHOOK: your-general-affairs-webhook-url
TASK_SHEET_ID: your-task-management-sheet-id
```
