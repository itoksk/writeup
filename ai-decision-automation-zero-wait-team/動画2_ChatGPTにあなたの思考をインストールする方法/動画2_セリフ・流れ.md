# 動画2: ChatGPTにあなたの思考をインストールする方法（10分）

## 導入フレーズ（0:00-1:00）
「動画1で作った10分システム、実は『頭脳』がまだ入っていません。『営業費用30万円、承認して』と聞かれて、あなたはどう判断しますか？金額だけ？効果は？リスクは？その瞬間、あなたの頭の中では複雑な判断基準が働いています。今日は、その『あなたの思考プロセス』をChatGPTに完璧にコピーする方法をお教えします。これができれば、AIがあなたの分身として、あなたと同じ品質で判断してくれるようになります」

## Before/After（1:00-2:30）
**Before: プロンプトなし・適当なプロンプト**
- 「承認して」→ AIの回答が毎回バラバラ
- 判断根拠が不明確「多分承認でいいと思います」
- 重要な観点の見落とし（法務リスク、予算超過）
- Makeで自動化しても使い物にならない

**After: 最適化プロンプト導入後**
- 毎回一貫した判断基準で分析
- 明確な根拠「ROI 280%、過去実績あり、リスク低」
- チェックリスト完備で見落としゼロ
- 信頼度スコア付きで安心して任せられる

## プロンプト設計の核心（2:30-5:00）
「では、実際にMakeで設定したChatGPTモジュールに、あなたの頭脳をインストールしましょう」

### 実演：プロンプトの3層構造

**1. 役割設定（Role）- AIの人格を決める**
```
実際にMakeの画面を表示
System Messageに入力：

「あなたは創業15年のIT企業で、10年以上の管理職経験を持つ
承認責任者です。コスト意識が高く、投資対効果を重視しますが、
チームの成長に必要な投資は積極的に承認する方針です。」
```

**2. 判断基準（Criteria）- 明確なルール**
```
続けて入力：

【判断基準】
- 30万円以下＋ROI 200%以上 → 即承認
- 50万円以下＋必要性明確 → 条件付き承認  
- 50万円超 or ROI不明 → 追加情報要求
- 法的リスクあり → 即却下

【重視ポイント】
1. 投資回収期間（12ヶ月以内が理想）
2. チーム生産性への影響
3. 競合他社の動向
```

**3. 出力形式（Format）- ブレない回答**
```
最後に追加：

必ず以下の形式で回答してください：
【判断】承認/条件付き承認/却下
【理由】最大3つの根拠を箇条書き
【信頼度】0-100%
【次のアクション】具体的な指示
```

### デモ実行（画面共有）
1. Makeでテスト実行
2. 同じ案件で3回実行→全て同じ品質の回答
3. 実際のSlackでの動作確認

## 実践的なプロンプト改善テクニック（5:00-7:30）

### 1. 具体例を入れる威力
```
【過去の成功事例】
- Slack有料版（月3万円）→ コミュニケーション効率40%向上
- Tableau導入（月5万円）→ 意思決定スピード2倍

【過去の失敗事例】  
- 高額CRM（月20万円）→ 活用率10%で解約
```

### 2. 条件分岐を明確に
```
IF 法務リスクあり → 即却下
ELSE IF 予算50%超過 → 役員承認必要
ELSE IF ROI不明 → 追加情報要求
ELSE → 通常判断プロセス
```

### 3. エラー処理も忘れずに
```
情報不足の場合は必ず以下を要求：
- 具体的な費用内訳
- 期待効果の数値化
- 実施スケジュール
- 責任者名
```

### 4. Make内でのプロンプト管理
- Variables機能でプロンプトを外部管理
- バージョン管理で改善履歴を記録
- A/Bテストで最適化

## 実装のコツと注意点（7:30-9:00）

### セキュリティ面での配慮
- 社名や機密情報は変数化
- 金額の上限設定は必須
- 個人情報は含めない

### プロンプトの育て方
```
1週目：基本動作の確認
2週目：エッジケースの追加
3週目：判断精度の微調整
4週目：チームフィードバック反映
```

### よくある失敗と対策
| 失敗例 | 原因 | 対策 |
|--------|------|------|
| 判断がブレる | プロンプトが曖昧 | 数値基準を明確に |
| 重要項目を見落とす | チェック項目不足 | 必須確認リスト追加 |
| 回答が長すぎる | 形式指定なし | 文字数制限を設定 |

## まとめ（9:00-9:30）
「これで、ChatGPTがあなたの『思考のコピー』として動くようになりました。重要なのは：
- **役割・基準・形式**の3層構造
- **具体例**でAIを教育
- **継続的な改善**で精度向上

月100時間の承認業務が、あなたの判断品質そのままで自動化されます」
