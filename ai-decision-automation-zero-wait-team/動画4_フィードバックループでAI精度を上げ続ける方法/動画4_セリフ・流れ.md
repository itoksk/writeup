## A/Bテストの基本概念と実装（2:30-4:30）
### なぜA/Bテストが必要？

「動画2で作った基本プロンプト、動画3で追加したFew-shot学習...どちらが本当に効果的でしょうか？」

**実例デモ：同じ案件を2つのプロンプトで判断**
```
案件：「マーケティングオートメーションツール導入 30万円」

プロンプトA（基本版）の判断：
→ 条件付き承認（信頼度: 82%）
「ROI予測が不明確なため、3ヶ月のトライアル実施を推奨」

プロンプトB（Few-shot学習版）の判断：
→ 承認（信頼度: 88%）
「過去の類似案件で平均ROI 250%を達成。早期導入を推奨」
```

### Google SheetsでのA/Bテスト管理
**画面共有：実際のテスト管理シートを作成**

1. **列構成の設計**
   - A列：日付（自動記録）
   - B列：案件内容（Slackから自動転記）
   - C列：使用プロンプト版（A/B自動振り分け）
   - D列：AI判断結果
   - E列：AI信頼度
   - F列：実際の結果（3ヶ月後に手動更新）
   - G列：正誤判定（=IF(D列=F列,1,0)）
   - H列：カテゴリ（システム/マーケティング/人材）
   - I列：金額レンジ
   - J列：改善ヒント（間違えた理由の分析）

2. **自動振り分けの実装**
   ```
   =IF(MOD(DAY(A2),2)=1,"プロンプトA","プロンプトB")
   ```

### STEP2: 2週間後の分析デモ
**実際のデータで統計分析を実演**

```javascript
// Google Apps Scriptで自動分析
function analyzeABTest() {
  // プロンプトAの精度計算
  const accuracyA = sheet.getRange("K2").setFormula(
    '=AVERAGEIF(C:C,"プロンプトA",G:G)'
  );
  
  // プロンプトBの精度計算
  const accuracyB = sheet.getRange("K3").setFormula(
    '=AVERAGEIF(C:C,"プロンプトB",G:G)'
  );
  
  // 統計的有意性の確認
  const sampleSize = sheet.getRange("K4").setFormula(
    '=COUNTIF(G:G,"<>"")'
  );
  
  // 勝者の判定と自動切り替え提案
  if (accuracyB > accuracyA && sampleSize > 30) {
    return "プロンプトBを採用推奨";
  }
}
```
