# 🚀 バイブコーディング実践講座：請求書管理システムをノリと勢いで作る！

## 🎯 コース概要

**バイブコーディングとは（Google Cloud公式定義）**  
AIを使用して自然言語プロンプトから機能コードを生成する新しいソフトウェア開発手法。2025年初頭にAI研究者Andrej Karpathy氏が考案。コードを1行ずつ記述するのではなく、AIアシスタントをガイドして、会話的なプロセスを通じてアプリケーションを生成・改良・デバッグします。

**この講座のアプローチ**  
「責任あるAIを活用した開発」として、AIを強力な「ペアプログラマー」として活用。Google Apps Script（GAS）で請求書管理システムを題材に、自然言語プロンプトでの開発を実践。各講座でデモプロンプトを使い、効果的なAIとの対話方法を体得します。

## 📚 カリキュラム構成（請求書管理システムを作りながら学ぶ）

### 動画1：迷わない開発を始めよう！チケット思考の基本と実践
**請求書管理の機能をチケット分解**
- 請求書管理システムの要件をバイブでチケット化
- 「請求書作成」「顧客管理」「PDF出力」を小さく分解
- ChatGPTでノリよくチケット生成
- バイブコーディングの第一歩

### 動画2：AIツールをどう使い分ける？ChatGPT・Gemini・Google AI Studioの活用法  
**請求書のUIをGeminiでバイブ作成**
- ChatGPTでGASの基本構造
- Geminiで請求書フォームのUI生成（手書きスケッチから）
- Google AI StudioでスプレッドシートAPI設計
- ツールを使い分けてバイブを加速

### 動画3："伝わる指示"はこう作る！要件定義テンプレートでAIへの指示ミスを防ぐ方法
**請求書システムの要件をバイブで高速定義**
- 「請求書管理っぽいやつ作って」→失敗
- テンプレートで「GASで請求書、PDF出力付き」→成功
- バイブを保ちながら精度を上げる
- 5分で要件定義、あとはノリで進む

### 動画4：進捗が見えると仕事が進む！チケットで開発を見える化する方法
**請求書機能の進捗をドキュメント化**
- Geminiで進捗ドキュメント自動生成
- 「今日は請求書作成完了！」を記録
- スプレッドシートで請求書開発ログ
- バイブの記録が次のバイブを生む

### 動画5：Gemini×AI Studioで同時進行！複数タスクを効率よく回すAI活用術
**請求書の各機能を並列バイブ開発**
- Gemini：請求書テンプレートUI
- ChatGPT：GASの計算ロジック
- AI Studio：データベース設計
- 3つ同時に進めてバイブMAX

### 動画6：Vibe Codingを毎日の習慣に！AI開発を続けるための仕組みづくり
**請求書システムを日々改善するバイブ習慣**
- 毎日15分の請求書機能追加タイム
- 「今日は消費税計算追加」のような小さなバイブ
- GASトリガーで自動レポート
- バイブログで振り返り

## 🎯 バイブコーディングのワークフロー（Google Cloud定義準拠）

**コードレベルのワークフロー（緊密な会話ループ）：**
1. **目標を説明する** - 「請求書のPDF出力機能を作って」
2. **AIがコードを生成** - 初期コードが自動生成される
3. **実行と観察** - 生成されたコードを実行して確認
4. **フィードバックを提供** - 「エラー処理を追加して」
5. **繰り返し** - 完成まで対話を継続

**アプリケーションのライフサイクル：**
1. **アイディエーション** - 「請求書管理システムを作りたい」
2. **生成** - AIが初期バージョンを生成
3. **反復的な改良** - 機能追加や修正を対話で実施
4. **テストと検証** - 品質とセキュリティをレビュー
5. **デプロイ** - Cloud Runなどにワンクリックでデプロイ

## 🛠️ 推奨ツール（Google Cloud公式）

**本講座で使用するバイブコーディングツール：**

| ツール | 用途 | スキルレベル |
|--------|------|------------|
| **Google AI Studio** | アイデアをすぐに形に。単一プロンプトでアプリ生成 | 初心者向け |

**デプロイ先：**
- **Cloud Run** - 生成したアプリを即座にデプロイ
