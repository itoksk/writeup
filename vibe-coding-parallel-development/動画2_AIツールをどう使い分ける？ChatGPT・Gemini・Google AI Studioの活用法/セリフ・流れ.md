# 第2回：AIツールをどう使い分ける？ChatGPT・Gemini・Google AI Studioの活用法（10分）

## 導入フレーズ（1分）

「GeminiもChatGPTもGoogle AI Studioも…ツールが多すぎて、何に何を使えばいいか迷っていませんか？

- 全部のツールで同じことを試している
- どれが一番いいか分からない
- 結局ChatGPTだけ使っている

実は、それぞれのツールには『得意分野』があるんです。

今日は、各ツールの強みを活かして、開発効率を2倍にする方法をお伝えします！」

## Before/After（1.5分）

「従来の使い方と、役割分担した使い方の違いを見てみましょう。

**Before：全部のツールで全部やる**
- ChatGPTで全て完結させようとする
- 同じ質問を複数ツールで試す
- ツールの切り替えで時間ロス
- どれが最適か分からない
- 結果：混乱と時間の無駄

**After：それぞれの得意分野を分ける**
- 要件整理→ChatGPT（対話が得意）
- UI設計→Gemini（ビジュアル強い）
- API実装→Google AI Studio（統合しやすい）
- ドキュメント→Claude（文章力高い）
- 結果：効率が格段に向上！

この使い分けが、開発速度を劇的に変えます！」

## デモ（2分）

「例：Todoアプリ開発で各ツールをどう役割分担するか、実際にお見せします。

【画面共有：マルチウィンドウ】

**Step 1：ChatGPTで要件整理とチケット化**
```
Todoアプリを作りたいです。
必要な機能をチケット形式で整理してください。
```
→ 明確なチケットリストが生成

**Step 2：GeminiでUI設計**
```
TodoアプリのUIを作って。
Material Design風で、モバイルファースト。
HTMLとCSSのコードをください。
```
→ 美しいUIコードが即座に生成

**Step 3：Google AI StudioでAPI設計**
```
TodoアプリのバックエンドAPI設計。
GASで実装、スプレッドシート連携。
CRUD操作ができる関数群を。
```
→ 完全なAPIコードセットが生成

このように、15分で全体の骨組みが完成しました！
各ツールが得意な部分を担当することで、品質も速度も向上します。」

## 構築手順（1分）

「まず最初に、開発のフェーズを設計→UI→ロジック→統合に分け、それにツールを割り当てます。

**開発フェーズとツールマッピング：**

1. **企画・設計フェーズ**
   - 要件定義：ChatGPT
   - チケット分解：ChatGPT

2. **実装フェーズ**
   - UI/UX：Gemini
   - ロジック：Google AI Studio
   - テスト：Claude

3. **統合フェーズ**
   - デバッグ：ChatGPT
   - ドキュメント：Claude

この割り当てを決めておくことで、迷いがなくなります！」

## 設定実演（3.5分）

「ChatGPTでは要件整理とチケット化、GeminiではUI試作、Google AI StudioではAPI設計をそれぞれ実行してみます。

【画面共有：実際の開発フロー】

**1. ChatGPT：要件整理（1分）**
```
請求書管理システムの要件を整理したい。
以下の観点で整理してください：
- 必須機能（MVP）
- あると嬉しい機能
- 技術的制約
- 開発順序
```

ChatGPTの出力：
```
【必須機能（MVP）】
1. 請求書の作成・保存
2. 顧客情報管理
3. 一覧表示

【開発順序】
1. データ構造設計（30分）
2. 基本CRUD実装（1時間）
3. UI作成（1時間）
```

**2. Gemini：UI設計（1.5分）**
```
請求書入力フォームのUIを作って。
条件：
- レスポンシブデザイン
- Material Design風
- 入力しやすいレイアウト
```

→ 完全なHTML/CSSが生成、プレビュー付き

**3. Google AI Studio：バックエンド実装（1分）**
```
GASで請求書データのCRUD操作。
スプレッドシート連携で。
```

→ 即座に使えるGASコードが生成

**結果：**
3つのツールを使い分けることで、通常2時間かかる作業が30分で完了！」

## 注意点・活用ポイント（1分）

「無理に全部のツールを使わず、目的に応じて"相性の良いものだけ"選ぶのがポイントです。

**ツール選択の判断基準：**

✅ **ChatGPTを選ぶとき：**
- 対話しながら考えたい
- 要件が曖昧
- アイデアを整理したい

✅ **Geminiを選ぶとき：**
- UIやデザインが必要
- Google系サービスと連携
- ビジュアル重視

✅ **Google AI Studioを選ぶとき：**
- プロトタイプを素早く
- APIやバックエンド開発
- 統合的な開発

**避けるべきアンチパターン：**
- 全部ChatGPTで済ませる
- ツールを切り替えすぎる
- 得意分野を無視する」

## まとめ（1分）

「ツールごとの得意領域を理解すれば、AI活用はもっと簡単になります。

**今日のポイント：**
- ChatGPT = 対話と要件整理のプロ
- Gemini = UIとビジュアルの専門家
- Google AI Studio = 統合開発の味方
- Claude = ドキュメントと品質管理

**実践のコツ：**
1. まずChatGPTで要件を明確に
2. 各フェーズで最適なツールを選択
3. 無理に全部使わない（2-3個でOK）

次回は『要件定義テンプレートでAIへの指示ミスを防ぐ』方法を解説！
曖昧な指示から卒業して、一発で期待通りの結果を得る秘訣をお伝えします。

今日から、ツールの使い分けを意識してみてください。
きっと開発スピードが変わるはずです！」
